avahnyābhāsasphuraṇapūrvako api jāyamāno dhūmābhāso dūrādau niyatiśaktyā kṛtasāmarthyād agnyābhāsād eva taddeśagatatattatpramātrantarasādhāraṇāt tasyaiva sa avyabhicāreṇa gamako, jātarūḍhas tu dhūmābhāsas tadvahnyābhāsasyātyantaparikṣayāt tattatpramātrantaragād dhūmābhāsād eva parokṣād api pūrvavat kṛtādhipatyāt ।।
