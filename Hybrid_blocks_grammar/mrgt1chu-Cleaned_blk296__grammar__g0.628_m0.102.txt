pāśāvṛtadṛkkriyāvṛttiṣv aṇuṣu tattatpāśaśaktyanuvartanadvāreṇa janmadrāvaṇādiduḥkhadāyitvād vāmo api parameśvaras tadabhyudayāyaiva pravṛttatvān na duḥkhahetur avagamyate yathā vaidyaḥ kṣāraśastrādinā rogiṇaṃ vyathayann api koṭau prānte abhimatasyārogyalakṣaṇasyārthasya sādhakatvāt vyathāheturapi na duḥkhadāyitvenaiva jñāyate ।। 
nanv atyantaśuddhasvarūpasya bhagavataḥ kiṃ pāśadharmānuvartanenety āha ।।
