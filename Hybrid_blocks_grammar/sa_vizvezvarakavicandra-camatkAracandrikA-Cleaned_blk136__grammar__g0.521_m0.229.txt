atrocchedasya sākṣādamaṅgalārthatve api śatrupadena guptatvādguṇatvam । kṛtyetyasyāmaṅgalāthatve api kṛtyākṛtye kāryākārye iti sāhacaryeṇa guptatvam । keśava ityasyāntyavarṇadvayenāmaṅgalārthasmaraṇahetutve api bhagavanmāhātmyena guptatvam । keśava ityasyāntyavarṇadvayenāmaṅgalārthasmaraṇahetutve api bhagavanmāhātmyena guptatvamityamīṣāṃ guptatvam ।
vrīḍākaramasabhyaṃ sayātpūrvavattadapi tridhā ।।
vaṃśāssahasraṃ santvanye śiśnodaraparāyaṇāḥ ।
eko recarlavaṃśo ayaṃ sādhurakṣāparāyaṇaḥ ।।
