[] atha guravaś ca tā rajanijanitatadvirahajvālākalitasnehapūravaśatayā muhur agurutām āsādyānavadyāmodam āviṣkurvatyaḥ pūrvadiṅmukhamahā-mandirālindād avateruḥ । tatra pūrvaṃ mātā vatsam iva mātā vatsaṃ militavatī, yatra rohiṇy api rohiṇīvad ūhāñcakre lokacakreṇa । [] śrī-govindaś ca dvayor api tayoḥ padāravindaṃ kramād vanditvā nanditvā, mānyānām anyāsām api yathānyāyaṃ mānam unnamayāmāsa ।
