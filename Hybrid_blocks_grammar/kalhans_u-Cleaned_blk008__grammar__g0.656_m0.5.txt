dātuṃ vāñchati dakṣiṇe api nayane vāmaḥ karaḥ kajjalaṃ bhaujaṅgaṃ ca bhuje aṅgadaṃ ghaṭayituṃ vāme api vāmetaraḥ ।
itthaṃ svaṃ svam aśikṣitaṃ bhagavator ardhaṃ vapuḥ paśyatoḥ sādhārasmitalāñchitaṃ diśatu vo vaktraṃ manovāñchitam ।।
