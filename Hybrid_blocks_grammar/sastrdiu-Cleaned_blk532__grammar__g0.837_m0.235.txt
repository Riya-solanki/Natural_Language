nanu vibhutve sarvaśarīreṣveka evātmāstu kiṃ nānātmabhirabhyupagataiḥ,naivam- ekatve sati yathā paraṇabhede api karturekatvāccakṣuṣā dṛṣṭamarthaṃ yajñadattaḥ pratyabhijānāti- yamahamadarśaṃ tamahaṃ spṛśāmīti, tathā devadattaśarīrasthenātmanā dṛṣṭamarthaṃ yajñadattaḥ pratyabhijānīyāt, śarīrabhede apyātmanasaḥ, śarīrabhede apyātmanaḥ pratyabhijñāturekatvāt ।
karturekatve api manobhedādvyavastheti cet, na, karaṇatvānmanasaḥ, karaṇabhedasya cakṣurāderbhedavyavasthāpakatvāyogādityuktam ।।
