poṭā dāsī dviliṃgā ca ghṛṣṭī gharṣaṇasūkarau ghaṭā ghoṣṭhyāṃ hastipaṅktau kṛpīṭamudare jale iti ṭāntāḥ puṃsi koṣṭho antarjaṭharaṃ kusūlo antargṛhaṃ tathā niṣṭhā niṣpattināśāntāḥ kāṣṭhotkarṣe sthitau diśi triṣu jyeṣṭho atiśaste api kaniṣṭho atiyuvālpayoḥ iti ṭhāntāḥ daṇḍo astrī laguḍe api syād guḍo golekṣupākayoḥ sarpa māṃsātpaśū vyāḍau gobhūvācas tviḍā ilāḥ kṣveḍavaṃśaśalākāpi nāḍī kāle api ṣaṭkṣaṇe kāṇḍo astrī daṇḍabāṇārvavargāvasaravāriṣu syād bhāṇḍamaśvābharaṇe amatre mūlavaṇigdhane iti ḍāntāḥ bhṛśapratijñayor bāḍhaṃ pragāḍhaṃ bhṛśakṛcchrayoḥ saṃghātagrāsayoḥ piṇḍī dvayoḥ puṃsi kalevare gaṇḍau kapolavisphoṭau muṇḍakas triṣu muṇḍite ikṣubhede api khaṇḍo astrī śikhaṇḍo barhacūḍayoḥ śaktasthūlau triṣu dṛḍhau vyūḍhau vinyastasaṃhatau iti ḍhāntāḥ bhrūṇo arbhake straiṇagarbhe bāṇo balisute śare kaṇo atisūkṣme dhānyāṃśe saṃghāte pramathe gaṇaḥ paṇo dyūtādiṣūtsṛṣṭe bhṛtau mūlye dhane api ca maurvyāṃ dravyāśrite satvaśauryasandhyādike guṇaḥ nirvyāpārasthitau kālaviśeṣotsavayoḥ kṣaṇaḥ varṇo dvijādau śuklādau stutau varṇaṃ tu vākṣare aruṇo bhāskare api syād varṇabhede api ca triṣu sthāṇuḥ śarvo apyatha droṇaḥ kāke apyājau rave raṇaḥ grāmaṇīr nāpite puṃsi śreṣṭhe grāmādhipe triṣu ūrṇā meṣādilomni syādāvarte cāntarā bhruvoḥ hariṇī syān mṛgī hemapratimā haritā ca yā triṣu pāṇḍau ca hariṇaḥ sthūṇā stambhe api veśmanaḥ triṣṇe spṛhāpipāse dve jugupsākaruṇe ghṛṇe vaṇikpathe ca vipaṇiḥ surā pratyak ca vāruṇī kareṇuribhyāṃ strī nebhe draviṇaṃ tu balaṃ dhanam śaraṇaṃ gṛharakṣitroḥ śrīparṇaṃ kamale api ca viṣābhimaraloheṣu tīkṣṇaṃ klībe khare triṣu pramāṇaṃ hetumaryādāśāstreyattāpramātṛṣu karaṇaṃ sādhakatamaṃ kṣetragātrendriyeṣvapi prāṇyutpāde saṃsaraṇamasaṃbādhacamūgatau ghaṇṭāpathe atha vāntānne samudgiraṇamunnaye atas triṣu viṣāṇaṃ syāt paśuśṛṅgebhadantayoḥ pravaṇe kramanimnorvyāṃ prahve nā tu catuṣpathe saṃkīrṇau nicitāśuddhā viriṇaṃ śūnyamūṣaram setau ca caraṇo veṇī nadībhede kacoccaye iti ṇāntāḥ devasūryau vivasvantau sarasvantaunadārṇavau pakṣitārkṣyau garutmantau śakuntau bhāsapakṣiṇau agnyutpātau dhūmaketū jīmūtau meghaparvatau hastau tu pāṇinakṣatre marutau pavanāmarau yantā hastipake sūte bhartā dhātari poṣṭari yānapātre śiśau potaḥ pretaḥ prāṇyantare mṛte grahabhede dhvaje ketuḥ pārthive tanaye sutaḥ sthapatiḥ kārubhede api bhūbhṛd bhūmidhare nṛpe mūrdhābhiṣikto bhūpe api ṛtuḥ strī kusume api ca viṣṇāvapyajitāvyaktau sūtas tvaṣṭari sārathau vyaktaḥ prājñe api dṛṣṭāntāvubhau śāstranidarśane kṣattā syāt sārathau dvāḥsthe kṣatriyāyāṃ ca śūdraje vṛttāntaḥ syāt prakaraṇe prakāre kārtsnyavārtayoḥ ānartaḥ samare nṛtyasthāna nīvṛdviśeṣayoḥ kṛtānto yama siddhānta daivākuśalakarmasu śleṣmādi rasa raktādi mahā bhūtāni tad guṇāḥ 
indriyāṇyaśma vikṛtiḥ śabdayoniś ca dhātavaḥ
kakṣāntare api śuddhānto bhūpasyāsarva gocare
kāsū sāmarthyayoḥ śaktir mūrtiḥ kāṭhinyakāyayoḥ ।।
