kalpadrumāvadānamālāyāṃ subhūtyavadānam ।। upaguptaṃ guruṃ natvā kṛtāñjalipuṭo avadat ।। yad uktaṃ śākyasiṃhena tan me gaditum arhasi ।। parṣadaṃ ca samālokya saṃbabhāṣa subhāṣitam ।।
subhūter avadānaṃ yat tat pravakṣye yathāśrutam ।। vidyācaraṇasaṃpannaḥ sugato lokavijjinaḥ ।। satkṛto mānitaḥ satvair gurukṛtaś ca pūjitaḥ ।। śreṣṭhibhiḥ sārthavāhaiś ca nānādeśasamāgataiḥ ।।
guruḍaiś ca mahāsarpais tathānyasatvajātikaiḥ ।। piṇḍapātāsanādīnāṃ śayyādīnāṃ tathaiva ca ।। sārdhaṃ saśrāvakaiḥ saṃghair bhikṣubhiś ca jitendriyaiḥ ।। bodhisatvamahāsatvaiḥ satvārthabodhivāñchibhiḥ ।।
vihāre vyaharad dharmaṃ deśayañ chubhayas sadā ।। āmantrya bhikṣavaḥ sarve samyagājñāpitā iti ।। deśān pratyabhigacchantaḥ prakāśayata saṃvṛtim ।। śrāvakā bhikṣavaś caiva pratasthus te niyoginaḥ ।।
deśayanti sma saddharmaṃ pratideśānupāsthitāḥ ।। tathottarāṃ diśaṃ gatvā vidikṣu cāpi sarvataḥ ।। meror upariṣaṇḍāyām adhyūṣur dhyānatatparāḥ ।। tam āśramam upānīya bhakṣitum upacakrame ।।
dṛṣṭvaiva suprasannena manasā praṇidhir dadhe ।। aham api ca bhūyāsaṃ tādṛgdharmasamāhitaḥ ।। tatraiva garuḍenaivaṃ bhakṣito abhūt sa nāgakaḥ ।। bhūtināmnā dvijasyāsau bhāryāyā garbham āviśat ।।
tatas tatsamaye prāpte dārakaḥ samajāyata ।। darśanīyaṃ subhadrāṅgaṃ muhuḥ paśyann ananda saḥ ।। bhavanto asya nu kiṃ nāma kriyatām iti so abravīt ।। sarve harṣasamāpannā bhūtiṃ tam abruvaṃs tathā ।।
tasmāt subhūtir ity eva nāmnā bhavatu viśrutaḥ ।। kumāratvaṃ kramāt prāpto rarāma sa vayonvitaḥ ।। kiṃcin nimittasaṃruṣṭo vigrahe nirato abhavat ।। sarvajñātiviruddhatvāt pitṛbhyām apy upekṣitaḥ ।।
viśrambhapraṇayaṃ tasmin naiva kaścid abhāṣata ।। tenaiva krodhinā sārdhaṃ samutsehur na kecana ।। subhūtiś ca guruṃ natvā lipim anvagrahīt kramāt ।। so adhītyaiṣāṃ suśīghreṇa pāraṃ prāpa subuddhimān ।।
atharvam apy adhītuṃ sa prārabhat tīkṣṇamānasaḥ ।। subhūtiṃ svātmajaṃ putraṃ dṛṣṭvaivaṃ samacintayat ।। kadācit kupito roṣāl loke anarthaṃ kariṣyati ।। ṛṣiṣu preṣayitvainaṃ yojayiṣye ca saṃyame ।।
ātharvaṇād vinirhṛtya prabodhayaṃs tam abravīt ।। tvaṃ hi vidvān mahāvijñaḥ sarvaśāstrāṅgapāragaḥ ।। viramya tadadhiṣṭhānād ṛṣicaryāṃ samācara ।। śāntātmā sukham āsthāya carasva vratam uttamam ।।
viviktāraṇyavāseṣu vasanti dhyāyinaḥ sadā ।। devānām api te mānyā vandanīyāḥ sadā khalu ।। avasanti puṇyatīrtheṣu te api hi paramarṣayaḥ ।। phalamūlodakais tuṣṭās te api dhanyā dvijottamāḥ ।।
bhikṣāśinaḥ samādhisthās te hi brahmavidāṃ varāḥ ।। nirmadā nirahaṃkārās ta eva brāhmaṇottamāḥ ।। vedaśāstrāgamais tasya kim eva svātmapoṣiṇaḥ ।। kiṃ bhāti muniveṣeṇa sa naṭarṣir ivonmadaḥ ।।
kiṃ tasya brahmavṛttena citte{na} parimohite ।। tasya kiṃ tapasā siddhe kevalaṃ pāpahetubhiḥ ।। sa kiṃ guhāniviṣṭo api na sādhur duṣṭajantuvat ।। tasya kiṃ brahmacaryeṇa kevalaṃ duḥkhahetunā ।।
nīco api sa dvijakalpo yato dātā prajāpatiḥ ।। sa eva brāhmaṇaḥ śuddhaḥ śrotriyo vedanān yatiḥ ।। caṇḍālo api sa vipraḥ syāl lokeśo hi kṣamākaraḥ ।। sa eva brāhmaṇo dhīro viśvakarmā yato vidhiḥ ।।
sa hi vipro mahābhijño brahmā jñānarato yataḥ ।। saiva dvijavaro vijño vedadharmāsthito dvijaḥ ।। svacitte bhāvitaṃ brahma sa eva brāhmaṇottamaḥ ।। sarvakleśān vinirjitya saddharmābhirato bhava ।।
kṛtāñjalis tathā natvā pitaram ity abhāṣata ।। tad ājñāṃ dehi me tāta cariṣye brahmasadvratam ।। pariṣvajyātmajaṃ putraṃ punar apy abravīn mudā ।। cara brahmavrataṃ samyagdhīracittasamāhitaḥ ।।
yāvat krodham anirjitya duṣṭāñjetuṃ na śaknuyāḥ ।। asusaṃsthitadharmāṇaṃ hanyur mārā hi sarvathā ।। viniḥkṛṣya prayatnena hantavyaḥ krodhapannagaḥ ।। tāvat sadguṇayukto api sevyate naiva sajjanaiḥ ।।
pravrajyāṃ samupāsiśritya cara brāhmaṇyam ādarāt ।। sahasā pitarau natvā munīnām āśramaṃ yayau ।। brāhmaṇyasaṃvaraṃ prāptuṃ pravrajyāṃ samayācata ।। pravrajyāṃ dehi me satyaṃ careyaṃ bhavamuktaye ।।
ehi vatsa cara brahmacaryaṃ jitvā ṣaḍ indriyam ।। brahmavihārasaṃpanno vinītaḥ śraddhayānvitaḥ ।। kiṃcin nimittamātre api vigrahavān asaṃyataḥ ।। atīva roṣasaṃkruṣṭo vicikṣepa yatīn api ।।
sarvatra munibhiś cāpi vijagrāhāsamāhitaḥ ।। subhūtiṃ brāhmaṇaṃ dṛṣṭvā guruś caivam acintayat ।। svasiddhāntasamāno adya krūravāgvigrahotsukaḥ ।। viśārado mahābhijño dharmasaṃyamatatparaḥ ।।
mahotsāho mahāvīraḥ siddhavidyo mahotkaṭaḥ ।। vedasiddhāntayogānāṃ pāragaś ca mahāsudhīḥ ।। kiṃcin nimittamātre api vikruṣṭo atha ruṣāśayaḥ ।। śāpāśaniprahāreṇa loke anarthaṃ kariṣyati ।।
samādhidhyānavaryāsu yojayeyaṃ sa sarvathā ।। śṛṇu vatsa hitaṃ vakṣye tatra bhava samāhitaḥ ।। brāhmaṇo asmīty ahaṃkāro na kartavyaḥ kadācana ।। jīvaś ced brāhmaṇas tāvad vṛthā syād dharmasaṃskṛtaiḥ ।।
tato dharmābhisaṃskāraiḥ sarve syur mānavā dvijāḥ ।। guṇadharmānusāraiś ca devā daityāś ca mānuṣāḥ ।। tamodharmadharā daityā iti siddhāntasaṃmatam ।। caturyonisamudbhūtāḥ ṣaḍgatiṣu bhramanti te ।।
jātiṣv api ca sarvāsu svakarmapariṇāmataḥ ।। te adhobhuvanasaṃjātā vasanti kleśabhāginaḥ ।। te satvā bhūmisaṃjātā vasanti mānuṣādayaḥ ।। te devā nirmalānandā vasanti svargatiṃ gatāḥ ।।
svakṛtaṃ karma bhuñjanto bhrāmyanti tribhavālaye ।। ।। śabdasparśau ca vāyor vai dvau guṇau parikīrtitau ।। śabdasparśarūparasāś catvāry eva samīraṇe ।।
evaṃ militayogaiś ca brahmāṇotpattir ucyate ।। caturaśīti lakṣāś ca proktā vai jīvajātayaḥ ।। miśrato miśrabhuktāra ity uktam avadānikaiḥ ।। sapta vyādhā daśāraṇye mṛgāḥ kāliñjare girau ।।
te api jātāḥ kurukṣetre brāhmaṇā vedapāragāḥ ।। mithyājīvena jīvan yaḥ patito brāhmaṇo hy asau ।। tayaiva sahasuptasya niṣkṛtir nopalabhyate ।। jīvamāno bhavec chūdro mṛtaś ca sa prajāyate ।।
śūdrāt pratigrahagrāhī brāhmaṇo jāyate kharaḥ ।। śvānaḥ saptati janmāni ity evaṃ manur abravīt ।। brāhmaṇo adattam ādāya babhūva vānarādhipaḥ ।। tataś ca mānavo bhūtvā pāṃśudātā hy abhūc chiśuḥ ।।
tatrāpi buddhanāthāya piṇḍapātaṃ dadau mudā ।। sarvapāramitāḥ pūrya buddho api sa bhaviṣyati ।। tasmāc caivaṃ vijānīyā na jīvo brāhmaṇaḥ khalu ।। jātyā ced brāhmaṇo bhūto vṛthā syāt saṃskṛter vidhiḥ ।।
।। tathā hi mānave dharme manunābhihitaṃ khalu ।। tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।। tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।।
tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।। tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।। tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।। tapasā brāhmaṇo jātas tasmāj jātir akāraṇam ।।
tapasā brāhmaṇā bhūtā brāhmaṇīgarbhasaṃbhavāḥ ।। dharmavṛttipramāṇena sarve syur brāhmaṇā narāḥ ।। dharmakalpavikalpena cāturvarṇyaṃ prakalpitam ।। ekendriyakriyārthāś ca tasmāc chīlaguṇair dvijāḥ ।।
brāhmaṇo api kriyāhīnaḥ śūdrāt pratyavaro bhavet ।। brahmajeṣu hi sarveṣu nareṣu kiṃ viśeṣatā ।। brāhmaṇe cānyajātau vā na viśeṣo asti vai tathā ।। brāhmaṇe cānyajātau vā viśeṣo naiṣa vidyate ।।
bhitvā vā pṛthivīṃ jāto jātavedā yathāraṇeḥ ।। śreṣṭhatve vṛṣalatve ca kiṃ vāsti bhedakāraṇam ।। varṇās sathava cāpy anye kā nu tatra viśeṣatā ।। dehasaṃsthānaliṅgaiś ca narāṇāṃ kiṃ viśeṣatā ।।
mukhādivarṇaśabdaiś ca narāṇāṃ nāsti bhedatā ।। tathaiva narajātīnāṃ naivāsti bhedalakṣaṇam ।। tathā nāsti manuṣyāṇām ākṛter bhedalakṣaṇam ।। mānavānāṃ tathā nāsti saṃsthānaṃ bhinnalakṣaṇam ।।
tathā nāsti manuṣyāṇāṃ varṇākāraviśeṣatā ।। mānavānāṃ tathā nāsti varṇagandhādibhedatā ।। varṇasaṃsthānagandhāś ca narāṇāṃ tu tathā na hi ।। manujānāṃ tathā nāsti māṃsāsthiguṇabhedatā ।।
tathā nāsti manuṣyāṇāṃ ṣaḍindriyaviśeṣatā ।। tathā nāsti manuṣyāṇāṃ saṃsthānavarṇabhedatā ।। tathā nāsti manuṣyāṇāṃ śarīrākārabhedatā ।। ekāṃśato viśeṣo na kuto deheṣu bhedatā ।।
pāṃśupuñjāni saṃpiṇḍya svayaṃ nāmāni kurvate ।। na ca bālasya vacanāt pāṃśavo annā bhavanti hi ।। pāṃśupuñjābhidhānena yogo apy eṣa na vidyate ।। na mukhena na nāsāyā na grīvayā na bāhunā ।।
norubhyām atha jaṅgābhyāṃ pāṇipādanakhair na ca ।। naikā viśeṣatā vāpi manuṣyeṣu na vidyate ।। sāmāny akāraṇaṃ manye kiṃcin na bhedalakṣaṇam ।। vaiśyāḥ śūdrās tathānye api saṃjñāmātre hi kīrtitāḥ ।।
tathaikam anujātānāṃ kiṃ viśeṣatvalakṣaṇam ।। cāturvarṇyam idaṃ loke sarvaṃ hi manusaṃbhavam ।। tathā ca procyate bauddhair avadānārthakovidaiḥ ।। brāhmaṇās te mahāśuddhāś caturbrahmavihāriṇaḥ ।।
ye bhajanti sadā brahma vānaprasthā hi te dvijāḥ ।। mahāyajñasamācārā upādhyāyā hi te dvijāḥ ।। bhikṣāśino vratasthās te bhikṣavo brahmavādinaḥ ।। tapanti puṇyakṣetreṣu mānavās te tapasvinaḥ ।।
satyavāco vratasthā ye ṛṣayas te dvijottamāḥ ।। vācaṃyamāś ca te bhadrā munayaḥ satyavādinaḥ ।। nirmamā nirahaṃkārā yatayo yogino api te ।। te api ca mānavā dhīrāḥ sthaṇḍilā jaṭilās tathā ।।
kāpālikāś ca te vīrāḥ śmaśānavratacāriṇaḥ ।। te hotāraś ca yajvāno vedadharmārthasādhakāḥ ।। satvarakṣāvratācārāḥ kṣatriyās te nṛpā narāḥ ।। rājānas te mahāvīrāḥ sarvadharmābhipālakāḥ ।।
veśayanti prajā dharme vaiśyās te hi narottamāḥ ।। manyante sevayā śuddhiṃ śūdrās te śreṣṭhinas tathā ।। kṛṣikās te narā dhānyaiḥ satvajīvānupoṣakāḥ ।। vaṇikkarmābhisaṃyuktā vaṇijas te mahodyamāḥ ।।
sādhayanti ca ratnāni sārthavāhāś ca te narāḥ ।। śilpinas te tathānye api svarṇakārādayo narāḥ ।। yugāntakālavijñātā gaṇakās te api mānavāḥ ।। bhaiṣajyaṃ ye dadanty eva bhiṣajas te hi vaidyakāḥ ।।
śamayanti ca ye bhūtān bhautikās te api mānavāḥ ।। tat tat karmānuśīlena jātidharmapravṛttikāḥ ।। caṇḍavṛttipracārāś ca caṇḍālā iti te smṛtāḥ ।। te śaivā manujā jñeyāḥ śivadharmānucārataḥ ।।
viṣṇudharmasamācārād vaiṣṇavās te api mānavāḥ ।। brahmadharmasamācārād brāhmaṇās te api mānavāḥ ।। mahāraudrāś ca te khyātā bhairavikāś ca mānavāḥ ।। māheśvarīvratādhārāḥ kālikās te api mānavāḥ ।।
te api ca mānavā bauddhāḥ saṃbodhipadasādhinaḥ ।। te api ca manujā jainā jainadharmānucāraṇāt ।। te api ca mānavāḥ sarve dharmacaryānuvarṇinaḥ ।। prajāpatir hi caikatve nirviśeṣo abhavad yataḥ ।।
brāhmaṇe cānyajātau vā naiṣāṃ kiṃcid viśiṣyate ।। śukraśoṇitasaṃbhūtaṃ yonijaṃ sarvam eva hi ।। brahmajā brāhmaṇā naivaṃ dharmasaṃskārajāḥ khalu ।। brāhmaṇy api tathā caiva brahmajā yadi sāṃpratam ।।
na bhāryā bhaginī yuktā brahmaṇāṃ brahmajā yadi ।। nihīnotkṛṣṭamadhyāś ca satvā nānāśrayāḥ pṛthak ।। vaiśyaśūdre tathānyeṣu samaṃ jñānaṃ pravartate ।। bahavo narā nīcakulaprasūtāḥ svargaṃ gatāḥ śīlam upetya dhīrāḥ ।।
caṇḍālo api hi śīlasthas taṃ devā brāhmaṇaṃ viduḥ ।। sarvabhūtadayā brahma etad brāhmaṇalakṣaṇam ।। sarvabhūtadayā nāsti etac caṇḍālalakṣaṇam ।। maithunaṃ nādhigacchanti te narā brāhmaṇāḥ khalu ।।
jīvamāno bhavec chūdro mṛtaḥ sa śvā prajāyate ।। varjitaḥ pitṛdevaiś ca rauravaṃ so adhigacchati ।। na tv etair hi vinā vipraḥ kiṃ syāt saṃskāramātrataḥ ।। saṃskṛtena dvijo vā cec chūdro api saṃskṛto dvijaḥ ।।
brahmahatyā ca bandhūnāṃ śarīradahanād bhavet ।। tasmād dhi brāhmaṇo naiva dehasaṃskāramātrataḥ ।। brāhmaṇo api bhavec chūdraḥ surayā lavaṇena ca ।। viprāṇāṃ patanaṃ dṛṣṭvā tato māṃsāni varjayet ।।
abhakṣyabhakṣaṇāc caiva brāhmaṇaḥ patito bhavet ।। tasmāj jñānaṃ vinā naiva śarīro brāhmaṇo bhavet ।। samāneṣu ca deheṣu kutrāpy asti viśeṣatā ।। yathā karoti bhāṇdāni mṛttikayaiva bhārgavaḥ ।।
kiṃ tu prakṣiptavastūnāṃ saṃjñayākhyāyate khalu ।। jñānadharmaguṇācārair lakṣyate mānavas tathā ।। jñānavijñānabhedena vartate guṇabhedatā ।। dharmabhedāt tataḥ karmabhedatā saṃpravartate ।।
tathācāraviśeṣeṇa jātibhedāḥ pravartitāḥ ।। sarvajantuśarīreṣu sameṣu kā viśeṣatā ।। jñānavijñānapātratvāt pūjyante nīcajā api ।। paśuvatsa narākāraḥ tataḥ pūjā na cākṛteḥ ।।
jñānena yadi vā vipraḥ śūdro api brāhmaṇo bhavet ।। jñānavantaś ca ye dhīrās te api syur brāhmaṇāḥ khalu ।। karmācārapramāṇena na jñānasya pramāṇatā ।। karmaṇā vai dvijāś caivaṃ sarve syur brāhmaṇāḥ khalu ।।
kṣatriyavaiśyaśūdrāś ca kathaṃ na brāhmaṇā nu te ।। nāpi svācāramātreṇa brāhmaṇāḥ syus tathā narāḥ ।। ye ye svācāravantaś ca te te syur brāhmaṇāḥ kila ।। vratopavāsadharmiṣṭā nīcajā api santi ca ।।
tasmād ācāramātreṇa brāhmaṇā naiva mānuṣāḥ ।। yadi vedair bhaved vipro rākṣaso api dvijaḥ khalu ।। sarve api rākṣasāś caivaṃ vedakarmānucārakāḥ ।। tasmāc ca vedamātreṇa naiva syur brāhmaṇāḥ khalu ।।
cāturvarṇyam idaṃ lokaṃ tīrthikair iti kalpitam ।। svayaṃbhūdehasaṃbhūtaṃ cāturvarṇyam idaṃ khalu ।। ūrubhyāṃ saṃbhavo vaiśyaḥ padbhyāṃ śūdraḥ samudbhavaḥ ।। agamyagamanāc caivaṃ kathaṃ dharmaviśuddhitā ।।
brāhmaṇy api mukhāj jātā svasā bhāryā kathaṃ nanu ।। kṣatriyasya bhaved bhāryā kṣatriyā bhaginī khalu ।। vaiśyasyāpi bhaved bhāryā vaiśyā tu bhaginī viśaḥ ।। śūdrasyāpi bhaved bhāryā śūdrī hi bhaginī khalu ।।
agamyagamanāc caivam adharma eva saṃbhavet ।। dharmakriyāviśeṣāt tu varṇāvasthāḥ pratiṣṭhitāḥ ।। vaiśampāyanam āgamya prāñjaliḥ paryapṛcchata ।। etad icchāmi bho jñātuṃ tadbhavān vyākarotu me ।।
pratyuvāceti kaunteya śṛṇu tat kathyate mayā ।। na hanti sarvabhūtāni prathamaṃ brāhmalakṣaṇam ।। adattaṃ naiva gṛhṇāti dvitīyaṃ brāhmalakṣaṇam ।। muktaś carati yo nityaṃ tṛtīyaṃ brāhmalakṣaṇam ।।
maithunaṃ hi sadā tyaktaṃ caturthaṃ brāhmalakṣaṇam ।। sarvabhūtadayā śaucaṃ tapaḥ śaucaṃ ca pañcamam ।। tam ahaṃ brāhmaṇaṃ brūyāṃ śeṣāḥ śudrā yudhiṣṭhira ।। cāṇḍālo api hi vṛttastho brāhmaṇaḥ sa yudhiṣṭhira ।।
phaleṣv anabhilipsātha brāhmaṇaḥ syād yudhiṣṭhira ।। karmakriyāviśeṣeṇa cāturvarṇyaṃ pratiṣṭhitam ।। ekendriyakriyārthāś ca tasmāc chīlaguṇair dvijāḥ ।। brāhmaṇo api kriyāhīnaḥ śūdrāt pratyavaro bhavet ।।
tasmai dānaṃ pradātavyam aprameyaṃ yudhiṣṭhira ।। guṇavidyānidhir vidvān brāhmaṇo brahmacārāṇāt ।। ahorātraṃ caren muktas taṃ devā brāhmaṇaṃ viduḥ ।। kāmeṣv asaktāḥ kaunteya brāhmaṇās te yudhiṣṭhira ।।
rāgadveṣanivṛttiś ca etad brāhmaṇalakṣaṇam ।। vidyā vidyānamādhītyam etad brāhmaṇalakṣaṇam ।। nādhitya caturo vedān sarvāśī sarvavikrayī ।। na tāṃ kratusahasreṇa prāpnuvanti yudhiṣṭhira ।।
yuktaś carati dharmaṃ yo taṃ devā brāhmaṇaṃ viduḥ ।। kāyena manasā vācā brahma saṃpadyate tadā ।। tena saṃpadyate brahma tasmān maitrīṃ vibhāvaya ।। tena saṃpadyate brahma tasmāt kāruṇiko bhava ।।
tasya saṃjāyate brahma tal loke modavāṃś cara ।। tasya saṃjāyate brahma tad upekṣāyutaś cara ।। jñātvā lokahitārthena cara brahmavihāriṇam ।। brahmapraṇidhim ālambya sthirībhava samādhiṣu ।।
pañcābhijñapadaprāpto brahmalokam avāpnuyāḥ ।। tatheti pratisaṃśrutya dhyānacaryām upāśrayat ।। sarvendriyavinirgatyā vyaharaddhyānatatparaḥ ।। karmādhānabalābhyāsān naiva śāntim upāyayau ।।
sā subhūtiṃ mahākrodhaṃ dṛṣṭvaivaṃ samacintayat ।। sarvamantravidhānajñaḥ sutīkṣṇakrodhabāhulaḥ ।। dhakṣyati parvatāṃś cāpi sapakṣijantumānavān ।। jñānavijñānadharmeṣu viśeṣaṃ nādhigacchati ।।
kṣipraṃ kleśān vinirjitya bodhicittaṃ ca lapsyati ।। bodhisatvo mahāvijño bhaviṣyati na saṃśayaḥ ।। taṃ subhūtiṃ samāgamya jagādaivaṃ puraḥ sthitā ।। dhanyo asi tvaṃ mahādhīra maharṣir dvijasattama ।।
niścittaḥ pratisaṃlīnaḥ kāṣṭhapāṣāṇavadvṛthā ।। buddhasya vacanaṃ śrutvā cara saṃbodhisatpathe ।। munīndraḥ śrīdhanaḥ śāstā sarvadharmānupālakaḥ ।। ṣaṭ ca pāramitāḥ khyātāḥ paratreha śivaṃkarāḥ ।।
sarvasatvahitārthena saṃbodhiguṇasādhakāḥ ।। triratnaśaraṇaṃ gatvā cara brahman vratottamam ।। sākṣād arhatpadaṃ prāpya nirvṛtisukham āpnuyāḥ ।। tathānumoditaḥ prāha tāṃ devatāṃ puraḥ sthitām ।।
triratnasamayaṃ prāptum iccāmi tvatprasādataḥ ।। saṃbuddhaṃ darśaya kṣipraṃ taddharmeṣu niveśaya ।। tathāśu kṛpayā nītvā māṃ vihāre praveśaya ।। vijñāya bodhimārgeṣu cittaṃ tathānumoditam ।।
ṛddhyā sākāśamārgeṇa nināya jinamandiram ।। bhagavantaṃ mahāsaumyaṃ lakṣaṇaiḥ samalaṃkṛtam ।। sahasrakiraṇādhikyaṃ ratnāṅgam iva jaṅgamam ।। sarvadevādhipaṃ samyaksaṃbodhiguṇasāgaram ।।
ādhāto yaś ca satveṣu sa prativigato apy abhūt ।। natvā pādau muner dharmaṃ śrotuṃ tasthau mudāḥ puraḥ ।। jñātvāryasatyadharmāṇi dideśaivaṃ savistaram ।। yadi te dharmavāñchāsti saṃbodhipadasādhane ।।
dharmamātā yato maitrī tan na tyājyā kadācana ।। kāruṇyād vardhate dharmas ta kāruṇyaṃ sadā kuru ।। muditāṃ hi samālambya bodhipadam avāpnuyāḥ ।। upekṣāto labhet saukhyaṃ tadupekṣāṃ sadā bhaja ।।
tatprāptyai sādhyatāṃ yatnāc caturbrahmavihāratā ।। kleśasaṃghān vinirjitya buddhadharmaṃ samaikṣata ।। vidārya jñānavajreṇa saṃsāraratiniḥspṛhaḥ ।। dṛṣṭasatyo atha saṃbuddhaṃ natvā caiva kṛtāñjaliḥ ।।
namas te bhagavan nātha sarvasatvānupālaka ।। tathā dharme ca saṃgheṣu saṃbodhiguṇaprāptaye ।। brahmacaryaṃ cariṣye ahaṃ tvadājñāṃ śirasā vahan ।। ehi bhikṣo carasveti pravadaṃs taṃ samagrahīt ।।
sadyaḥ praśāntendriya eva tasthau bhikṣuḥ subhūtiḥ sugataprabhāvāt ।। vyāyacchamānaḥ khalu bodhimārge saṃbuddhadharme ghaṭamāna eva ।। kleśāṃś ca sarvān pravihāya saṃyataḥ sākṣāc ca so arhann abhavan maharddhikaḥ ।। bhindann avidyādrim ivāṇḍakośaṃ prāpadabhijñāḥ pratisaṃvidaś ca ।।
pūjyaś ca mānyo abhivādanīyo babhūva sa brahmavihāracārī ।। kutaś cyuto aham āyātaḥ kutra kena ca karmaṇā ।। nāgayonisamutpannas tataś cyutvāham āgataḥ ।। tenaiva hetunā cāhaṃ mahadvyasanam āptavān ।।
yasyaiva hetunā lokā bhramanti narakeṣu te ।। saṅgād dhi jāyate māyā māyāyāṃ jāyate ratiḥ ।। mūḍhasya drūyate cittaṃ sveṣṭakāryopaghātataḥ ।। krodhānalasamuddīpto dahate sa parān api ।।
tāvat kiṃ tapasāpy etan nirarthaṃ duḥkhahetave ।। kṛtaṃ kalpasahasrair yad dahet krodhānalaḥ kṣaṇāt ।। ekānte hi vaseyaṃ ca viviktāraṇyagocare ।। yatīn dṛṣṭvānumodaṃ ca kurvan mṛtyum avāptavān ।।
sarvakleśān vinirjitya brahmacārī bhavāmy aham ।। kenacit kleśitaś cāhaṃ bhraṣṭim evam avāpnuyām ।। vivikte araṇyavāse sa niḥsaṅgo nyavasat sudhīḥ ।। phalamūlāmbusaṃtuṣṭo brahmacārī mumoda saḥ ।।
bhikṣāhetor vihartuṃ vā sa sakāmo abhavad yadā ।। aho deśeṣu sarvatra bhavanti nirguṇā janāḥ ।। tat kathaṃ saṃcariṣye atra bhikṣāhetoḥ kule kule ।। yad dhetor janāś caivaṃ bhramanti durgatiṣv api ।।
tad ahaṃ sarvasatveṣu kuntapipīlikādiṣu ।। tam eva dharmam ādhyāya yatir mokṣam avāpnuyāt ।। vivikte araṇyadeśe api nyavasad dhyānasaṃrataḥ ।। ity evaṃ cintayāmāsa lokānugrahakāraṇāt ।।
kiyat kālaṃ ca jīveyaṃ kāṣṭhapāṣāṇavat sthitaḥ ।। sukhaṃ labdhvāpi kiṃ sāraṃ satvānugrahaṇaṃ vinā ।। ṛddhiṃ pradarśya saṃbodhau sthāpayiṣye mahajjanān ।। satvānāṃ vinayārthena prātihāryam adarśayat ।।
pañca kulaśatāny atra prasasrire samantataḥ ।। itas tataḥ samudbhrāntāḥ subhūteḥ śaraṇaṃ yayuḥ ।। sarve nāgāḥ suparṇebhyaḥ paritrātāś ca sarvataḥ ।। svarddhivalaprabhāveṇa mahān nāgo vinirmitaḥ ।।
abhidrutāni nāgena samantata itas tataḥ ।। itas tataḥ samudbhrāntāḥ subhūteḥ śaraṇaṃ yayuḥ ।। svarddhibalaprabhāveṇa samāśvāsya surakṣitāḥ ।। dṛṣṭvā sarve janaughās te saharṣādbhutam āyayuḥ ।।
yenaite rakṣitāḥ sarve nāgāś ca garuḍā api ।। saddharme vinayārthena maitrīdharmam upādiśat ।। yadi me śaraṇaṃ yātha ramadhvaṃ maitramānasāḥ ।। ye ete duḥkhino loke sarve te krodhino narāḥ ।।
yāvac citte sthitaṃ krodhaṃ tāvan maitrī na bhāvyate ।। tasmān maitrī prayatnena bhāvanīyā sadādarāt ।। na nidrāṃ na dhṛtiṃ yāti dveṣaśalye hṛdi sthite ।। krodham ekaṃ tu yo hanyāt tena sarve dviṣo hatāḥ ।।
dahaty ātmānam evādau parān dhakṣyati vā na vā ।। bandho dharmārthakāmānāṃ tasmāt krodho nivāryatām ।। supto api na labhen nidrāṃ krodhaparyākulo naraḥ ।। dagdhā janapadāś cāpi krodhāc chāpahutāśanaiḥ ।।
dadāha brahmajaṃ kāmaṃ tac ca krodhaprabhāvataḥ ।। mṛtyuṃ yānti janaiḥ sārdhaṃ tac cāpi krodhabhāvataḥ ।। satyadharmāv anādṛtya ghnanti krodhād anāryakāḥ ।। tān api saṃmukhaṃ ghnanti durvāgbāṇai ruṣā khalāḥ ।।
svātmajān nirdayā ghnanti tac ca krodhaprabhāvataḥ ।। avigaṇayya pāpāni ghnanti krodhaprabhāvataḥ ।। anādṛtya bhayaṃ pāpā ghnanti krodhoddhatā narāḥ ।। nirdayas tāḍayan hanti tasmāt krodho mahāripuḥ ।।
vigṛhṇanti mahākruddhās tasmāt krodho mahāripuḥ ।। tāḍitā tyajyate bhartrā krodhāt tato mahad bhayaṃ ।। kuladharmam anādṛtya jahāti krodhatas tathā ।। tān api saṃmukhaṃ duṣṭās tāḍayanti ruṣānvitāḥ ।।
so api śakalito rājñā svayam evāsinā ruṣā ।। anyonyaṃ vigrahaṃ kṛtvā pramathnanti ruṣākulāḥ ।। vasanti narake ghore te api sarve ruṣānvitāḥ ।। patanti narake ghore te api sarve ruṣāśrayāt ।।
krodhena hiṃsyate jantus tasmāt krodho mahāripuḥ ।। tenaiva pātakenaiva bhrāntacitto abhavac chivaḥ ।। tatpāpakarmaṇā hy eva śivo apy abhūd digambaraḥ ।। krodhena tyajyate satyaṃ tasmāt krodho mahāripuḥ ।।
tāni sarvāṇi duṣṭāni krodhacittodbhavāni ca ।। tasmāt krodhavināśāya prayatadhvaṃ samāhitāḥ ।। tena sarve jitā duṣṭā śatravo duḥkhadāyakāḥ ।। sa sādhupuruṣaś cāpi naiva viśvasyate janaiḥ ।।
sa vidvān api nāsevyo yathā vṛkṣo ahiveṣṭitaḥ ।। krodhavān na vibhāty eva ahipūrṇo yathā hradaḥ ।। astramantrādyabhijño api krodhavān naiva sevyatām ।। krodhavān hīyate mitraiḥ krodhavān paribhūyate ।।
tasmāt krodhavināśāya prayatadhvaṃ samāhitāḥ ।। vikīrṇaḥ kliśyate māraiḥ kleśito adhairyatāṃ vrajet ।। duṣṭamitropadeśena kupathe carate kudhīḥ ।। viparītānubodhena bhaved āryāpavādakaḥ ।।
ityādipātakaṃ kṛtvā pañcānantaryam āpnuyāt ।। narakān narakaṃ gatvā mahāduḥkham avāpnuyāt ।। narakebhyas tam uddhartuṃ jino api naiva śaknuyāt ।। tāni sarvāṇi jānīdhvaṃ krodhacittodbhavāni hi ।।
dharmāṇāṃ tu kṣamā mūlaṃ yataḥ saukhyaṃ pravartate ।। maitrīcittaṃ samālambya viharadhvaṃ yathāsukhaṃ ।। kasya nāma bhavec cittam adharmapraṇayāśivam ।। manovākkāyavispandaiḥ svajane api yathā jane ।।
suvṛṣṭir iva śasyāni guṇān sā hi prasūyate ।। vivṛddhā tasyaivaṃ parahitarucir maitryanugatā pradānakṣāntyādīñ janayati guṇān kīrtyanusṛtān ।। na saṃrambhakṣobhaḥ prabhavati dayādhīr ahṛdaye na kopāgniś citte jvalayati hi dayātoyaśiśire ।। tasmāt putra ivātmanīva ca dayāṃ nītvā prakarṣaṃ jane sanmaitryā viharanta eva muditāṃ prodbhāvayadhvaṃ sadā ।।
tasmād dayāṃ hṛdi sthāpya maitrī loke prasāryatām ।। viśvasyate sadā sadbhir bāndhavaiḥ svajanair janaiḥ ।। maitrīmān sanmatir bandhur lokānāṃ jagatām api ।। maitrīmāñ jagatāṃ mitro maitrīmāñ jagatāṃ sakhā ।।
tatra tatraiva sarvatra pūjyate svajanair yathā ।। so api śāstā vibhāty evaṃ maitryā saṃskārayañ jagat ।। pūjyante satvalokaiś ca te api maitryāḥ prabhāvataḥ ।। sarvasatvahitārthasthās te api maitrīpracāriṇaḥ ।।
pāti snehopacāreṇa tac ca maitrīprabhāvataḥ ।। pāti snehopacāreṇa tac ca maitrīprabhāvataḥ ।। durjanān mardayan sarvān tac ca maitrīprabhāvataḥ ।। prabodhya bālakāñ chāsti tac ca maitrīprabhāvataḥ ।।
prarakṣanti svapakṣāṃś ca tac ca maitrīprabhāvataḥ ।। satvān pāti dadad dānaṃ tac ca maitrīprabhāvataḥ ।। anapekṣya svajīve api tac ca maitrīprabhāvataḥ ।। anuśocan muhuś cāpi tac ca maitrīprabhāvataḥ ।।
svasutān snehataḥ pānti tac ca maitrīprabhāvataḥ ।। svabandhūn snehataḥ pānti tad dhi maitrīprabhāvataḥ ।। bāndhavāṃs te api rakṣanti tad dhi maitrīprabhāvataḥ ।। arthibhyaḥ prārthitaṃ vastu tad dhi maitrīprabhāvataḥ ।।
pālayanti mahāsnehāt tac ca maitrīprabhāvataḥ ।। patir mitraḥ suhṛd bandhus tasmān maitrī prasādhyatām ।। na muditā na copekṣā tasmān maitrī pradhīyatām ।। trailokyabhartṛkā nāthāḥ sarvasatvānupālakāḥ ।।
kalpakoṭisahasrāṇi taptvāpi duṣkaraṃ tapaḥ ।। etā vinā na sidhyanti sarvapāramitāratāḥ ।। tāni sarvāṇi jānīta maitrīmūlodbhavāni hi ।। maitrīṃ citte samādhāya kurudhvaṃ prāṇiṣu kṣamām ।।
yāvajjīvaṃ sukhaṃ bhuktvā saukhāvatīṃ gamiṣyatha ।। vairānuśayatāṃ tyaktvā babhūvur maitricāriṇaḥ ।। dharmānumodanaṃ kṛtvā babhūvur maitricāriṇaḥ ।। maitrīdharmopadeśena vinītā dharmasatpathe ।।
bhikṣūn āmantrayāmāsa saṃvṛticārakān api ।। yenaite garuḍā nāgā vinītā dharmasatpathe ।। subhūtiḥ kulaputro ayam agro araṇāvihāriṇām ।। nirdiṣṭaḥ sarvabhikṣūṇām agro araṇāvihāriṇām ।।
cchetāraṃ saṃśayānāṃ taṃ papracchur evam īśvaram ।। nirdiṣṭo bhavatā yena jyeṣṭho araṇāvihāriṇām ।। śṛṇudhvaṃ bhikṣavaḥ sarve tat kṛtaṃ yat subhūtinā ।। yenaiva yat kṛtaṃ karma tenaiva tat prabhujyate ।।
varṣasahasram āyuṣ ca viṃśatiguṇitaṃ yadā ।। vidyācaraṇasaṃpannaḥ sugato lokavij jinaḥ ।। sarvajño bhagavān nāthaḥ ṣaḍabhijño munīśvaraḥ ।। vyaharat sarvasatvānām saddharmaṃ samupādiśan ।।
ayaṃ pravrajito bhūtvā mahādātāpy abhūt tadā ।। praṇidhānaṃ tathā cāyam akarod brahmavittamaḥ ।। yo asau bhagavatānena kāśyapena sutāyinā ।। māṇava tvaṃ prajānāṃ tu yadā varṣaśatāyuṣi ।।
saṃbuddho bhagāvan nāthas tathāgato bhaviṣyasi ।। arhatām agrasaṃprāpto bhūtvāraṇāvihāriṇām ।। arhatāṃ jyeṣṭhatāṃ prāptas tathāraṇāvihāriṇām ।। yenaiva nāgayonau ca samutpanno babhūva saḥ ।।
śaikṣāśaikṣagāṇānāṃ ca bhikṣūṇāṃ brahmacāriṇām ।। cittāni saṃpradūṣyaiva vikalāni kṛtāny api ।। tenaiva pātakenaivaṃ pañca janmaśatāny api ।। yac cānena punas tatra pravrajya buddhaśāsane ।।
tenedānīṃ tathārhatvaṃ prāpya sākṣātkṛtaṃ mudā ।। iti hi bhikṣavo yūyaṃ jānīdhvaṃ karmatāphalam ।। sāmagrīṃ prāpya kālaṃ ca phalanti khalu dehināṃ ।। udakaiḥ klidyate naiva bhūmiṣv api na naśyati ।।
miśrataiva tu miśrāṇāṃ ṣaḍgatau bhujyate dhruvam ।। yatitavyaṃ śubheṣv eva karmasu sukhavāñchibhiḥ ।। buddhavacomṛtaṃ pītvā nanandur anumoditāḥ ।। iti matvā tvayā rājan parātmaśubhavāñchinā ।।
maitrīṃ bhāvaya satveṣu svātmajeṣu yathā sadā ।। tatheti nṛparājaḥ sa nananda sasabhājanaḥ ।। te lokā maitracittās tribhuvanasukhadāḥ kṣāntisaurabhyayuktāḥ yātāḥ saukhāvatīṃ te apy amitarucimuner dharmam ārādhayanti ।।
