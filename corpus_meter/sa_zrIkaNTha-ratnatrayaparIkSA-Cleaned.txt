nirvikalpārthasaṃvittis tadaharjātabālavat ।। yathārthasthitiṃ gṛḥnāti svasaṃvedyā cidavyayā ।। na hi kartṛbahutvasya gamakaṃ vidyate kvacit ।। saṃvidāśrayate śaśvacchivaṃ paramakāraṇam ।।
tirodadhāti bhagavān anugṛhṇāti cātmanaḥ ।। śaktimānāhitodyogaḥ pravṛttaśceti deśikaiḥ ।। tasmin mukulitevāste kriyākhyā śaktiraiśvarī ।। patyurāviṣkarotyuccaiḥ paramānandasantatim ।।
pañcamantratanuḥ śrīmān devaḥ sakalaniṣkalaḥ ।। mantrarūpā hi tacchaktir mananatrāṇadharmiṇī ।। sā bhāti binduśāntyādipañcādhiṣṭheyagocarā ।। bhogasthānāni pañcaiṣāṃ bindusaṃjñā śiveritā ।।
tadgocarā parā mūrtir aparā kāryagocarā ।। saiva mūrtiḥ kriyābhedāt sādākhyā tanurucyate ।। bhogasthānaniviṣṭānāṃ nirmalānāṃ śivātmanām ।। tatkāryaṃ sā kriyāśaktir asāvapi tathocyate ।।
icchājñānakriyādyā yat prabhavāḥ karyayonayaḥ ।। dehendriyādayaḥ śuddhāḥ subhagāḥ svadhikārakāḥ ।। tatsambandhācchivo aśeṣakāryāṇāṃ heturucyate ।। patikṛtyādhikāreṣu sadāśivamaheśvarān ।।
pañcasrotomukhaḥ śāntaḥ prabhuḥ śaktiśirāḥ śivaḥ ।। īśānamūrdhā puṃvaktro daśadigbāhumaṇḍalaḥ ।। pravṛttimānayaṃ devaḥ sakalaḥ sarvapāvanaḥ ।। layabhogādhikāreṣu tau hi cinmātrarūpiṇau ।।
svātmani pravilīyante tenaike samadhiṣṭhitāḥ ।। samāliṅgya mahāmāyām apare vṛttiśālinīm ।। tathā pare paravyoma samāśrityādhvamūrdhani ।। tataḥ sa teṣāmeko api layabhogādhikāriṇām ।।
tadvadeva matā śaktis tayorbhedo adhunocyate ।। samāśrayaḥ sa vijñeyaḥ svaniṣṭhaghanacinmayaḥ ।। paramaiśvaryarūpā ca mahimā vyāptirūpiṇī ।। parānapekṣaṃ rūpaṃ yad vijñānaṃ śivasaṃjñitam ।।
ākāradvayasaṃvittir aśeṣasyāpi vastunaḥ ।। nityoditānavacchinnavibhūterīśvarasya yat ।। tena śaktiḥ parāpekṣaṃ rūpamīśasya yujyate ।। kāryaputrādirūpāṇām anyathaivopalambhanāt ।।
virodhenātha paryāyān naivamapyavināśi yat ।। avināśi ca tannityam iṣṭamasmād vibhidyate ।। kāryopādhivaśācchaktisaṃjñā syādapi cidghane ।। karaṇena vinā kāryaṃ kurvan kartā bhavediti ।।
kiñcāgṛhītamapi ced upādhiḥ kāryamīśituḥ ।। kathaṃ vāyaṃ bhavet kartā kāraṇajñānavarjitaḥ ।। idamandhatamaḥ kṛtsnam antarā śaktidīpikām ।। ananyāpi tathā śambhor vibhinnā śaktiriṣyate ।।
ananyāpi vibhinnātaḥ śambhoḥ sā samavāyinī ।। na hyeṣa bhagavān śaktyā svātmano atyantabhinnayā ।। anāhitaviśeṣo api heturdevo maheśvaraḥ ।। binduḥ pralīnakāryo asau śivaśaktisamīritaḥ ।।
cirasthāpitaviśvo hi bindureva pravartate ।। mṛdavasthāntarāpattyā pūrvamevamasāvapi ।। kāryabhedāya ghaṭate nirvikāre api śūlini ।। bhedasaṃśoṣacūrṇānām avasthābhedajanmanām ।।
kartā sargādikāryāṇām avikāro api śaktimān ।। dravatāśuṣkatāhetur avikāro api bhāskaraḥ ।। aṇūnāmavikāro api bandhamokṣakaraḥ śivaḥ ।। karmaṇoḥ sannipātena śaivī śaktiḥ patatyaṇoḥ ।।
praśāntamalakāluṣyam ātmānamanupaśyati ।। nityanaimittikenaiṣa karmaṇā vartate param ।। prāpnuvanti mahādhīrā dhanyā hi śivayoginaḥ ।। nadījanapadodyānapurādīni svalīlayā ।।
namastathāvidhāyāsmadgurave śivatejase ।। rāmakaṇṭhakṛtālokanirmalīkṛtacetasā ।। ratnatrayaparīkṣārthaḥ saṃkṣepeṇa prakāśitaḥ ।।
