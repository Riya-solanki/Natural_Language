jhaṭiti prativaktavyaṃ sabhāsu vijigīṣubhiḥ ।। hāso rājñaḥ stavaś ceti pañcaite jayahetavas ।। paṇḍito yadi tatraiva pakṣapāto adhiropyatām ।। ātmotkarṣo nigamanam anumāneṣv ayaṃ vidhis ।।
jigīṣunā hriyaṃ tyaktvā kāryaḥ kolāhalo mahān ।। evam ca tathyavyutpattir āyuṣo ante bhaven na vā ।। na stauti cet svayaṃ ca svaṃ kadā tasyāstu nirvṛtis ।। iti pāṭhayatāṃ granthe kāṭhinyaṃ kutra vartate ।।
trayaḥ śiṣyaguṇā hy ete mūrkhācāryasya bhāgyajās ।। māntrikās tu bhaviṣyāmo yogino yatayo api vā ।। vilambe karmabāhulyaṃ vikhyāpyāvāpyate dhanam ।। anugrahāyate yeṣāṃ te dhanyāḥ khalu māntrikās ।।
tāvan māhātmyarūpeṇa paryavasyati māntrike ।। grahacāraparijñānaṃ teṣām āvaśyakaṃ yatas ।। garbhapraśneṣu kathayan daivajño vijayī bhavet ।। jīvanto bahumanyante mṛtāḥ prakṣyanti kaṃ punas ।।
sarvaṃ vyāmiśram iva ca vaktavyaṃ daivacintakais ।। bruvāṇāḥ sarvathā grāhyā lokair jyautiṣikā janās ।। daivajñānām upālambho nityaḥ kāryaviparyaye ।। deśo hy aratnimātre api nāsti daivajñavarjitas ।।
tritayaṃ ye vijānanti te vācaspatayaḥ svayam ।। nisargaśatravaḥ sṛṣṭā daivajñānām amī trayas ।। kātarā dīrgharogāś ca bhiṣajāṃ bhāgyahetavas ।। naiścintyān nādime dānaṃ nairāśyād eva nāntime ।।
ārogyaṃ vaidyamāhātmyād anyathātvam apathyatas ।। sarvam apy upadekṣyanti rogiṇaḥ sadane striyas ।। rogatattveṣu śanakair vyutpadyante cikitsakās ।। bahumānārtham ante ca jihīrṣanti cikitsakās ।।
dāyādāḥ saṃprarohanti daivajñā māntrikā api ।। śanair anādaras śāntau snāto vaidyaṃ na paśyati ।। ekaikam arthalābhāya dvitriyogas tu durlabhas ।। satyaṃ vaiduṣyam ity eṣa yogo dāridryakārakas ।।
sarvaṃ mārjanti kavayaḥ śālīnāṃ muṣṭikiṃkarās ।। kiṃ cid astuvatām teṣāṃ jihvā phuraphurāyate ।। kītaḥ kaś cid alir nāma kiyatī tatra varṇanā ।। antato annāya vastrāya tāmbūlāya ca kalpate ।।
vikrīyate katipayair vṛthānyair viniyujyate ।। labdhvāpi kāmadhenuṃ te lāṅgale viniyuñjate ।। kathaṃ tarantu kavayaḥ kāmapāramyavādinas ।। samādher api taj jyāyāḥ śaṃkaro yadi varṇyate ।।
prāṇinām kalinā sṛṣṭāḥ pañca prāṇā ime apare ।। ajñātā eva gṛhiṇāṃ bhakṣayanty ākhuvad gṛhe ।। śvaśurasya balaṃ bhāryā svayam evātither balam ।। prabudhyamāne sāralyaṃ prabuddhe asmin palāyanam ।।
śvaśuras tu pravāsīti jāmātur bhāgyadhoraṇī ।। sakṛd āgatya gacchantī kanyā nirmārṣṭi mandiram ।। patipakṣyāṃs tu bahvāśān kṣīrapāṃs taskarān api ।। aśrāntakalaho nāma yogo ayaṃ gṛhamedhinām ।।
jīrṇau ca mātāpitarāv ekaikaṃ narakādhikam ।। aho mahad idaṃ bhūtam uttamarṇābhiśābdhitam ।। na kālaniyamaḥ kaś cid uttamārṇasya vidyate ।। uttamārṇam avekṣyaiva tathāpy udvejite manas ।।
mṛtyau mṛtyuñjayadhyānaṃ dāridrye tu na kiṃ cana ।। dīpayaty udare vahniṃ dāridryaṃ paramauṣadham ।। protsāhayati yācñāyāṃ rājājñeva daridratā ।। ākiṃcanyabalāḍhyānām antato aśmāpi jīryati ।।
dainyaṃ rājyād api jyāyo yadi tattvaṃ prabudhyate ।। protsāhayati dāyādāṃl lākṣmīḥ kiṃ cid upasthitā ।। ta eva tu viḍambyante śriyā kimcidupekṣitās ।। kīṭabuddhir manuṣyeṣu nūtanāyāḥ śriyaḥ phalam ।।
viḍambanāni dhanikāḥ stotrāṇīty eva manvate ।। svairaṃ hasanti pārśvasthā bālonmattapiśācavat ।। na bibheti na jihreti tathāpi dhaniko janas ।। lakṣmīmadas tu mūrkhāṇām ādeham anuvartate ।।
vikāras tu tadārabdho nityaṃ laśunagandhavat ।। lakṣmīmadas tu sarvāṅge putradāramukheṣv api ।। kule apy avataraty eṣa kuṣṭhāpasmāravat katham ।। vismārayanti jātiṃ svāṃ varāṭāḥ pañcaṣā kare ।।
yāvad yācakasādharmyaṃ tāval loko na mṛṣyati ।। kathaṃ te taṃ laghūkartuṃ yatante aparathā svatas ।। dviguṇaṃ triguṇaṃ caiva kṛtānto lālayiṣyati ।। rājakarṇe japaḥ sadyaḥ sarvakarmavināśanas ।।
parārtheṣu pravartante śaṭhāḥ santaś ca tulyavat ।। khalo gṛhasamīpasthaḥ sadyo anarthāya dehinām ।। japayajñaś ca yajñeṣu rocate lobhaśālinām ।। kiṃ prakṣyatīti lubdho api tāvad udvijate tatas ।।
bhikṣākabalam ekaikaṃ ye hi paśyanti meruvat ।। dhanalubdhaḥ piśācas tu na kasmai cana ditsate ।। kartṛkarmavyatīhārād aho nimnonnataṃ kiyat ।। niścityaivaṃ svayam api bhuṅkte lubdhaḥ kathaṃ cana ।।
vicitrān ullikhed vighnāṃs tiṣṭhāsur atithiś ciram ।। sahasraṃ dāmbhike loke śrotriye tu na kimcana ।। tāvad arthāḥ prasiddhyanti yāvac cāpalam āvṛtam ।। vaktavyaṃ kiṃ cid ity uktaṃ vastutas tulasī parā ।।
viśvasaṃmohinī vittadāyinīti guṇadvayam ।। maunam ekāsikā ceti mūrkhasaṃjīvanāni ṣaṭ ।। adhyāpanāvṛttayaś ca kīrtanīyā dhanārthibhis ।। deśadharmas tv anācāre pṛcchatāṃ siddham uttaram ।।
tathā dharmaṃ caret sarvaṃ na vṛthā kiṃ cid ācaret ।। sarvaṃ brahmeti vādaś ca sadyaspratyayahetavas ।। satataṃ śuciveṣaś cety etad dambhasya jīvitam ।। tāvatsaṃkṣipyate sarvaṃ yāvad draṣṭā na vidyate ।।
kiṃ tasya sādhanair anyaiḥ kiṃkarāḥ sarvapārthivās ।। durjanānām ato nyāyyaṃ dūrād eva visarjanam ।। saṃpūrṇadānaṃ prakṛtir virāmo vairakāraṇam ।। apatyasaṃbandhavidhiḥ svānarthāyaiva kevalam ।।
milanti yadi catvāri tad diśe api namo namas ।। paramarmāsu vācaṃ ca khalānām asṛjad vidhis ।। kalinā daśamāṃśena sarvaḥ kālo api dāruṇas ।। yatra svapratibhā mānaṃ tasmai śrīkalaye namas ।।
kālakālaṃ prapannānāṃ kālaḥ kiṃ naḥ kariṣyati? ।। racitaṃ viduṣāṃ prītyai rājāsthānānumodanam ।।
