itthaṃ daitya-cayaḥ samudra-mathane yena kṣaṇād vañcitas tasmai sādaram oṃ-namo astu kapaṭa-strī-rūpiṇe viṣṇave ।। citraṃ durbhaga-rūpa-varṇana-mahā-pāṇḍitya-vidyāya te vyājānāṃ prathamāya garbha-gaṇikā-rāgāya tubhyaṃ namaḥ ।। āścaryaṃ ca śarīra-dānam api yat kāryaṃ vikāraṃ vinā tad veśyaia tapasvinī kali-yuge loko api tad-bhāvitaḥ ।। raktaṃ pātum apīate ca sakalaṃ tārkṣyasya mūrtir yathā veśyā kiṃ tu bhujaṅga-bhakṣaṇa-vidhau nāyāi viśrāmyati ।।
veśyānāṃ śaśa-śṛṅga-bhaṅgi-sadṛśaṃ vāllabhyam āsādya ye sarvasva-kṣapaṇāya kevalam aho mūrkhāḥ sukhenāate ।। nimbe sādhaya mādhurīṃ surabhitām ādau rasoe kuru premāṇaṃ gaṇikā-jane api caturaḥ paścāt sakhe drakṣyasi ।। agniś ced atiśītalaḥ khala-janaḥ sarvoakārī sa ced āyuṣyaṃ yadi vā bhaviṣyati viṣaṃ veśyāpi tad-rāgiṇī ।। bhītaḥ śānta-matir vitaḥ sa-vinayaḥ karṇejapo dhārmikaḥ kiṃ kutrāi kadāi ko api bhavitā veśyā-jano rāgavān ।।
dharmo nāti ca nāstikasya vibhavo nāti pramattāmanaḥ snehānāṃ kaṇikāi nāti gaṇikā-lokasya ca prāyaśaḥ ।। kṣīṇe arthe vidhureṣu bandhuṣu dṛḍhe lokāavāde śanair jñeyā kāmi-janeṣu saia gaṇikā-vargasya naisargikī ।। saṃbhūyāi ca yat surāura-balair unmathya pāthonidher veśyā paśyata līlayaia kurute tat tad gṛhe kāminām ।। vācāṭasya vidagdha-parṣad avadhir laulyaṃ vratasyāadhir bandho viddhi dhanāasānam avadhir veśyāurāgasya ca ।।
dāsī-putram ivodhataḥ kula-dharo nīco dhanīāthinaṃ paryante tṛṇa-tulyam eva gaṇikā niḥsvaṃ janaṃ paśyanti ।। bandī tyāgibhir akṣarair janayitā lābhāirekair vaṇig rogātair bhiṣag arbhakaiś ca gaṇikā garbheśvarais tuṣyati ।। grāmīṇair viṭa-peṭakaḥ parijanaiḥ svāmī śiśuḥ kandukaiḥ śrīputrais taruṇais tathā nava-navair vārāṅganā krīḍati ।। yogī bhoga-viśeṣa-lampaṭatayā rogī kupathyāśanaī rāgī catvara-kāminī-paricayaiḥ kṣiprād adhaḥ kṣipyate ।।
svas-bhānur vidhu-maṇḍalasya vaḍavā-vakro nidher ambhasā māsāṃ kāmuka-saṃpadāṃ ca gaṇikā-kṣudraḥ kīloadravaḥ ।। aiśvaryasya vikāratā mukharatā vidyā-prakarṣasya ca bhrātar-vibhrama-narmaṇaś ca gaṇikā sarvāmanā dūṣaṇam ।। dharmaṃ prāṇivadhena vāñchati dhanaṃ dyūta-prasādeba yo veśyābhiś ca vilāsam ātmani narāāro asti gaur eva saḥ ।। kaṣṭaṃ nāhva-samānam indriya-samo nāriḥ para-strī-samaṃ nāāyuṣyam avaśya-nindyam atha no veśyā-samaṃ dṛśyate ।।
kiṃ kāco api maṇir mṛgo api karaṭiḥ preṣyo api vā mānuṣaḥ kiṃ veśyāi vilāsinī sa ca tad-āsakto api kiṃ kāmukaḥ ।। sā kiṃ śrīr upabhogam arhati na yā sevyaḥ sa kiṃ yo antaraṃ no jānāti tad asti kiṃ vilasitaṃ veśyāsu yat kṛtrimam ।। bhogāḥ pralayaṃ prayāntu katham apy aucitya-bhaṅgena ye yāyātur nidhanaṃ varaṃ nidhuvanaṃ veśyāsu mūlyena yat ।। saubhāgyasya vimānanā viguṇatā saundarya-sāra-śriyaḥ śṛṅgārasya viḍambanā kim aparaṃ veśyā-rata-ḍambaraḥ ।।
yuktāukta-vicāraṇā-parihṛtaḥ kāmaṃ sa kāmī krimiḥ paṇya-strīṣu nitamba-nāmani mahā-nimbe ratiṃ vindati ।। sauhārdasya ca vibhramasya ca rater bhartuś ca yūnāṃ tathā yat saṃtyajya kulāṅganāṃ bahu-mukho dāsī-janaḥ sevyate ।। chāyāi cchalanāmikā parimalogāro api hālāhalaṃ veśyānāṃ viṣayeṣu kiṃ puno aho saktir mahā-sāhasam ।। vikreyaṃ nijam aṅgam aṅgam api cāaṅgaḥ sahāyaḥ svayaṃ tasyāartha-śatāmakasya gaṇikā-vyādheḥ kim asty auṣadham ।।
uttāneṣu vivartaneṣu śaphari vitta-cchale mūṣikā veśyā kāmuka-vañcanāya bhuvane rūpair anekaiḥ sthitā ।। ācāro bahu-gāli-dānam adhamo mitraṃ dhanaṃ jīvitaṃ veśyānāṃ puracāra tha bhagnaḥ priyaḥ ।। mānyaḥ prāhata ity aho kitava ity āptaś ca kiṃ vāhikaṃ sarvāastha iti pramāṇa-puruṣo veśyābhir abhyarcyate ।। atrāte gṛha-devatā vidadhatī dig-ghaṭṭanaṃ kuṭṭanī yasyāḥ pratyaham āḍhya-kāmuka-paśu-vrātoahārair baliḥ ।।
kuṭṭanyāḥ puna utkaṭokaṭam idaṃ tatrāty agastya-vrataṃ yat prāṇāutir ekakaia sakalaī ratnāaraiḥ kāmibhiḥ ।। veśyānāṃ vipule nitamba-phalake śāraiḥ kaṭākṣair ito yad vaḥ krīḍitam atra dāsyati puro dāridram evotaram ।। kaṃ cin nosahate sa rāja-tanayo jānāti mātā mamey evaṃ mugdha-bhujaṃga rūkṣa-gaṇikā-vāco acirāc chroṣyasi ।। veśyā dīpa-śikhea bhāti rajanau rūpa-bhramādhīkṛto yatrāaṃ kurute pataṅga-patanaṃ hā hā bhujaṃga-vrajaḥ ।।
utkampaṃ mati-vibhramaṃ vidadhatī yūnāṃ śanair ninditā tāruṇyasya janāavāda veśyā ca saṃdṛśyate ।। teneṣor iva kāmukasya sa-rasasyāāya sāraṃ paraṃ veśyābhiḥ kriyate bahir yadi paraṃ niṣkāsanaṃ śalkavat ।। ity-ādi prathamaṃ yadea surata-prauḍhoacāra-krame veśyā paśyata tat tad eva kurute nirvāsane kāminām ।। yat sadbhāva-kathānakaṃ rahasi yat kiṃ cid abhyarthanaṃ veśyānāṃ viṣa-vat tad eva nipuṇaṃ cintyaṃ sadā kāmukaiḥ ।।
sītkārātara-satyam artha-haraṇaṃ sarvatra ye jānate veśyānāṃ paramaṃ rahasyam avanau te ke api medhāvinaḥ ।। kiṃ cit kāñcana-ratna-vastra-viṣayair nānā-vidhair dohadair dvārāṇy artha-vinirgamasya gaṇikāḥ kurvanti śṛṅgāriṇām ।। saṅgītaṃ rajanī vidhur madhu-madaḥ spardhā sa-patnais tathā veśyānām anurakta-vitta-haraṇe kurvanti sāhāyakam ।। yasmān na praṇayena na praṇatibhiḥ premṇā na na prīṇanair na prāṇair api te bhujaṃga gaṇikā vittaṃ vinā tuṣyati ।।
ity uktvā sahasā bhujaṅga-purato yad veśyayā rudyate bhuktochiṣṭa-daridra-kāmi-vibhavās te kevalaṃ tad viduḥ ।। taj jāgarmi varaṃ niśāsu yad asau nidrāi me drohiṇīy ālāpair gaṇikā-gaṇena na ca ko viśvāsyate kāmukaḥ ।। ātmīyāṃ cala-citta-vṛtti-racanāṃ veśyā varākī sadā yūnāṃ darśayatīa te yadi paraṃ mūḍhā na taj jānate ।। ity ukto api na budhyate khalu yadā rāgī daridro jaḍaś ceṭībhir bahu-bhāṣitair api tadā hastāgalair vāryate ।।
kṛtvā taṃ prati ca vratāi virahaṃ svaṃ nāṭayitvā tatas tasmin mitra-mukhena saṃdhir aparo veśyābhir utpādyate ।। tasyaiāya nirāśa durbhaga paśo nirlajja gacchāhunā dhik tvāṃ nirdhana-caṅgam ity apavadan hrīto na veśyā-janaḥ ।। skandhāakta-paṭac-carā dya nucarā śokād a-kiṃ-cit-karās te amī kāmi-varāḥ purā samabhavan veśyāsu ye gocarāḥ ।। ruddho dyūta-karaiḥ karoti vidhuraś cauryaṃ tatas taskaro vadhyaḥ syān nṛpater aho nu viṣayāakter durantā gatiḥ ।।
yat-kim-cit-karaṇe para-sva-haraṇe vrīḍā na pīḍā-karī no vā rāja-bhayaṃ ca hī bata sukhaṃ jīvanti vāra-striyas ।। prādhānyena tatas tad eva kathitaṃ yat tat-prasaṅgena ca vyākhyātaṃ guṇa-doṣa-jātam ucitaṃ cintyaṃ tad apy ādarāt ।। nāi svāmy-asamarpaṇena na sakhe sakhyā virodhena vā gūḍhātarmukha-sāhasa-vyatikarā-rakṣyāḥ paraṃ yoṣitaḥ ।। pratyāśā-punarukta-hāraṇa-jagad-drohaṃ vivādāpadaṃ dāridryasya nimantraṇaṃ kim aparaṃ dhig dyūta-līlāyitam ।।
śūnyāaṇya-sadā-pravāsa-virasa-vyāyāma-mithyā-guṇāḥ kāya-kleśa-phalāadhir matimatāṃ sevyā mṛgavyāi kim ।। yat-tad-vādda-kali-pramāda-vividhomādaiḥ piśācāyitaṃ madyaṃ mūrtam amedhyam etad itarād anyasya kasya priyam ।। jihvāre kara-pattra-mitram aniśaṃ tat karkaśaṃ durvaco yasyāte vada kad-vadaḥ katham aho sa api svayaṃ jīvati ।। saumyaḥ saumya-karo ativiśva-nayanāando abhinandyaḥ satāṃ sa-śrīkaḥ kamalāaraś ca suṣamāṃ kāṃ kām aho nāhati ।।
siddhāñā garalasya tīrtha-saraṇaṃ vṛddhatva-vaiklavyayoḥ saptāṅga-vyasanāaleḥ praśamanoāyaś ca śikṣā satām ।। kaṣṭānām antya-kāṣṭhā dhana-rasika-vaṇig-lokayātrā-prasaṅgo jīvaj-janmātaraṃ cey alam atibahunā svasti deśātarāya ।। tatrāi sva-gṛheṣu guṇavad-goṣṭhī-gariṣṭhī-kṛta- sphāro anārata-bhāratī-rasa-pariṣyandaḥ sakhe sundaraḥ ।। vandyās te api tad-antarāla-patitā ye bhṛṅga-bhaṅgī-juṣaḥ sāndrāanda-karambitaṃ jagad idaṃ vindanti nindanti ca ।।
kāruṇyāt taruṇaṃ janaṃ prati satām abhyarthanābhis tathā so ayaṃ saṃprati Jalhaṇena kavinā Mudghopadeśaḥ kṛtaḥ ।। asmākaṃ sura-sindhu-rodhasi punaḥ śaṃbhāva-daṃbhācanair vṛddhatvasya viśuddhatāṃ janayituṃ kartavya-śeṣaḥ sthitaḥ ।।
