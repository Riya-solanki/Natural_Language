jhaṭiti prativaktavyaṃ sabhāsu vijigīṣubhiḥ ।। hāso rājñaḥ stavaś cei pañcaie jaya-hetavas ।। paṇḍito yadi tatraia pakṣa-pāto adhiropyatām ।। ātmotkarṣo nigamanam anumāneṣv ayaṃ vidhis ।।
jigīṣunā hriyaṃ tyaktvā kāryaḥ kolāhalo mahān ।। evam ca tathya-vyutpattir āyuṣo ante bhaven na vā ।। na stauti cet svayaṃ ca svaṃ kadā tasyātu nirvṛtis ।। iti pāṭhayatāṃ granthe kāṭhinyaṃ kutra vartate ।।
trayaḥ śiṣya-guṇā hy ete mūrkhāāryasya bhāgya-jās ।। māntrikās tu bhaviṣyāmo yogino yatayo api vā ।। vilambe karma-bāhulyaṃ vikhyāpyāāpyate dhanam ।। anugrahāyate yeṣāṃ te dhanyāḥ khalu māntrikās ।।
tāvan māhātmya-rūpeṇa paryavasyati māntrike ।। graha-cāra-parijñānaṃ teṣām āvaśyakaṃ yatas ।। garbha-praśneṣu kathayan daivajño vijayī bhavet ।। jīvanto bahumanyante mṛtāḥ prakṣyanti kaṃ punas ।।
sarvaṃ vyāmiśram iva ca vaktavyaṃ daiva-cintakais ।। bruvāṇāḥ sarvathā grāhyā lokair jyautiṣikā janās ।। daivajñānām upālambho nityaḥ kārya-viparyaye ।। deśo hy aratni-mātre api nāsti daivajña-varjitas ।।
tritayaṃ ye vijānanti te vācas-patayaḥ svayam ।। nisarga-śatravaḥ sṛṣṭā daivajñānām amī trayas ।। kātarā dīrgha-rogāś ca bhiṣajāṃ bhāgya-hetavas ।। naiścintyān nāime dānaṃ nairāśyād eva nātime ।।
ārogyaṃ vaidya-māhātmyād anyathātvam apathyatas ।। sarvam apy upadekṣyanti rogiṇaḥ sadane striyas ।। roga-tattveṣu śanakair vyutpadyante cikitsakās ।। bahumānātham ante ca jihīrṣanti cikitsakās ।।
dāyādāḥ saṃprarohanti daivajñā māntrikā api ।। śanair anādaras śāntau snāto vaidyaṃ na paśyati ।। ekaiam artha-lābhāya dvi-tri-yogas tu durlabhas ।। satyaṃ vaiduṣyam ity eṣa yogo dāridrya-kārakas ।।
sarvaṃ mārjanti kavayaḥ śālīnāṃ muṣṭi-kiṃkarās ।। kiṃ cid astuvatām teṣāṃ jihvā phuraphurāyate ।। kītaḥ kaś cid alir nāma kiyatī tatra varṇanā ।। antato annāya vastrāya tāmbūlāya ca kalpate ।।
vikrīyate katipayair vṛthāyair viniyujyate ।। labdhvāi kāma-dhenuṃ te lāṅgale viniyuñjate ।। kathaṃ tarantu kavayaḥ kāma-pāramya-vādinas ।। samādher api taj jyāyāḥ śaṃkaro yadi varṇyate ।।
prāṇinām kalinā sṛṣṭāḥ pañca prāṇā ime apare ।। ajñātā eva gṛhiṇāṃ bhakṣayanty ākhu-vad gṛhe ।। śvaśurasya balaṃ bhāryā svayam evāither balam ।। prabudhyamāne sāralyaṃ prabuddhe asmin palāyanam ।।
śvaśuras tu pravāsīi jāmātur bhāgya-dhoraṇī ।। sakṛd āgatya gacchantī kanyā nirmārṣṭi mandiram ।। pati-pakṣyāṃs tu bahv-āśān kṣīra-pāṃs taskarān api ।। aśrānta-kalaho nāma yogo ayaṃ gṛha-medhinām ।।
jīrṇau ca mātā-pitarāv ekaikaṃ narakāhikam ।। aho mahad idaṃ bhūtam uttama-rṇāhiśābdhitam ।। na kāla-niyamaḥ kaś cid uttamārṇasya vidyate ।। uttamārṇam avekṣyaia tathāy udvejite manas ।।
mṛtyau mṛtyuñjaya-dhyānaṃ dāridrye tu na kiṃ cana ।। dīpayaty udare vahniṃ dāridryaṃ paramauṣadham ।। protsāhayati yācñāyāṃ rājāñea daridratā ।। ākiṃcanya-balāḍhyānām antato aśmāi jīryati ।।
dainyaṃ rājyād api jyāyo yadi tattvaṃ prabudhyate ।। protsāhayati dāyādāṃl lākṣmīḥ kiṃ cid upasthitā ।। ta eva tu viḍambyante śriyā kim-cid-upekṣitās ।। kīṭa-buddhir manuṣyeṣu nūtanāyāḥ śriyaḥ phalam ।।
viḍambanāni dhanikāḥ stotrāṇīy eva manvate ।। svairaṃ hasanti pārśva-sthā bālomatta-piśāca-vat ।। na bibheti na jihreti tathāi dhaniko janas ।। lakṣmī-madas tu mūrkhāṇām ā-deham anuvartate ।।
vikāras tu tad-ārabdho nityaṃ laśuna-gandha-vat ।। lakṣmī-madas tu sarvāṅge putra-dāra-mukheṣv api ।। kule apy avataraty eṣa kuṣṭhāasmāra-vat katham ।। vismārayanti jātiṃ svāṃ varāṭāḥ pañcaṣā kare ।।
yāvad yācaka-sādharmyaṃ tāval loko na mṛṣyati ।। kathaṃ te taṃ laghūkartuṃ yatante aparathā svatas ।। dvi-guṇaṃ tri-guṇaṃ caia kṛtāto lālayiṣyati ।। rāja-karṇe japaḥ sadyaḥ sarva-karma-vināśanas ।।
parātheṣu pravartante śaṭhāḥ santaś ca tulya-vat ।। khalo gṛha-samīpa-sthaḥ sadyo anarthāya dehinām ।। japa-yajñaś ca yajñeṣu rocate lobha-śālinām ।। kiṃ prakṣyatīi lubdho api tāvad udvijate tatas ।।
bhikṣākabalam ekaiaṃ ye hi paśyanti meru-vat ।। dhana-lubdhaḥ piśācas tu na kasmai cana ditsate ।। kartṛ-karma-vyatīhārād aho nimnonataṃ kiyat ।। niścityaiaṃ svayam api bhuṅkte lubdhaḥ kathaṃ cana ।।
vicitrān ullikhed vighnāṃs tiṣṭhāsur atithiś ciram ।। sahasraṃ dāmbhike loke śrotriye tu na kim-cana ।। tāvad arthāḥ prasiddhyanti yāvac cāpalam āvṛtam ।। vaktavyaṃ kiṃ cid ity uktaṃ vastutas tulasī parā ।।
viśva-saṃmohinī vitta- dāyinīi guṇa-dvayam ।। maunam ekāikā cei mūrkha-saṃjīvanāni ṣaṭ ।। adhyāpanāṛttayaś ca kīrtanīyā dhanāthibhis ।। deśa-dharmas tv anācāre pṛcchatāṃ siddham uttaram ।।
tathā dharmaṃ caret sarvaṃ na vṛthā kiṃ cid ācaret ।। sarvaṃ brahmei vādaś ca sadyas-pratyaya-hetavas ।। satataṃ śuci-veṣaś cety etad dambhasya jīvitam ।। tāvat-saṃkṣipyate sarvaṃ yāvad draṣṭā na vidyate ।।
kiṃ tasya sādhanair anyaiḥ kiṃkarāḥ sarva-pārthivās ।। durjanānām ato nyāyyaṃ dūrād eva visarjanam ।। saṃpūrṇa-dānaṃ prakṛtir virāmo vaira-kāraṇam ।। apatya-saṃbandha-vidhiḥ svāarthāyaia kevalam ।।
milanti yadi catvāri tad diśe api namo namas ।। para-marmāsu vācaṃ ca khalānām asṛjad vidhis ।। kalinā daśamāṃśena sarvaḥ kālo api dāruṇas ।। yatra sva-pratibhā mānaṃ tasmai śrī-kalaye namas ।।
kāla-kālaṃ prapannānāṃ kālaḥ kiṃ naḥ kariṣyati? ।। racitaṃ viduṣāṃ prītyai rājāsthānāumodanam ।।
