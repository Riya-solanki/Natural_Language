jayaty ullāsitānanda- mahimā parameśvaraḥ ।। vidyoddyotitamāhātmyaḥ sa jayaty aparājitaḥ ।। namo anantaprakāśāya śaṃkarakṣīrasindhave ।। ativāllabhyataḥ sādhu viśvaṅ no dhṛtavān asi ।।
viyanmāyāsvarūpāya vibhave śambhave namaḥ ।। namāmaḥ sarvasāmānyaṃ rūpaṃ tat pārameśvaram ।। brahmāṇḍād api nairguṇya- guṇāya sthāṇave namaḥ ।। parameśvarahaṃsasya śaktiṃ haṃsīm iva stumaḥ ।।
jagaccitraṃ namas tasmai kalāślāghyāya śūline ।। śivajñānaṃ svato dugdhaṃ dehy ehi harahaṃsa naḥ ।। paramārthaikabhāvāya baliṃ yāmo bhavāya te ।। unmṛṣṭatamasaṃ ramyām antarbhava bhavadguhām ।।
nāthaṃ svapne api paśyanti paramānandadāyinam ।। punarbhavabhayoccheda- dakṣā kasmai na rocate ।। tvatprasādāt tvadekāgra- manaskatvena yā sthitiḥ ।। namo anantaphalotpāda- kalpavṛkṣāya śambhave ।।
tvanmayībhūya nirdvandvāḥ kaccit syāmāpi karhicit ।। ananyāpekṣasāmarthya- śālini śūline namaḥ ।। nāmāpi dhyāyatāṃ dhyānaiḥ kim anyālambanaiḥ phalam ।। ślāghyās te śāmbhavīṃ bhūtim upabhoktuṃ ya udyatāḥ ।।
kiṃ dhyānaṃ yena na dhyeyaḥ kiṃ vā kiṃ nāsi yat prabho ।। rasaḥ srotaḥsahasreṇa tvayi me bhava vardhatām ।। nāthāya sthāṇave tubhyaṃ nāgayajñopavītine ।। bhava tattatpradānena prasādaḥ kriyatāṃ mayi ।।
praṇantavyaḥ praṇāmo api yadīya iha dhīmatām ।। bhakticintāmaṇiṃ śārvaṃ tataḥ prāpya na kiṃ jitam ।। jñeyo asi kila ke apy ete ye tvāṃ jānanti dhūrjaṭe ।। niṣkāmo api prakṛtyā yaḥ kāmanānāṃ paraṃ phalam ।।
tat kṣīrodadam aiśvaryaṃ tavaiva sahajaṃ vibho ।। smṛtyā nutyā ca dadate śambhave śaṃ bhave abhave ।। vedyaṃ kiṃ karma vā nātha nānantyāya tvayārpyate ।। tasya te nātha kāryāṇāṃ ko vetti kiyatī gatiḥ ।।
upary upari dhāvanti labdhuṃ dhāma namāmi tam ।। iti śaktilatā yasya puṣpitā pātv asau bhavaḥ ।। dūragair api yasyānto durgam astaṃ stumo mṛḍam ।। prāptau cānandavṛndāya dayitāya kapardine ।।
smayena tvanmayo asmīti māmi nātmani kiṃ mudā ।। viśrāmyan bhava bhāvatka- cittānande rame bhṛśam ।। sthūlo atha kiṃ sudarśo na brahmādibhir api prabho ।। kaḥ kleśaṃ deva vāgjāleṣv akariṣyat sudhīs tadā ।।
dṛśyo asīty upadeśena prasādaḥ kriyatāṃ mama ।। sarvasya spṛhaṇīyāya niḥspṛhāya kapardine ।। yayānantaphalāṃ bhaktiṃ vapanti tvayy amī prabho ।। tvaddhyānanidhilābhe api mugdho lokaḥ ślathāyate ।।
viśvam astu svatantras tu kartā tatraikako bhavān ।। uddhartuṃ bhavato anyasya kasya śaktiḥ kṛpāthavā ।। guṇo api ca guṇaḥ ko nu tvāṃ nāptuṃ yaḥ samāsthitaḥ ।। lobhāyāpi namas tasmai tvallābhālambanāya me ।।
ābrahmakrimi yasmin no muktaye adhikriyeta kaḥ ।। tadantarvṛttayaś citrās tavaiveśo dhiyaḥ pathi ।। tvadbhaktirasapīyūṣān nātha nānyatra dṛśyate ।। avataṃsitasomāya somāya svāmine namaḥ ।।
sarvānugrāhikā śaktiḥ śāṃkarī śaraṇaṃ mama ।। labhyate bhava kutrāṃśe paraḥ pratinidhis tava ।। vayaṃ vañcyāmahe adyāpi māyayāmeyayā tava ।। amī bhava bhavadbhakti- kalpapādapapallavāḥ ।।
iti labdho api karhi tvaṃ lapsyase nātha kathyatām ।। bhūribhūtisitāṅgāya mahāhaṃsāya śambhave ।। tvadbodhadīpikā me astu nātha tvadbhaktidīpikā ।। prastāvya hara saṃhartuṃ tvattaḥ ko anyaḥ kaviḥ kṣamaḥ ।।
svatantrāyāsvatantrāya vyayaiśvaryaikaśāline ।। sa bindur yasya taṃ vande devam ānandasāgaram ।। namo namaḥ śivāyeti japanty āhlādavihvalāḥ ।। anāditve api viśvasya bhoktre bhava namo astu te ।।
lipseran nopakāraṃ ke yataḥ sampūrṇadharmaṇaḥ ।। smṛte yasmin prasīdanti matayas taṃ śivaṃ stumaḥ ।। yadicchayā pratāyete trailokyasya layodayau ।। sukṛtasyāpi saukṛtyaṃ yato anyatra na so asi bhoḥ ।।
iti bhaktirasādhmātā dhanyā dhāvanti dhūrjaṭim ।। yasya vedātmikājñeyam aho gambhīrasundarī ।। dharmagrāmaḥ pravṛtto yas tvatto na sa kuto bhavet ।। jñānānandaṃ ca nirdvandvaṃ deva vṛtvā vivṛṇvate ।।
māyām ekanimeṣeṇa muṣṇaṃs tāṃ pātu naḥ śivaḥ ।। naiḥspṛhyasya parāṃ koṭiṃ bibhratāṃ tvaṃ prabho prabhuḥ ।। jagatkalyāṇakalyāṇaṃ kiyat tvam iti vetti kaḥ ।। sarvasthāne mameśa tvaṃ ya uddhartā bhavārṇavāt ।।
śaivayogabalākṛṣṭā divyapīyūṣavipruṣaḥ ।। codayed api kaccin naḥ sa dhiyaḥ satpathe prabhuḥ ।। śāśvatīṃ kuruṣe yad vā tuṣṭaḥ sarvaṃ kariṣyasi ।। bhavatas tv īśa nāmāpi mokṣaparyantasiddhidam ।।
śaktir yā te tayā nātha ko manasvī na vismitaḥ ।। ekavīrā smṛtir yasya taṃ smarāmaḥ smaradviṣam ।। yannāmnāpi mahātmānaḥ kīryante pulakāṅkuraiḥ ।। svaśaktivyūhasaṃvyūḍha- trailokyārambhasaṃhṛtiḥ ।।
sa mahyaṃ dehi taṃ mantraṃ tvanmantraḥ syāṃ yathā prabho ।। saṃsārīti kim īśaiṣa svapnaḥ so api kutas tvayi ।। tvayi karmaphalanyāsa- kṛtām aiśvaryam īśa yat ।। yo asau sa dayito asmākaṃ devadevo vṛṣadhvajaḥ ।।
nirmalīkaraṇe nātha tvadbhaktiḥ paramāñjanam ।। padaṃ dehy ehi me deva tūrṇaṃ tat kiṃ pratīkṣase ।। aho tattaraṇopāyaḥ paraḥ ko api maheśvaraḥ ।। mastakanyastagaṅgāya yathāyuktārthakāriṇe ।।
nātha prārthayamānānāṃ tvad ṛte kā parā gatiḥ ।। tvadicchānugṛhītas tu vahed brāhmīṃ dhuraṃ na kaḥ ।। nameyuḥ kaṃ paraṃ kaṃ vā namayeyur na dhīdhanāḥ ।। bhavajjñānāmbudher madhyam adhyāsīyāpi dhūrjaṭe ।।
paramārthaphalaṃ nātha paripūrṇaṃ prayacchasi ।। iti nātha numaḥ kiṃ tvāṃ kiṃ nindāmo na manmahe ।। tat pātum api ko anyo alaṃ kim u dātuṃ jagatpate ।। tvattas tvadbhaktim evāptuṃ prārthaye nātha sarvathā ।।
trilokīnāthatādāna- nirvināyakaśaktaye ।। svāmin so asīti niścitya kas tvāṃ na śaraṇaṃ gataḥ ।। ity āśaṃseha śobheta śambhau bhaktimataḥ param ।। tvadadhīnatvadarpeṇa sarvatrātrāsmi nirvṛtaḥ ।।
cittasyeśa tamaḥsparśo manye vandhyātmajānujaḥ ।। manorathaḥ sthito yasya seviṣye śivam ity ayam ।। ekaiveśa bhavacchaktiḥ svatantraṃ tantram īkṣase ।। tasmai namo astu te yas tvaṃ tannirvāṇāmṛtahradaḥ ।।
niśchadmā ced bhaved eṣā kiṃphaleti tvayocyatām ।। phalaṃ tad aja jātaṃ yat tvatkathākalpapādapāt ।। tadantarāyahṛtaye tvam īśa śaraṇaṃ mama ।। kutra kālakalāmātre na yo navanavāyate ।।
kalpayann api ko apy eko nirvikalpo jayaty ajaḥ ।। jñānadīpena deva tvāṃ kadā nu syām upasthitaḥ ।। padaṃ citte tadā śambho kiṃ na sampādayiṣyasi ।। bhavet sālambanā tasya yas tvadālambanaḥ prabho ।।
bhavadbhaktis tu jananī śubhasyaiveśa kevalam ।। tvatpraveśāt prasīdet tad iti dolāyate janaḥ ।। prasādo manasaḥ svāmin sā siddhis tat paraṃ padam ।। tvatprasādena tad bhūyād bhavadbhāvaikabhūṣaṇam ।।
bhaktilakṣmyālayaṃ śambho bhaṭṭanārāyaṇo vyadhāt ।।
