atas tvām ārādhyāṃ hariharavirañcyādibhir api praṇantuṃ stotuṃ vā katham akṛtapuṇyaḥ prabhavati ।। vahaty enaṃ śauriḥ katham api sahasreṇa śirasāṃ haraḥ saṃkṣubhyainaṃ bhajati bhasitoddhūlanavidhim ।। daridrāṇāṃ cintāmaṇiguṇanikā janmajaladhau nimagnānāṃ daṃṣṭrā muraripuvarāhasya bhavatī ।। bhayāt trātuṃ dātuṃ phalam api ca vāñchāsamadhikaṃ śaraṇye lokānāṃ tava hi caraṇāv eva nipuṇau ।।
smaro api tvāṃ natvā ratinayanalehyena vapuṣā munīnām apy antaḥ prabhavati hi mohāya mahatām ।। tathāpy ekaḥ sarvaṃ himagirisute kām api kṛpāṃ apāṅgāt te labdhvā jagad idam anaṅgo vijayate ।। dhanur bāṇān pāśaṃ sṛṇim api dadhānā karatalaiḥ purastād āstāṃ naḥ puramathitur āhopuruṣikā ।। śivākāre mañce paramaśivaparyaṅkanilayāṃ bhajanti tvāṃ dhanyāḥ kati cana cidānandalaharīm ।।
mano api bhrūmadhye sakalam api bhittvā kulapathaṃ sahasrāre padme saha rahasi patyā viharasi ।। avāpya svāṃ bhūmiṃ bhujaganibham adhyuṣṭavalayaṃ svam ātmānaṃ kṛtvā svapiṣi kulakuṇḍe kuhariṇi ।। trayaś catvāriṃśad vasudalakalāśratrivalayatrirekhābhiḥ sārdhaṃ tava caraṇakoṇāḥ pariṇatāḥ ।। yad ālokyautsukyād amaralalanā yānti manasā tapobhir duṣprāpām api giriśasāyujyapadavīm ।।
galadveṇībandhāḥ kucakalaśavisrastasicayā haṭhāt truṭyatkāñcyo vigalitadukūlā yuvatayaḥ ।। divi dviḥṣattriṃśan manasi ca catuḥṣaṣṭir iti ye mayūkhās teṣām apy upari tava pādāmbujayugam ।। sakṛn na tvā natvā katham iva stāṃ saṃnidadhate madhukṣīradrākṣāmadhurimadhurīṇā bhaṇitayaḥ ।। virañcipreyasyās taruṇataraśṛṅgāralaharīgabhīrābhir vāgbhir vidadhati satāṃ rañjanam amī ।।
sa kartā kāvyānāṃ bhavati mahatāṃ bhaṅgisubhagair vacobhir vāgdevīvadanakamalāmodamadhuraiḥ ।। bhavanty asya trasyadvanahariṇaśālīnanayanāḥ sahorvaśyā vaśyāḥ katikati na gīrvāṇagaṇikā ।। sa sadyaḥ saṃkṣobhaṃ nayati vanitā ity atilaghu trilokīm apy āśu bhramayati ravīndustanayugām ।। sa sarpāṇāṃ darpaṃ śamayati śakuntādhipa iva jvarapluṣṭān dṛṣṭyā sukhayati sudhādhārasirayā ।।
mahāpadmāṭavyāṃ mṛditamalamāyena manasā mahāntaḥ paśyanto dadhati paramāhlādalaharīm ।। tadaiva tvaṃ tasmai diśasi nijasāyujyapadavīṃ mukundabrahmendrasphuṭamukuṭanīrājitapadām ।। tathā hi tvadrūpaṃ sakalam aruṇābhaṃ trinayanaṃ kucābhyām ānamraṃ kuṭilaśaśicūḍālamukuṭam ।। sadāpūrvaḥ sarvaṃ tad idam anugṛhṇāti ca śivas tavājñām ālambya kṣanacalitayor bhrūlatikayoḥ ।।
tathā hi tvatpādodvahanamaṇipīṭhasya nikaṭe sthitā hy ete śaśvan mukulitakarottaṃsamukuṭāḥ ।। vitandrī māhendrī vitatir api saṃmīlitadṛśā mahāsaṃhāre asmin viharati sati tvatpatir asau ।। praṇāmaḥ saṃveśaḥ sukham akhilam ātmārpaṇadṛśā saparyāparyāyas tava bhavatu yan me vilasitam ।। tavāsmin mandārastabakasubhage yātu caraṇe nimajjan majjīvaḥ karaṇacaraṇaiḥ ṣaccaraṇatām ।।
karālaṃ yat kṣveḍaṃ kavalitavataḥ kālakalanā na śaṃbhos tanmūlaṃ tava janani tāṭaṅkamahimā ।। praṇamreṣv eteṣu prasabham abhiyātasya bhavanaṃ bhavasyābhyutthāne tava parijanoktir vijayate ।। punas tvannirbandhād akhilapuruṣārthaikaghaṭanāsvatantraṃ te tantraṃ kṣititalam avātītarad idam ।। amī hṛllekhābhis tisṛbhir avasāneṣu ghaṭitā bhajante varṇās te tava janani nāmāvayavatām ।।
japanti tvāṃ cintāmaṇigunanibaddhākṣavalayāḥ śivāgnau juhvantaḥ surabhighṛtadhārāhutiśataiḥ ।। ataḥ śeṣaḥ śeṣīty ayam ubhayasādhāraṇatayā sthitaḥ saṃbandho vāṃ samarasaparānandaparayoḥ ।। tvam eva svātmānaṃ pariṇamayituṃ viśvavapuṣā cidānandākāraṃ śivayuvati bhāvena bibhṛṣe ।। yam ārādhyan bhaktyā raviśaśiśucīnām aviṣaye nirāloke loko nivasati hi bhālokabhavane ।।
yayoḥ kāntyā yāntyā śaśikiraṇasārūpyasaraṇiṃ vidhūtāntardhvāntā vilasati cakorīva jagatī ।। yadālāpād aṣṭādaśaguṇitavidyāpariṇatir yad ādatte doṣād guṇam akhilam adbhyaḥ paya iva ।। yadāloke lokān dahati mahati krodhakalile dayārdrā dṛṣṭis te śiśiram upacāraṃ racayati ।। tava śyāmaṃ meghaṃ kam api maṇipūraikaśaraṇaṃ niṣeve varṣantaṃ haramihirataptaṃ tribhuvanam ।।
ubhābhyām etābhyām ubhayavidhim uddiśya dayayā sanāthābhyāṃ jajñe janakajananīmad jagad idam ।। sa nīḍeyacchāyācchuraṇaśabalaṃ candraśakalaṃ dhanuḥ śaunāsīraṃ kim iti na nibadhnāti dhiṣaṇām ।। yadīyaṃ saurabhyaṃ sahajam upalabdhuṃ sumanaso vasanty asmin manye balamathanavāṭīviṭapinām ।। tanotu kṣemaṃ nas tava vadanasaundaryalaharīparīvāhasrotaḥsaraṇir iva sīmantasaraṇiḥ ।।
darasmere yasmin daśanarucikiñjalkarucire sugandhau mādyanti smaramathanacakṣurmadhulihaḥ ।। viparyāsanyāsād ubhayam api saṃbhūya ca mithaḥ sudhālepasyūtiḥ pariṇamati rākāhimakaraḥ ।। dhanur manye savyetarakaragṛhītaṃ ratipateḥ prakoṣṭhe muṣṭau ca sthagayati nigūḍhāntaram ume ।। tṛtīyā te dṛṣtir daradalitahemāmbujaruciḥ samādhatte saṃdhyāṃ divasaniśayor antaracarīm ।।
avantī dṛṣṭis te bahunagaravistāravijayā dhruvaṃ tattannāmavyavaharaṇayogyā vijayate ।। amuñcantau dṛṣṭvā tava nvarasāsvādataralāv asūyāsaṃsargād alikanayanaṃ kiṃcid aruṇam ।। harāhibhyo bhītā sarasiruhasaubhāgyajayinī sakhīṣu smerā te mayi janani dṛṣṭiḥ sakaruṇā ।। ime netre gotrādharapatikulottaṃsakalike tavā-karṇākṛṣṭasmaraśaravilāsaṃ kalayataḥ ।।
punaḥ sraṣṭuṃ devān druhiṇaharirudrān uparatān rajaḥ sattvaṃ bibhrat tama iti guṇānāṃ trayam iva ।। nadaḥ śoṇo gaṅgā tapanatanayeti dhruvam amuṃ trayāṇāṃ tīrthānām upanayasi saṃbhedam anaghe ।। iyaṃ ca śrīr baddhacchadapuṭakapāṭaṃ kuvalayam jahāti pratyūṣe niśi ca vighaṭayya praviśati ।। tvadunmeṣāj jātaṃ jagad idam aśeṣaṃ pralayataḥ paritrātuṃ śaṅke parihṛtanimeṣās tava dṛśaḥ ।।
anenāyaṃ dhanyo bhavati na ca te hānir iyatā vane vā harmye vā samakaranipāto himakaraḥ ।। tiraścīno yatra śravaṇapatham ullaṅghya vilasan apāṅgavyāsaṅgo diśati śarasaṃdhānadhiṣaṇām ।। yam āruhya druhyaty avaniratham arkenducaraṇaṃ mahāvīro māraḥ pramathapataye svaṃ jitavate ।। camatkāraślāghācalitaśirasaḥ kuṇḍalagaṇo jhaṇatkārais tāraiḥ prativacanam ācaṣṭae iva te ।।
vahann antar muktāḥ śiśirataraniśvāsaghaṭitāḥ samṛddhyā yas tāsāṃ bahir api ca muktāmaṇidharaḥ ।। na bimbaṃ tadbimbapratiphalanalābhād aruṇitaṃ tulām adhyāroḍhuṃ katham iva vilajjeta kalayā ।। atas te śītāṃśor amṛtalaharīm amlarucayaḥ pibanti svacchandaṃ niśiniśi bhṛśaṃ kāñcikadhiyā ।। yadagrāsīnāyāḥ sphaṭikadṛṣadacchacchavimayī sarasvatyā mūrtiḥ pariṇamati māṇikyavapuṣā ।।
viśākhendropendraiḥ śaśiviśadakarpūraśakalā vilīyante mātas tava vadanatāmbūlakabalāḥ ।। tvadīyair mādhuryair apalapitatantrīkalaravāṃ nijāṃ vīṇāṃ vāṇī niculayati colena nibhṛtam ।। karagrāhyaṃ śaṃbhor mukhamukuravṛntaṃ girisute kathaṃkāraṃ brūmas tava cibukam aupamyarahitam ।। svataḥ śvetā kālāgarubahalajambālamalinā mṛṇālīlālityam vahati yadadho hāralatikā ।।
virājante nānāvidhamadhurarāgākarabhuvāṃ trayāṇāṃ grāmāṇāṃ sthitiniyamasīmāna iva te ।। nakhebhyaḥ saṃtrasyan prathamamathanād andhakaripoś caturṇāṃ śīrṣāṇāṃ samam abhayahastārpaṇadhiyā ।। kayā cid vā sāmyaṃ bhajatu kalayā hanta kamalaṃ yadi krīḍallakṣmīcaraṇatalalākṣāruṇadalam ।। yad ālokyāśaṅkākulitahṛdayo hāsajanakaḥ svakumbhau herambaḥ parimṛśati hastena jhaṭiti ।।
pibantau tau yasmād aviditavadhūsaṃgamarasau kumārāv adyāpi dviradavadanakrauñcadalanau ।। kucābhogo bimbādhararucibhir antaḥ śabalitāṃ pratāpavyāmiśrāṃ puravijayinaḥ kīrtim iva te ।। dayāvatyā dattaṃ draviḍaśiśur āsvādya tava yat kavīnāṃ prauḍhānām ajani kamaniyaḥ kavayitā ।। samuttasthau tasmād acalatanaye dhūmalatikā janas tāṃ jānīte janani tava romāvalir iti ।।
vimardād anyoanyaṃ kucakalaśayor antaragataṃ tanūbhūtaṃ vyoma praviśad iva nābhīkuhariṇīm ।। rater līlāgāraṃ kim api tava nābhīti girije biladvāraṃ siddher giriśanayanānāṃ vijayate ।। ciraṃ te madhyasya truṭitataṭinītīrataruṇā samāvasthāsthemno bhavatu kuśalaṃ śailatanaye ।। tava trātuṃ bhaṅgād alam iti vilagnaṃ tanubhuvā tridhā naddhaṃ devi trivali lavalīvallibhir iva ।।
atas te vistīrṇo gurur ayam aśeṣāṃ vasumatīṃ nitambaprāgbhāraḥ sthagayati laghutvaṃ nayati ca ।। suvṛttābhyāṃ patyuḥ praṇatikaṭhinābhyāṃ girisute vijigye jānubhyāṃ vibudhakarikumbhadvayam api ।। yadagre dṛśyante daśaśaraphalāḥ pādayugalīnakhāgracchadmānaḥ suramukuṭaśāṇaikaniśitāḥ ।। yayoḥ pādyaṃ pāthaḥ paśupatijaṭājūṭataṭinī yayor lākṣālakṣmīr aruṇaharicūḍāmaṇiruciḥ ।।
asūyaty atyantaṃ yad abhihananāya spṛhayate paśūnām īśānaḥ pramadavanakaṅkelitarave ।। cirād antaḥśalyaṃ dahanakṛtam unmūlitavatā tulākoṭikvāṇaiḥ kilikilitam īśānaripuṇā ।। paraṃ lakṣmīpātraṃ śriyam atisṛjantau samayināṃ sarojaṃ tvatpādau janani jayataś citram iha kim ।। kathaṃ vā bāhubhyām upayamanakāle purabhidā yad ādāya nyastaṃ dṛṣadi dayamānena manasā ।।
phalāni svasthebhyaḥ kisalayakarāgreṇa dadatāṃ daridrebhyo bhadrāṃ śriyam aniśam ahnāya dadatau ।। prakṛtyā mūkānām api ca kavitākāraṇatayā yad ādhatte vāṇīmukhakamalatāmbūlarasatām ।। suvikṣepe śikṣāṃ subhagamaṇimañjīraraṇitacchalād ācakṣāṇaṃ caraṇakamalaṃ cārucarite ।। bhṛśaṃ tanvī madhye pṛthur urasijārohaviṣaye jagat trātuṃ śaṃbhor jayati karuṇā kā cid aruṇā ।।
tathā hy ete nītāḥ śatamakhamukhāḥ siddhim atulāṃ tava dvāropāntasthitibhir aṇimādyābhir amarāḥ ।। tvadīyānāṃ bhāsāṃ pratiphalanalābhāruṇatayā śarīrī śṛṅgāro rasa iva dṛśāṃ dogdhi kutukam ।। atas tvadbhogena pratidinam idaṃ riktakuharaṃ vidhir bhūyobhūyo nibiḍayati nūnaṃ tava kṛte ।। kim āścaryaṃ tasya trinayansamṛddhiṃ tṛṇayato mahāsaṃvartāgnir viracayati nīrājanavidhim ।।
mahādevaṃ hitvā tava sati satīnām acarame kucābhyām āsaṅgaḥ kuravakataror apy asulabhaḥ ।। turīyā kāpi tvaṃ duradhigamaniḥsīmamahimā mahāmāye viśvaṃ bhramayasi parabrahmamahiṣi ।। ciraṃ jīvann eṣa kṣapitapaśupāśavyatikaraḥ parabrahmābhikhyaṃ rasayati rasaṃ tvadbhajanavān ।। svakīyair ambhobhiḥ salilanidhisauhityakaraṇaṃ tvadīyābhir vāgbhis tava janani vācāṃ stutir iyam ।।
dadhāti tvadvaktraṃ pratiphalitam aśrāntivikacaṃ nirātaṅkaṃ candrān nijahṛdayapaṅkeruham iva ।। harasya tvadbhrāntiṃ manasi janayām āsa madano bhavatyā ye bhaktāḥ pariṇatir amīṣām iyam ume ।। niyatyā nirmukte nikhilanigamāntastutipade nirātaṅke nitye nigamaya mamāpi stutim imām ।।
