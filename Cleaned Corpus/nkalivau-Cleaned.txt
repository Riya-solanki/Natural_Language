

na bhetavyaṃ na boddhavyaṃ na śrāvyaṃ vādino vacaḥ
jhaṭiti prativaktavyaṃ sabhāsu vijigīṣubhiḥ ।।

asaṃbhramo vilajjatvam avajñā prativādini
hāso rājñaḥ stavaś cei pañcaie jaya-hetavas ।।

uccair udghoṣya jetavyaṃ madhya-sthaś ced apaṇḍitas
paṇḍito yadi tatraia pakṣa-pāto adhiropyatām ।।

lobho hetur dhanaṃ sādhyaṃ dṛṣṭāntas tu purohitas
ātmotkarṣo nigamanam anumāneṣv ayaṃ vidhis ।।

abhyāsyaṃ lajjamānena tattvaṃ jijñāsunā ciram
jigīṣunā hriyaṃ tyaktvā kāryaḥ kolāhalo mahān ।।

pāṭhanair grantha-nirmāṇaiḥ pratiṣṭhā tāvad āpyate
evam ca tathya-vyutpattir āyuṣo ante bhaven na vā ।।

stotāraḥ ke bhaviṣyanti mūrkhasya jagatī-tale
na stauti cet svayaṃ ca svaṃ kadā tasyātu nirvṛtis ।।

vācyatāṃ samayo atītaḥ spaṣṭam agre bhaviṣyati
iti pāṭhayatāṃ granthe kāṭhinyaṃ kutra vartate ।।

agatitvam atiśraddhā jñānāhāsena tṛptatā
trayaḥ śiṣya-guṇā hy ete mūrkhāāryasya bhāgya-jās ।।

yadi na kvāi vidyāyāṃ sarvathā kramate matis
māntrikās tu bhaviṣyāmo yogino yatayo api vā ।।

avilambena saṃsiddhau māntrikair āpyate yaśas
vilambe karma-bāhulyaṃ vikhyāpyāāpyate dhanam ।।

sukhaṃ sukhiṣu duḥkhe api jīvanaṃ duḥkha-śāliṣu
anugrahāyate yeṣāṃ te dhanyāḥ khalu māntrikās ।।

yāvad ajñānato maunam ācāro vā vilakṣaṇas
tāvan māhātmya-rūpeṇa paryavasyati māntrike ।।

cārān vicārya daivajñair vaktavyaṃ bhūbhujāṃ phalam
graha-cāra-parijñānaṃ teṣām āvaśyakaṃ yatas ।।

putra ity eva pitari kanyakey eva mātari
garbha-praśneṣu kathayan daivajño vijayī bhavet ।।

āyus-praśne dīrgham āyur vācyaṃ mauhūrtikair janais
jīvanto bahumanyante mṛtāḥ prakṣyanti kaṃ punas ।।

sarvaṃ koṭi-dvayoetaṃ sarvaṃ kāla-dvayāadhi
sarvaṃ vyāmiśram iva ca vaktavyaṃ daiva-cintakais ।।

nirdhanānāṃ dhanāāptiṃ dhaninām adhikaṃ dhanam
bruvāṇāḥ sarvathā grāhyā lokair jyautiṣikā janās ।।

śatasya lābhe tāmbūlaṃ sahasrasya tu bhojanam
daivajñānām upālambho nityaḥ kārya-viparyaye ।।

api sāgara-paryantā vicetavyā vasuṃdharā
deśo hy aratni-mātre api nāsti daivajña-varjitas ।।

vārān ke cid grahān ke cit ke cid ṛkṣāṇi jānate
tritayaṃ ye vijānanti te vācas-patayaḥ svayam ।।

naimittikāḥ svapna-dṛśo devatānām [[amī trayaḥ]]
nisarga-śatravaḥ sṛṣṭā daivajñānām amī trayas ।।

svasthair asādhya-rogaiś ca jantubhir nāti kiṃ cana
kātarā dīrgha-rogāś ca bhiṣajāṃ bhāgya-hetavas ।।

nāidhairyaṃ pradātavyaṃ nāibhītiś ca rogiṇi
naiścintyān nāime dānaṃ nairāśyād eva nātime ।।

bhaiṣajyaṃ tu yathākāmaṃ pathyaṃ tu kaṭhinaṃ vadet
ārogyaṃ vaidya-māhātmyād anyathātvam apathyatas ।।

nidānaṃ roga-nāmāni sātmyāātmye cikitsitam
sarvam apy upadekṣyanti rogiṇaḥ sadane striyas ।।

jṛmbhamāṇeṣu rogeṣu mriyamāṇeṣu jantuṣu
roga-tattveṣu śanakair vyutpadyante cikitsakās ।।

pravartanātham ārambhe madhye tv auṣadha-hetave
bahumānātham ante ca jihīrṣanti cikitsakās ।।

lipsamāneṣu vaidyeṣu cirād āsādya rogiṇam
dāyādāḥ saṃprarohanti daivajñā māntrikā api ।।

rogasyoakrame sāntvaṃ madhye kiṃ cid dhana-vyayas
śanair anādaras śāntau snāto vaidyaṃ na paśyati ।।

daivajñatvaṃ māntrikatā bhaiṣajyaṃ cāṭu-kauśalam
ekaiam artha-lābhāya dvi-tri-yogas tu durlabhas ।।

anṛtaṃ cāṭuvādaś ca dhana-yogo mahān ayam
satyaṃ vaiduṣyam ity eṣa yogo dāridrya-kārakas ।।

kātaryaṃ durvinītatvaṃ kārpaṇyam avivekatām
sarvaṃ mārjanti kavayaḥ śālīnāṃ muṣṭi-kiṃkarās ।।

na kāraṇam apekṣante kavayaḥ stotum udyatās
kiṃ cid astuvatām teṣāṃ jihvā phuraphurāyate ।।

stutaṃ stuvanti kavayo na svato guṇa-darśinas
kītaḥ kaś cid alir nāma kiyatī tatra varṇanā ।।

ekaia kavitā puṃsāṃ grāmāyāśvāya hastine
antato annāya vastrāya tāmbūlāya ca kalpate ।।

śabdāhyam aparaṃ brahma saṃdarbheṇa pariṣkṛtam
vikrīyate katipayair vṛthāyair viniyujyate ।।

varṇayanti narāhāsān vāṇīṃ labdhvāi ye janās
labdhvāi kāma-dhenuṃ te lāṅgale viniyuñjate ।।

praśaṃsanto narāhāsān pralapanto anyathāyathā
kathaṃ tarantu kavayaḥ kāma-pāramya-vādinas ।।

yat sandarbhe yad ullekhe yad vyaṅgye nibhṛtaṃ manas
samādher api taj jyāyāḥ śaṃkaro yadi varṇyate ।।

[bandhavaḥ]

gṛhiṇī bhaginī tasyāḥ śvaśurau śyāla ity api
prāṇinām kalinā sṛṣṭāḥ pañca prāṇā ime apare ।।

jāmātaro bhāgineyā mātulā dāra-bāndhavās
ajñātā eva gṛhiṇāṃ bhakṣayanty ākhu-vad gṛhe ।।

mātulasya balaṃ mātā jāmātur duhitā balam
śvaśurasya balaṃ bhāryā svayam evāither balam ।।

jāmātur vakratā tāvad yāvac chyālasya bālatā
prabudhyamāne sāralyaṃ prabuddhe asmin palāyanam ।।

bhāryā jyeṣṭhā śiśuḥ śyālaḥ śvaśrūḥ svātantrya-vartinī
śvaśuras tu pravāsīi jāmātur bhāgya-dhoraṇī ।।

bhūṣaṇair vāsanaiḥ pātraiḥ putrāṇām upalālanais
sakṛd āgatya gacchantī kanyā nirmārṣṭi mandiram ।।

gṛhiṇī sva-janaṃ vakti śuṣkāāraṃ mitāśanam
pati-pakṣyāṃs tu bahv-āśān kṣīra-pāṃs taskarān api ।।

bhārye dve putra-śālinyau bhaginī pati-varjitā
aśrānta-kalaho nāma yogo ayaṃ gṛha-medhinām ।।

bhārye dve bahavaḥ putrā dāridryaṃ roga-saṃbhavas
jīrṇau ca mātā-pitarāv ekaikaṃ narakāhikam ।।

[uttama-ṛṇāḥ]

smṛte sīdanti gātrāṇi dṛṣṭe prajñā vinaśyati
aho mahad idaṃ bhūtam uttama-rṇāhiśābdhitam ।।

antako api hi jantūnām anta-kālam apekṣate
na kāla-niyamaḥ kaś cid uttamārṇasya vidyate ।।

na paśyāmo mukhe daṃṣṭrāṃ na pāśaṃ vā karāñjale
uttamārṇam avekṣyaia tathāy udvejite manas ।।

[dāridryam]

śatrau sāntvaṃ pratīkāraḥ sarva-rogeṣu bheṣajam
mṛtyau mṛtyuñjaya-dhyānaṃ dāridrye tu na kiṃ cana ।।

śaktiṃ karoti saṃcāre śītoṣṇe marṣayaty api
dīpayaty udare vahniṃ dāridryaṃ paramauṣadham ।।

giraṃ skhalantīṃ mīlantīṃ dṛṣṭiṃ pādau visaṃsthulau
protsāhayati yācñāyāṃ rājāñea daridratā ।।

jīryanti rāja-vidveṣā jīryanty avihitāny api
ākiṃcanya-balāḍhyānām antato aśmāi jīryati ।।

nāya corā na piśunā na dāyādā na pārthivās
dainyaṃ rājyād api jyāyo yadi tattvaṃ prabudhyate ।।

[dhaninaḥ]

prakāśayaty ahaṃkāraṃ pravartayati taskarān
protsāhayati dāyādāṃl lākṣmīḥ kiṃ cid upasthitā ।।

viḍambayanti ye nityaṃ vidagdhān dhanino janās
ta eva tu viḍambyante śriyā kim-cid-upekṣitās ।।

prāmāṇya-buddhiḥ stotreṣu devatā-buddhir ātmani
kīṭa-buddhir manuṣyeṣu nūtanāyāḥ śriyaḥ phalam ।।

śṛṇvanta eva pṛcchanti paśyanto api na jānate
viḍambanāni dhanikāḥ stotrāṇīy eva manvate ।।

āvṛtya śrī-madenādhān anyonya-kṛta-saṃvidas
svairaṃ hasanti pārśva-sthā bālomatta-piśāca-vat ।।

stotavyaiḥ stūyante nityaṃ sevanīyaiś ca sevyate
na bibheti na jihreti tathāi dhaniko janas ।।

kṣaṇa-mātraṃ grahāeśo yāma-mātraṃ surā-madas
lakṣmī-madas tu mūrkhāṇām ā-deham anuvartate ।।

śrīr māsam ardha-māsaṃ vā ceṣṭitvā vinivartate
vikāras tu tad-ārabdho nityaṃ laśuna-gandha-vat ।।

kaṇṭḥe madaḥ kodrava-jo hṛdi tāmbūla-jo madas
lakṣmī-madas tu sarvāṅge putra-dāra-mukheṣv api ।।

yatrāīd asti vā lakṣmīs tatromadaḥ pravartatām
kule apy avataraty eṣa kuṣṭhāasmāra-vat katham ।।

adhyāpayanti śāstrāṇi tṛṇīkurvanti paṇḍitān
vismārayanti jātiṃ svāṃ varāṭāḥ pañcaṣā kare ।।

bibhartu bhṛtyān dhaniko dattāṃ vā deyam arthiṣu
yāvad yācaka-sādharmyaṃ tāval loko na mṛṣyati ।।

[piśunāḥ]

dhana-bhāro hi lokasya piśunair eva dhāryate
kathaṃ te taṃ laghūkartuṃ yatante aparathā svatas ।।

śramāurūpaṃ piśune kim upakriyate nṛpais
dvi-guṇaṃ tri-guṇaṃ caia kṛtāto lālayiṣyati ।।

Gokarṇe Bhadrakarṇe ca japo duṣkarma-nāśanas
rāja-karṇe japaḥ sadyaḥ sarva-karma-vināśanas ।।

na svāthaṃ kim-cid icchanti na preryante ca kena cit
parātheṣu pravartante śaṭhāḥ santaś ca tulya-vat ।।

kālātare hy anarthāya gṛdhro gehoari sthitas
khalo gṛha-samīpa-sthaḥ sadyo anarthāya dehinām ।।

[lobhinaḥ]

śuṣkoavāso dharmeṣu bhaiṣajyeṣu ca laṅghanam
japa-yajñaś ca yajñeṣu rocate lobha-śālinām ।।

kiṃ vakṣyatīa dhanikād yāvad udvijate adhanas
kiṃ prakṣyatīi lubdho api tāvad udvijate tatas ।।

sarvam ātithya-śāstrāthaṃ sākṣāt kurvanti lobhinas
bhikṣākabalam ekaiaṃ ye hi paśyanti meru-vat ।।

dhana-pālaḥ piśāco hi datte svāminy upasthite
dhana-lubdhaḥ piśācas tu na kasmai cana ditsate ।।

dātāro arthibhir arthyante dātṛbhiḥ puno arthinas
kartṛ-karma-vyatīhārād aho nimnonataṃ kiyat ।।

svasminn asati nāthasya rakṣakaḥ saṃbhaved iti
niścityaiaṃ svayam api bhuṅkte lubdhaḥ kathaṃ cana ।।

prasthāsyamānaḥ praviśet pratiṣṭheta dine dine
vicitrān ullikhed vighnāṃs tiṣṭhāsur atithiś ciram ।।

[dhārmikāḥ]

pradīyate viduṣy ekaṃ kavau daśa naṭe śatam
sahasraṃ dāmbhike loke śrotriye tu na kim-cana ।।

ghaṭakaṃ samyag-ārādhya vairāgyaṃ paramaṃ vahet
tāvad arthāḥ prasiddhyanti yāvac cāpalam āvṛtam ।।

ekataḥ sarva-śāstrāṇi tulasī-kāṣṭham ekatas
vaktavyaṃ kiṃ cid ity uktaṃ vastutas tulasī parā ।।

vismṛtaṃ vāhaṭeneaṃ tulasyāḥ paṭhatā guṇan
viśva-saṃmohinī vitta- dāyinīi guṇa-dvayam ।।

kaupīnaṃ bhasitāepo darbhā rudrāṣa-mālikā
maunam ekāikā cei mūrkha-saṃjīvanāni ṣaṭ ।।

vāsaḥ puṇyeṣu tīrtheṣu prasiddhaś ca mṛto gurus
adhyāpanāṛttayaś ca kīrtanīyā dhanāthibhis ।।

mantra-bhraṃśe saṃpradāyaḥ prayogaś cyuta-saṃkṛtau
deśa-dharmas tv anācāre pṛcchatāṃ siddham uttaram ।।

yathā jānanti bahavo yathā vakṣyanti dātari
tathā dharmaṃ caret sarvaṃ na vṛthā kiṃ cid ācaret ।।

sadā japa-paṭo haste madhye madhye akṣi-mīlanam
sarvaṃ brahmei vādaś ca sadyas-pratyaya-hetavas ।।

ā-madhyānaṃ nadī-vāsaḥ samāje devatācanam
satataṃ śuci-veṣaś cety etad dambhasya jīvitam ।।

tāvad dīrghaṃ nitya-karma yāvat syād draṣṭṛ-melanam
tāvat-saṃkṣipyate sarvaṃ yāvad draṣṭā na vidyate ।।

ānanda-bāṣpa-romāñcau yasya svecchā-vaśaṃvadau
kiṃ tasya sādhanair anyaiḥ kiṃkarāḥ sarva-pārthivās ।।

[durjanāḥ]

daṇḍyamānā vikurvanti lālyamānās tatas tarām
durjanānām ato nyāyyaṃ dūrād eva visarjanam ।।

adānam īṣad-dānaṃ ca kim-cit-kopāya durdhiyām
saṃpūrṇa-dānaṃ prakṛtir virāmo vaira-kāraṇam ।।

jyāyān asaṃstavo duṣṭair īrṣyāyai saṃstavaḥ punas
apatya-saṃbandha-vidhiḥ svāarthāyaia kevalam ।।

jñāteaṃ jñāna-hīnatvaṃ piśunatvaṃ daridratā
milanti yadi catvāri tad diśe api namo namas ।।

para-chidreṣu hṛdayaṃ para-vārtāsu ca śravas
para-marmāsu vācaṃ ca khalānām asṛjad vidhis ।।

viṣeṇa puccha-lagnena vṛścikaḥ prāṇinām iva
kalinā daśamāṃśena sarvaḥ kālo api dāruṇas ।।

yatra bhāryā-giro vedā yatra dharmo artha-sādhanam
yatra sva-pratibhā mānaṃ tasmai śrī-kalaye namas ।।

kāmam astu jagat sarvaṃ kālasyāya vaśaṃvadam
kāla-kālaṃ prapannānāṃ kālaḥ kiṃ naḥ kariṣyati? ।।

kavinā nīlakaṇṭhena kaler etad viḍambanam
racitaṃ viduṣāṃ prītyai rājāsthānāumodanam ।।

iti Nīlakaṇṭha-viracitaṃ Kaliviḍambanaṃ saṃpūrṇam
