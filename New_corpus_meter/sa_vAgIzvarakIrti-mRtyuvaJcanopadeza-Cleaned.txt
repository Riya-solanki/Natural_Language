Vāgīśvarakīrti Mṛtyuvañcanopadeśa

।। oṃ namaḥ sarvajñāya ।।

adhigataparamārthaḥ śrīghanavyūha eva prati tuṣitajanārthaṃ yo abhavac chvetaketuḥ 
tata iha jagadarthaṃ cābhavac chākyaketuḥ sa jayatu jitamṛtyur darśitānantamāyaḥ ।। 

ajarāmaratāṃ prāptam ajarāmarakārakam 
ajarāmaratāprāptyai natvā saṃbuddham ādarāt ।। 

durbodhaṃ mandabuddhīnāṃ duṣkaraṃ calacetasām 
sāriṣṭaṃ vañcanaṃ mṛtyor ucyate śāstrasaṃgatam ।। 

sarvathā vañcanaṃ mṛtyor muktānām eva yujyate 
amuktānāṃ ca sarveṣāṃ kiyatkālavilambanam ।। 

āyurindriyacittānāṃ nirodho mṛtyulakṣaṇam 
upāyair jīvitaṃ dīrghaṃ mṛtyuvañcanam ucyate ।। 

yāvaj jīvati saṃsāre dharmācārī jitendriyaḥ 
tāvat puṇyādisaṃbhāraṃ prasūte bodhisādhakam ।। 

na vinā jīvitāt sāraṃ saṃsāre kiṃ cid īkṣyate 
nānopāyair atas tūrṇaṃ mṛtyuvañcanam ārabhet ।। 

na duṣkaraṃ yataḥ kiṃ cid upāyair yuktiyojitaiḥ 
maṇimantrauṣadhīnāṃ ca prabhāvaḥ sarvasaṃmataḥ ।। 

lakṣaṇād dūrato mṛtyuṃ jñātvā yuktā pratikriyā 
maste patati vajrāgnau suprajño api karoti kim ।। 

bāhyādhyātmikabhedena dvidhā maraṇalakṣaṇam 
kathyate jñāyate yena maraṇaṃ bhāvi tatparaiḥ ।। 

mṛtyor lakṣaṇam ājñātuṃ vāñchā cet kasya cid bhavet 
tena svasthaśarīreṇa jñātavyaṃ saṃśayo anyathā ।। 

dhātuvaiṣamyato rogāt timirādiprabhāvataḥ 
bhayaśokādisāmarthyāt pratibhāti yato anyathā ।। 

daivabhūtobhayakṣobhān mṛtyuṃ trividhalakṣaṇam 
prāpyāriṣṭasamudbhūtir ato bhedena lakṣayet ।। 

bāhyābhyantarabhedena daivakṣobho dvidheṣyate 
bhūtakṣobho api dvividhas tathaiveti nirūpayet ।। 

daivasaṃkṣobhato mṛtyur ādhidaivika ucyate 
daivakṣobho anaśanādivajrapātādilakṣaṇaḥ ।। 

bhūtasaṃkṣobhato mṛtyur ādhibhautika ucyate 
bhūtakṣobhas tu pittādivahnitoyādyupadravaḥ ।। 

ubhayakṣobhato mṛtyur ubhayājjo nigadyate 
ubhayakṣobho vajrādipittaśleṣmādyupadravaḥ ।। 

ariṣṭānām anantatvān notpattir yugapan matā 
kramaśo api na sarveṣām api tv ekādisaṃkhyayā ।। 

adṛṣṭvā lakṣaṇaṃ yatra mṛtyur dṛṣṭaḥ kva cid bhavet 
niścetur dūṣaṇaṃ tatra na tu lakṣaṇanāstitā ।। 

dṛṣṭvāpi lakṣaṇaṃ yatra na mṛtyur dṛśyate kva cit 
abhrāntau tatra dharmādibādhakāntarakalpanam ।। 

niścite dūrato mṛtyau prasiddhir mahatī bhavet 
yathāśakti parityāge pratīkāre ca vartanam ।। 

sadā samāhito bhūtvā tadgataikāgramānasaḥ 
bāhyāni prathamaṃ tāval lakṣaṇāny upalakṣayet ।। 

lakṣaṇānāṃ paricchedakālo dvividha iṣyate 
niyato aniyataś ceti vijñeyaḥ sa tu yatnataḥ ।। 

saṃkrāntipūrṇamāsyādau prabhātādikṣaṇātmakaḥ 
kālo niyata ity ukto aniyatas tu viparyayāt ।। 

sphuratsaṃdhi kapolaṃ syād atyarthaṃ mṛtyudarśanāt 
vātaśītādisaṅgāc ca lakṣaṇīyam ataḥ pṛthak ।। 

karkaṭo rāśir āṣāḍhaḥ puṣyas tu makaro mataḥ 
karkaṭe makare cāpi jāyate mṛtyulakṣaṇam ।। 

āṣāḍhasaṃkrāntidine madhyāhne padamātrikā 
uttarābhimukhasthasya cchāyā svābhāvikī bhavet ।। 

puṣyasaṃkrāntidivase saiva saptapadībhavet 
vyabhre nabhasi madhyāhne cchāyāyāś ca nirīkṣaṇam ।। 

pratimāsaṃ padavṛddhiḥ pratimāsaṃ padakṣayaḥ 
amuṣmād anyathābhāve mṛtyuśaṅkā prajāyate ।। 

ekaikapadavṛddhiḥ syād āṣāḍhān makarāvadheḥ 
ekaikapādyahrāsaḥ syān makarāt karkaṭāvadheḥ ।। 

āṣāḍhāt kramaśo vṛddhiḥ puṣyād dhrāsaḥ kramāt tathā 
chāyāvikāram ālocya māsān maraṇam ādiśet ।। 

kṣayakāle bhaved vṛddhir vṛddhikāle kṣayo yadi 
tam eva kālam ārabhya mṛtyur māsād viniścitaḥ ।। 

vṛddhikāle apy ativṛddhau nānārogād upadravaḥ 
kṣayakāle apy atikṣaye mṛtyur eva na saṃśayaḥ ।। 

āṣāḍhapuṣyayor ante pūrvaṃ karkaṭamatsyayoḥ 
yāni ṣaṭ ṣaḍ dināny atra kālākālanirūpaṇam ।। 

āṣāḍhāntadinair ebhiḥ śrāvaṇādau śubhāśubham 
puṣyaśeṣadinaiś cāpi māghādiṣu śubhāśubham ।। 

pādayos tālukāṃ viddhvā nābhau vedho yadā bhavet 
ahorātratrayād ūrdhvaṃ pañcatvaṃ jāyate tadā ।। 

pādayos tālukāṃ viddhvā yadi cakṣuś ca vidhyati 
tadā māsatrayād ūrdhvaṃ gamanaṃ yamasadmani ।। 

pādayos tālukāṃ viddhvā nāsikāṃ yadi vidhyati 
tribhir dinais tadā nūnam ekākī yāti pañcatām ।। 

kuṭiprasrāvayoḥ kāle tulyaṃ syād yadi hañchikā 
tasyām eva hi velāyāṃ mṛtyur varṣeṇa tasya hi ।। 

nāsikāgaddrikāvedhe maraṇaṃ pañcavatsaraiḥ 
jihvāgrādarśane mṛtyur vāsaraiḥ syāt tribhir nṛṇām ।। 

māsaiś caturbhir mṛtyuḥ syāt karṇāgre tīkṣṇavedhanāt 
sadyo mṛtyur bhaven nūnam ūrṇāsthāne pravedhanāt ।। 

tulyakālaṃ yadā vedhaś candrasūryagato bhavet 
tadā māsād bhaven mṛtyuḥ puṇyam evārjayet tataḥ ।। 

bhagaliṅgasamāyoge madhye śeṣe ca hañchikā 
syāc cet tulyaṃ tadā māsaiḥ pañcatvaṃ yāti pañcabhiḥ ।। 

hastayoḥ pādayoś caiva kaniṣṭhānāṃ ca saṃdhayaḥ 
catvāro yasya vidhyanti tulyaṃ māsāt sa mṛtyubhāk ।। 

hṛtkaṇṭhamadhyayor vedhas tulyakālaṃ yadā bhavet 
pakṣatrayeṇa mṛtyuḥ syād yadi dharmaṃ na sevate ।। 

tulyakālaṃ yadā vidhyet tālukānāṃ trayaṃ muhuḥ 
tribhir dinais tadā mṛtyur yadi śakrasamo bhavet ।। 

vāmākṣiputtalīcchāyāṃ yo na paśyati darpaṇe 
saptāhān mriyate nūnaṃ yadi na syāt pratikriyā ।। 

maithune sati ghaṇṭāyā nādaś cet karṇayor bhavet 
tribhir māsais tadā mṛtyur yadi brahmasamo bhavet ।। 

kṛṣṇaṃ yadi bhavec chukraṃ śuklāyāṃ pratipattithau 
ṣaḍbhir māsais tadā mṛtyur lohitaṃ vyādhisūcakam ।। 

karṇamūle bhruvor madhye mastakāgre pṛthakpṛthak 
yadi vedho apratīkāro mṛtyuḥ sadyas tadā bhavet ।। 

pādāṅguṣṭhaṃ samārabhya nābhiparyantago bhavet 
vedhaḥ sadāho yasyāsau ṣaḍmāsān mriyate dhruvam ।। 

pādāṅguṣṭhād dhṛdayāntaṃ yāvat kaṇṭhaṃ śiro athavā 
māsatrayāt pakṣatrayān mṛtyur ekadināt kramāt ।। 

nāsāgramāṃsaśaithilyān mṛtyuḥ syāt saptarātrataḥ 
kapolamāṃsavicchedān mṛtyuḥ syāt pañcamāsataḥ ।। 

cakṣurnāsikayor madhye spandanaṃ cen na dṛśyate 
jñātvā pañcadinān mṛtyum ādadyād dharmasaṃvalam ।। 

netrayos tārake mliṣṭe sahasā bhavato yadi 
ṣaḍmāsābhyantare mṛtyuḥ pratikārakriyā na cet ।। 

akasmān nāsikā vakrā karṇau bhraṣṭau svadeśataḥ 
netre ca vartulībhūte saptāhān mṛtyudarśanāt ।। 

akasmād dhṛdayaṃ nimnaṃ syāc cet pakṣāt tadā mṛtiḥ 
grīvāpārśvasthayor nāḍyor vicchedād api pakṣataḥ ।। 

akasmāt kṛṣṇarekhā syāj jihvāyā yadi madhyataḥ 
dṛḍho vā dantasaṃdaṃśas tadā mṛtyur dvirātrataḥ ।। 

karṇaśaṣkulikāpṛṣṭhanāḍīvicchedadarśanāt 
tasminn eva dine mṛtyur athavā pañcavāsaraiḥ ।। 

nakheṣu raktavarṇeṣu sahasā yady araktatā 
ṣaḍbhir māsais tadā mṛtyur yadi mantrādi nācaret ।। 

akasmāj jāyate sthūlaḥ kṛśaḥ kruddho bhayākulaḥ 
yas tasya mṛtyur varṣeṇa yadi dharmaṃ na sevate ।। 

indriyāṇi na gṛhṇanti yathāsvaṃ yasya gocarān 
muhuḥ skhalati vāṇī ca muhuś ca mativibhramaḥ ।। 

cakṣuṣī sravato nityaṃ dṛṣṭarūpe api vibhramaḥ 
darpaṇe salile vāpi svacchāyāṃ yo na paśyati ।। 

rātrāv indradhanuḥ paśyed divā nakṣatramaṇḍalam 
ameghe vidyutaṃ paśyet sphurantīṃ dakṣiṇāśritām ।। 

divā chāyāpathaṃ paśyed ulkāyāḥ patanaṃ tathā 
haṃsakākamayūrāṇāṃ paśyed ekatra melakam ।। 

candradvayaṃ dvisūryaṃ vā svaśirojvalanaṃ tathā 
gandharvanagaraṃ paśyed vṛkṣāgre śikhare gireḥ ।। 

paśyet pretapiśācān adṛśyān anyāṃś ca bhīṣaṇān 
prakampate bhṛśaṃ caiva mūrchito vā bhaven muhuḥ ।। 

chardiṃ mūtraṃ purīṣaṃ ca suvarṇarajataprabham 
paśyed ekaikaśas tasya mṛtyur māsāvadher bhavet ।। 

kaṇṭḥoṣṭhatālujihvānāṃ chedaṃ chidraṃ tanau tathā 
yaḥ paśyet tasya mṛtyuḥ syāt ṣaḍmāsābhyantare dhruvam ।। 

kaṇṭhoṣṭhatālurasanādantā yasya pṛthakpṛthak 
śuṣyanty abhīkṣṇaṃ ṣaḍmāsān maraṇaṃ tasya nirdiśet ।। 

nakṣatracakrasaptarṣidiśāṃ rātrāv adarśanam 
saptāhamṛtyor jāyeta jvālā vākasmikī tanau ।। 

arundhatīṃ rohiṇīṃ ca dhruvaṃ chāyāpathaṃ tathā 
rātrāv apaśyato akasmāt ṣaḍmāsān mṛtyum ādiśet ।। 

kalaṅkarahitaṃ candraṃ sūryaṃ raśmivivarjitam 
saraśmikaṃ citrabhānuṃ candraṃ cāpi saraśmikam ।। 

rātrau sūryaṃ divā candraṃ svanetre jvalanaṃ tathā 
tārāṃ merupramāṇāṃ ca samudraṃ ca nadīm iva ।। 

nābhau hikkāṃ gude hañchāṃ varṇaṃ padmopamaṃ mukhe 
galake piṭakān raktān gātre varṇavicitratām ।। 

hṛdaye krandanaṃ sāśru prakampam atha tāluni 
candrāṃśuṃ vahnisaṃsparśaṃ sūryāṃśuṃ himaśītalam ।। 

candracchidraṃ ravicchidraṃ chidraṃ bhūmau tathāmbare 
ātmanaiva hi yaḥ paśyet paśyen mṛtyuṃ sa pakṣataḥ ।। 

mūtraśukrapurīṣāṇi tulyakālaṃ patanti cet 
varṣān mṛtyur bhavet tatra bhaiṣajyādi pratikriyā ।। 

ardhaṃ śītaṃ tathā coṣṇam ardhaṃ yasya kalevaram 
saptarātrād bhaven mṛtyuḥ paralokaṃ vicintayet ।। 

hakāraḥ śītasaṃsparśaḥ phukāro vahnisaṃnibhaḥ 
yasyānubhūyate tasya daśāhād yamadarśanam ।। 

anāmikānāṃ mūleṣu dṛṣṭayā kṛṣṇarekhayā 
aṣṭādaśadinād ūrdhvaṃ mṛtyuḥ prākkarmadoṣataḥ ।। 

yasya mastakam āruhya trivarṇo yāti vegataḥ 
kṛkalāso mṛtis tasya pañcamāsād viniścitā ।। 

śabdo na śrūyate dehād dhastābhyām apamārjitāt 
sarvāṅgaśītalatvena daśāhān maraṇaṃ dhruvam ।। 

yugapan nirgamo bāhujaṅghayor yasya gacchataḥ 
vināśo brahmasūtrasya mṛtyus tasyaikavarṣataḥ ।। 

kardame pāṃśudeśe vā purataḥ pṛṣṭhato api vā 
khaṇḍaṃ pādodaye nyūnaṃ mṛtyur māsacatuṣṭayāt ।। 

akasmāt puruṣaṃ paśyet kṛṣṇapiṅgalavarṇakam 
varṣatrayātyayān mṛtyur yadi bhrāntyā na kalpitaḥ ।। 

dvimāsamṛtyur hṛtpādau snātamātrasya śuṣyataḥ 
trirātramṛtyur durgandho jāyate vikṛtākṛtiḥ ।। 

kharārkadivase toyair āpūrya mukhakoṭaram 
pratyahaṃ phūtkṛtaṃ kurvan kramāt tanmadhyasaṃsthitam ।। 

paśyed indradhanuś citraṃ dīrghāyuḥ puruṣaḥ sadā 
ṣaḍmāsāyur na tat paśyed iti karmavicitratā ।। 

kharārkadivase paśyet puro gaganamadhyataḥ 
muktāhārān ivālūnān muktāphalavibhūṣaṇān ।। 

indranīlamaṇicchāyān keśoṇḍukasamākṛtīn 
dṛśyadeśaṃ parityajya deśāntaravisarpiṇaḥ ।। 

nāgān anekaśaḥ paśyet saṃyatān iva saṃtatam 
teṣām adṛṣṭau ṣaḍmāsān maraṇaṃ parikīrtitam ।। 

prabhāte vātha sāyāhne jyotsnāyāṃ vā ciraṃ puraḥ 
vitatya bāhū svacchāyāṃ dṛṣṭvā paśyec chanair nabhaḥ ।। 

tatrāpi dṛśyate chāyā dhavalā nararūpiṇī 
śiraso adarśanāt tasya mṛtyuḥ syād varṣamadhyataḥ ।। 

putrabhāryāvināśaḥ syād vāmapāṇer adarśanāt 
dakṣiṇādarśanāt pitṛbhrātrādīnāṃ mahīyasām ।। 

jānūpari sthitau bāhū kṛtvā mūrdhni tathāñjalim 
rambhāphalanibhāṃ chāyāṃ lakṣayen madhyatas tayoḥ ।। 

yadā tv ekaṃ dalaṃ tatra vikāsi parilakṣyate 
tasyām eva tithau nūnaṃ ṣaḍmāsān maraṇaṃ bhavet ।। 

sūryamārgagate vāyau lakṣaṇīyam idaṃ trayam 
śrotre śirasi netre ca śabdo dhūmo dyutis tathā ।। 

karṇau pidhāya sudṛḍhaṃ gambhīro gulgulo dhvaniḥ 
na śrūyate pañcadinaṃ daśa pañcadaśāthavā ।। 

viṃśatiṃ divasān yāvat pañcaviṃśatim eva vā 
bāṇābdhiguṇanetrenduvarṣair mṛtyur yathākramam ।। 

eṣām apy antarāleṣu śodhyāḥ ṣoḍaśa vāsarāḥ 
varṣasyaiva caturthāṃśaiḥ kramaśaś cābhivardhitaiḥ ।। 

mastakopari niryāntīṃ saptāhaṃ dhūmamālikām 
yo na paśyati tasya syān mṛtyur varṣatrayāvadheḥ ।। 

svanetrayoś caturdikṣu dṛśyante aṅgulipīḍanāt 
mārjāranetratulyāni jyotīṃṣi jhaṭitikramāt ।। 

apīḍane api taddṛṣṭau mṛtyuḥ syāc chatavāsaraiḥ 
etat sāmānyato jñeyaṃ viśeṣas tv ayam ucyate ।। 

candrasyādhas tathordhvaṃ ca karṇanāsopakaṇṭhayoḥ 
naṣṭe jyotiṣi mṛtyuḥ syāt ṣaṭtrinetrendumāsataḥ ।। 

sūrye api yadi candrasya kramaḥ syāt pūrvavat tadā 
digbāṇaguṇanetreṣu dineṣu maraṇaṃ bhavet ।। 

svarūpasya viparyāsāt ṣaḍmāsān maraṇaṃ bhavet 
pañcamāsād bhaven mṛtyur nāsāgrasya vināśataḥ ।। 

bādhirye śrotrayor mṛtyuś caturmāsātyayād bhavet 
svasthendriyasya patane mṛtyur māsatrayād bhavet ।। 

bhrūmadhye jyotiṣo adṛṣṭau dvimāsād yamadarśanam 
ekamāsātyayān mṛtyur vṛṣaṇe golakakṣayāt ।। 

parākṣṇoḥ pratibimbasyādṛṣṭau pakṣād vināśanam 
jihvāyā aparāvṛttau daśāhān mṛtyusaṃgamaḥ ।। 

nābher viparyayān mṛtyuḥ pañcāhād upaśliṣyati 
pārśvadvayasyākṛṣṭau tu mṛtyuḥ syād vāsaratrayāt ।। 

samastagātrastabdhatve mṛtyur ekāhiko bhavet 
lalāṭasthatrirekhāṇāṃ nāśān mṛtyur ahastrayāt ।। 

svasthasya lālājihvā cen niḥsparśā dravavarjitā 
yāvat pañcadinaṃ mṛtyuḥ pañcāśaddivasais tathā ।। 

amlāditvaṃ ca rasato nīlāditvaṃ ca varṇataḥ 
vikāraḥ śukramūtrāṇāṃ ṣaḍmāsān mṛtyudarśanāt ।। 

yugapatpañcadhāraṃ vā vāmāvartaṃ vigandhi ca 
mūtraṃ yasya bhaven mṛtyus tasya ṣaḍmāsamadhyataḥ ।। 

kṛṣṇe vā yadi vā śukle pakṣe reto atikṛṣṇakam 
mṛtyuḥ ṣaḍmāsato no cet tenaiva syāt pratikriyā ।। 

kṛṣṇabhāgaṃ parityajya gṛhītvā śuklabhāgakam 
candre prabhañjane vāti praveśāt syāt pratikriyā ।। 

sthiratve api svadehasya tacchāyā cañcalā yadi 
caturmāsād bhaven mṛtyur ity āgamavicakṣaṇāḥ ।। 

dakṣiṇāśāgatāṃ chāyām ātmano yadi paśyati 
adyaiva mṛtyur asmākam iti paśyed anityatām ।। 

nirarthako visaṃvādī mithyetyādi prakāśanāt 
svapno apy asatyaḥ sarveṣāṃ nātaḥ śraddhā na yujyate ।। 

devatādarśitaḥ svapnaḥ satyasvapno atha yo naraḥ 
sa svapne pratyayaṃ kṛtvā mṛtyuliṅgāni paśyatu ।। 

kiṃśukaṃ kovidāraṃ ca puṣpitaṃ karavīrakam 
yo abhirohati svapnānte sa syāt ṣaḍmāsamṛtyukaḥ ।। 

vihārayaṣṭiṃ svapnānte yo abhirohati mānavaḥ 
vālukābhasmarāśiṃ vā maraṇaṃ tasya pūrvavat ।। 

gardabhaṃ yaḥ samārūḍhaḥ prayāyād dakṣiṇāṃ diśam 
bhūyaś ca na nivarteta tadvat tasyāpi jīvitam ।। 

yaś cāpi vānarārūḍhaḥ prayāyād dakṣiṇāṃ diśam 
tatrasthaś ca vibudhyeta mṛtyuḥ syāt tasya pūrvavat ।। 

daṇḍaṃ vā caityayūpaṃ vā naro yas tv abhirohati 
vālmīkaṃ pāṃśurāśiṃ vā ṣaḍmāsān na sa jīvati ।। 

kālī kumārī saṃkruddhā yā tu badhnāti bandhanaiḥ 
kālarātrī tu sā jñeyā saptāhān mṛtyukārikā ।। 

kṛṣṇavastrā tu yā nārī kālī kāmayate naraṃ 
karavīramālā svapne yāty asau yamamaṇḍalam ।। 

tamo vā praviśet svapne śvabhraṃ vā cārakaṃ tathā 
vṛkṣād vā prapatet svapne sa mṛtyor antike sthitaḥ ।। 

vṛkṣaṃ tṛṇaṃ vā kāṣṭhaṃ vā viphalaṃ yas tu paśyati 
suptaḥ śīrṣe śarīre vā ṣaḍmāsān na sa jīvati ।। 

yo vānararathārūḍho gacchet pūrvāṃ diśaṃ naraḥ 
pratibuddho vijānīyād rātrir eṣā mamāntimā ।। 

śvakākagṛdhragomāyurakṣaḥpretapiśācakaiḥ 
bhakṣyate prohyate cāpi kharapotryuṣṭrasaurabhaiḥ ।। 

ekābdavigame nūnaṃ maraṇaṃ tasya nirdiśet 
svapnadṛṣṭanimittānāṃ śāntir māyopamā matiḥ ।। 

lohadaṇḍadharaṃ kṛṣṇaṃ naraṃ kṛṣṇaparicchadam 
prasuptenāgrato dṛṣṭvā trimāsān mṛtyur āpyate ।। 

raktagandhapraliptāṅgaṃ raktamālyavibhūṣitam 
tailābhyaktam atibhītaṃ muṇḍitaṃ raktavāsasam ।। 

kharam āruhya vegena vrajantaṃ dakṣiṇāṃ diśam 
ātmānam īdṛśaṃ svapne paśyet ṣaḍmāsamṛtyukaḥ ।। 

raktamālyāni gandhāṃś ca raktāmbaram athāpi vā 
yaḥ svapne labhate tasya mṛtyuḥ syād aṣṭamāsataḥ ।। 

saṃkhyātītāni bāhyāni mṛtyuliṅgāni tāni kaḥ 
ekavaktreṇa śaknoti vaktuṃ varṣaśatair api ।। 

etāni mṛtyoḥ pratipādakāni liṅgāni bāhyāni yathoditāni 
vijñāya tāni tvaritaṃ vidheyo mantrādibhir vañcana eva yatnaḥ ।। 

।। iti śrīpaṇḍitavāgīśvarakīrtiviracite mṛtyuvañcanopadeśe prathamaḥ paricchedaḥ ।।

idānīm āntaraṃ mṛtyor lakṣaṇaṃ kathyate sphuṭam 
tac ca vāyūdayāj jñeyaṃ vāyur ghrāṇāsyagocaraḥ ।। 

bāhyalakṣaṇamātrān na mṛtyoḥ samyag viniścayaḥ 
yāvad ādhyātmikaṃ samyag lakṣaṇaṃ naiva niścitam ।। 

ubhayor niścayān mṛtyur niścitaḥ syāt parisphuṭam 
bāhye buddhiviparyāso rogāder api saṃskṛteḥ ।। 

puṣyādimāsasaṃkrāntau pūrṇamāsyādiparvasu 
sāmānyalakṣaṇaṃ mṛtyor lakṣayet prātarāditaḥ ।। 

nirūpakanaro yatra dine kālaṃ nirūpayet 
tad ārabhya dinaṃ mṛtyur mitakālātyayād bhavet ।। 

samaprakṛtiyuktānāṃ sarvaprāṇabhṛtāṃ sadā 
nāsikādakṣiṇetare puṭe pratyekam āśritaḥ ।। 

vāyur vahati yāmārdhaṃ sūryaśītāṃśusaṃjñake 
ato viparyayāt kalpyaṃ maraṇādi manīṣibhiḥ ।। 

pratipadaṃ samārabhya śuklāṃ vahati candragaḥ 
dināni trīṇi pavanaḥ tataḥ sūrye dinatrayam ।। 

dinatrayaṃ punaś candre tataḥ sūrye dinatrayam 
krameṇānena niyamād yāvat pañcadaśī sitā ।। 

evaṃ kṛṣṇaṃ samārabhya sūrye vahati mārutaḥ 
dināni trīṇi pūrvavad yāvat pañcadaśītarā ।। 

kramasyāsya viparyāsāt tribhiḥ pakṣaiḥ suniścitāt 
ṣaḍbhir māsair bhaven mṛtyur dharmam eva smaret tataḥ ।। 

pakṣadvayaviparyāsāt suhṛdbandhuvipad bhavet 
ekapakṣaviparyāsād dāruṇavyādhisaṃbhavaḥ ।। 

ekadvitricatuḥpañcaṣaḍdināni viparyayāt 
vahed vāyur yadi tadā kalyudvegādi jāyate ।। 

trimārgago bhaved vāyuḥ prakṛteḥ parivarjanāt 
madhyāhnāt parato mṛtyuḥ pratīkāravivarjitaḥ ।। 

saṃkrāntirahito vāyur ubhayor api mārgayoḥ 
vahed yasya daśāhāni saṃkrāntau mriyate hi saḥ ।। 

viṣuvatkṣaṇasaṃprāptau spandete yasya locane 
tasya nāśaṃ vijānīyād ahorātrān na saṃśayaḥ ।। 

itaś cetaś ca bahudhā yasyārdhapraharaṃ vahet 
vāyus tasya vijānīyāt pūjanaṃ lābham eva ca ।। 

saṃkrāntipañcakaṃ yasya samatītya mukhe vahet 
vāyus tasyārthanāśaḥ syād udvego vā bhayādi vā ।। 

vāmanāsāpuṭe vāyuḥ saṃkrāntīś ca trayodaśa 
samatītya vahed yasya tasya rogādi jāyate ।। 

puṭadvayaṃ parityajya yadā vaktreṇa gacchati 
tad ahar jīvitaṃ tasya yadi vajrasamo apy asau ।। 

mārgaśīrṣasya saṃkrāntikālam ārabhya mārutaḥ 
pañcāhaṃ ced vahet sūrye mṛtyur aṣṭādaśābdataḥ ।। 

aśvayuṅmāsasaṃkrāntikālam ārabhya mārutaḥ 
pañcāhaṃ ced vahen mṛtyur bhavet pañcadaśābdataḥ ।। 

śrāvaṇasyāpi ced evaṃ mṛtyur dvādaśavarṣataḥ 
jyeṣṭhe api ced bhaved evaṃ mṛtyuḥ syān navavarṣataḥ ।। 

caitramāsasya saṃkrāntikālam ārabhya ced vahet 
pañcāhaṃ māruto mṛtyuḥ ṣaḍbhir varṣais tadā bhavet ।। 

māghamāsasya saṃkrānteḥ pañcāhaṃ ced vahen marut 
varṣatrayātyayān mṛtyur iti kālavicakṣaṇāḥ ।। 

sarvatra dvitricaturo vāyuś ced divasān vahet 
abdabhāgais tu te śodhyā yathāvad anupūrvaśaḥ ।। 

evam eva vahec candre vāyuś cen mṛtyuvarjitāḥ 
vyādhidurbhikṣakāntāraśokarājādyupadravāḥ ।। 

sāmānyakālasaṃbodhaḥ sphuṭo ayam upadarśitaḥ 
viśeṣakālabodhāya gatir anyā pradarśyate ।। 

samasaptagate sūrye janmarkṣe candramā yadi 
kālo asau pauṣṇanāmeti mṛtyunirṇayakārakaḥ ।। 

yatra rāśau naro jātas tasmād yaḥ saptamo aparaḥ 
samasapta iti khyātas tatrārkaḥ samasaptagaḥ ।। 

rāśau tatraiva candraś cej janmanakṣatrago bhavet 
pauṣṇaḥ kālaḥ sa vijñeyas tatra mṛtyor nirūpaṇam ।। 

sarvatra sūryamārgāntargate satatagāmini 
kālaṃ nirūpayed vidvān itaratra vikālakam ।। 

yatra velākṣaṇe vāyor gatir anyā pravartate 
tatra velākṣaṇe pūrṇe mṛtyur nāsty atra saṃśayaḥ ।। 

pauṣṇe kāle vahed vāyur dinārdhaṃ yadi sūryagaḥ 
vatsarair manusaṃkhyātair lakṣayen maraṇaṃ tadā ।। 

dinam ekam ahorātraṃ dinadvitricatuṣṭayam 
vahec ced arkadikśailaṣaḍvedābdaiḥ kramān mṛtiḥ ।। 

pañcāhaṃ ca daśāhaṃ ca tathā pañcadaśāhakam 
vahec cet tridvyekavarṣair bhaven mṛtyur yathākramam ।। 

tathaiva ced vahed vāyur viṃśatiṃ pañcaviṃśatim 
ṣaḍbhir māsais tribhiś caiva yathāsaṃkhyaṃ bhaven mṛtiḥ ।। 

ṣaṭsaptāṣṭāviṃśatiṃ ced ūnatriṃśaddinaṃ vahet 
syād dvyekamāsaiḥ pakṣeṇa digdinaiś ca kramān mṛtiḥ ।। 

triṃśat tu caikatriṃśac ca dvātriṃśad divasān yadi 
pañcatridvidinair mṛtyur yathāsaṃkhyaṃ bhavet tadā ।। 

trayastriṃśaddinaṃ yāvat sūrye ced vāti mārutaḥ 
tasminn eva dine mṛtyur viṣṇor api na saṃśayaḥ ।। 

pañcāhapañcaviṃśatyor antarāleṣu vāsarāḥ 
noktāḥ ṣoḍaśasaṃkhyā ye teṣāṃ śodhanam ucyate ।। 

ṣaḍdinaṃ cet tribhir varṣaiś caturviṃśativarjitaiḥ 
saptāhaṃ cet tribhir varṣaiḥ pūrvoktadviguṇonitaiḥ ।। 

aṣṭāhaṃ cet tribhir varṣair dvāsaptatidinonitaiḥ 
navāhaṃ cet tribhir varṣaiḥ ṣaṇṇavatidinonitaiḥ ।। 

vahed ekādaśāhaṃ cen mārutaḥ sūryagocaraḥ 
caturviṃśatirātronān mṛtyur varṣadvayād bhavet ।। 

dvādaśāhaṃ varṣadvayāt pūrvoktadviguṇonitāt 
trayodaśāhaṃ dvivarṣād dvāsaptatidinonitāt ।। 

caturdaśāhaṃ dvivarṣāt ṣaṇṇavatidinonitāt 
ṣoḍaśadinavāhena dvādaśonaikavarṣataḥ ।। 

vahet saptadaśāhaṃ ced vāyuḥ sūryaikagocaraḥ 
caturviṃśatyahanyūnād varṣān mṛtyur udīritaḥ ।। 

aṣṭādaśāhaṃ ṣaṭtriṃśaddinanyūnaikavarṣataḥ 
ūnaviṃśati ced aṣṭacatvāriṃśaddinonitāt ।। 

ekaviṃśati ṣaḍmāsāt ṣaḍvāsaraviyojitāt 
dvāviṃśatiṃ cet ṣaḍmāsair dvādaśāhavivarjitaiḥ ।। 

trayoviṃśatyaharvāhe aṣṭādaśāhavivarjitāt 
caturviṃśatyaharvāhe catuḥṣaṭkadinonitāt ।। 

trayastriṃśaddineṣv evaṃ mṛtyulakṣaṇasaṃbhavaḥ 
vāyūdayavaśāj jñeyo nāta ūrdhvam asaṃbhavāt ।। 

āha cātra ।

ādau kṛtvā dinārdhaṃ sakaladinam athā-harniśaṃ yāvad eva tasmād ahnor dvayaṃ ca tridinam atha catur-vāsarān vyāpya yāvat 
prāṇo nāḍyāśrito yo vahati dinapater udgame savyahīne tasmin vijñeyam etad bhuvanaravidiśo maṅgalaṃ ṣaṭ catuṣkam ।। 

pañcabhyaḥ pañcaviṃśaddivasagatir ihā-rohate pañcavṛddhyā tasmād ekottareṇa triguṇitadaśakaṃ tryuttaraṃ yāvad eva 
kāle pauṣṇe samās tās trinayanaśaśinaḥ ṣaṭtriyugmendavo ye māsās te ahāni śeṣās tithidigiṣuguṇa-dvīndavo jīvitasya ।। 

tripuṭaṃ cakram ālikhya saptatriṃśadgṛhānvitam 
āyuṣaḥ prāṇavāyoś ca dināny aṅkakramāl likhet ।। 

ity āntarāṇy api mayā kathitāni mṛtyor liṅgāny ariṣṭam iti cāparanāmakāni 
vijñāya tāni sapadi tvaritair vidheyo dhyānādibhir maraṇa-vañcana eva yatnaḥ ।। 

।। iti śrīpaṇḍitavāgīśvarakīrtiviracite mṛtyuvañcano"padeśe dvitīyaḥ paricchedaḥ ।।

bāhyam ādhyātmikaṃ caiva jñātvāriṣṭakadambakam 
bāhyaś cādhyātmikaś cāpi jñeyo vañcanasaṃcayaḥ ।। 

kāyavākkarmabhir bāhyam āntaraṃ dhyānakarmabhiḥ 
maṇimantrauṣadhair bāhyam āntaraṃ yogaśaktibhiḥ ।। 

śraddhayā śakyate kartuṃ mṛtyuvañcanam udyataiḥ 
śraddhām ato dṛḍhīkuryād anyathā syāc chramaḥ paraḥ ।। 

kāyakleśair bahuvidhair na cāpy arthasya rāśibhiḥ 
dharmaḥ saṃprāpyate sūkṣmaḥ śraddhāhīnaiḥ surair api ।। 

śraddhā dharmaḥ paraṃ sūkṣmaḥ śraddhā jñānaṃ tapo hutam 
śraddhā svargaś ca mokṣaś ca śraddhā sarvam idaṃ jagat ।। 

sarvasvaṃ jīvitaṃ cāpi dadyād aśraddhayā yadi 
naivāpnuyāt phalaṃ kiṃ cic chraddadhānas tato bhavet ।। 

ādau dharmaparo bhūyān mṛtyuvañcanavāñchayā 
dharmo hi trāyate mṛtyor mṛtyur dharmeṇa vañcyate ।। 

na dharmarahitas tiṣṭhet pramādāt kṣaṇamātrakam 
brahmāpi hi vinā dharmaṃ na prabhur mṛtyuvañcane ।। 

viduṣāṃ saṃmato dharmaḥ satyaṃ dānaṃ tapaḥ kṣamā 
viśeṣeṇa kṛpā yeṣāṃ kiṃ teṣāṃ dharmavistaraiḥ ।। 

kāruṇyasyandi hṛdayaṃ yeṣāṃ sattveṣu saṃtatam 
teṣāṃ jñānaṃ ca mokṣaś ca kim anyair dharmavistaraiḥ ।। 

ahiṃsā paramo dharmaḥ sarvasiddhāntasaṃmataḥ 
tām evābhyasyato nityaṃ mṛtyur dūrāyate kramāt ।। 

alpāyur api dīrghāyuḥ sadyaḥ syāj jīvayann imān 
pakṣimatsyamṛgavyāḍacaurādīn vadhyadeśagān ।। 

dīrghāyur api mandāyuḥ sadyaḥ syān mārayann imān 
cauramatsyamṛgavyāḍapakṣiṇo jīvakāṅkṣiṇaḥ ।। 

daśākuśalasaṃtyāgāt triratnaśaraṇāt tathā 
pañcāṣṭaśikṣākaraṇān mṛtyur dūragato bhavet ।। 

bandhanasthān uruvyādhipīḍitān bhayakātarān 
trāyamāṇo bhaven nūnaṃ dīrghāyur mṛtyuvañcakaḥ ।। 

dīnānāthasalajjebhyaḥ śīlavadbhyo api cādarāt 
īpsitārthapradānena dhruvam āyur vivardhate ।। 

mātāpitṛgurujyeṣṭhajñānavṛddhādigauravāt 
mānasatkāradānāc ca mṛtyur naivopasarpati ।। 

sveṣṭadaivatatadbhaktaliṅgināṃ prātar ādarāt 
darśanād vandanān nityam āyuṣo vṛddhir āpyate ।। 

bhagnasphuṭitacaityādāv iṣṭakādipradānataḥ 
syād alpāyuḥ sudīrghāyuḥ paiṇḍapātikabhikṣuvat ।। 

vihārastūpabimbādeḥ kriyayārāmaropaṇāt 
mahāṣṭasthānapūjābhir āyur nityaṃ vivardhate ।। 

chandakabhikṣayopāttair dravyaiḥ saṃghasya bhojanāt 
saptāhāyuś cirāyuḥ syād dharmāśokanarendravat ।। 

vālukācaityakaraṇāt tathā sañcakatāḍanāt 
caityādivandanāc caiva dhruvam āyur vivardhate ।। 

mahāsamājakaṃ sūtram āṭānāṭīyakaṃ tathā 
sakṣudrakaṃ mahāmeghaṃ gāthāḥ susvastisaṃjñitāḥ ।। 

satkṛtya vācayed bhikṣūn svayaṃ vā saṃpravartayet 
dīrgham āyus tathā prāpya mṛtyor bhavati vañcakaḥ ।। 

gaṇḍavyūhādisūtrāntān mahāyānaprakāśakān 
ekāgro vācayen nityaṃ mṛtyuvañcanam aśnute ।। 

buddhabhāṣitaniḥśeṣayogatantrādivācanāt 
pañcarakṣāvidhānād vā mṛtyuvañcanam āpyate ।। 

uṣṇīṣagarbhacaityādipradakṣiṇavidhānataḥ 
dhāraṇījapataś cāpi vañcyate mṛtyur udyataiḥ ।। 

grahādimātṛkāpāṭhāt tadvidhānapuraḥsarāt 
sahasrāvartadhāraṇyā ratnolkāyāś ca jāpataḥ ।। 

dhāraṇīsaṃcayādyuktadhāraṇījāpatatparāḥ 
āsannam api dūrasthaṃ mṛtyuṃ kurvanti paṇḍitāḥ ।। 

maṇḍalālekhanād dhomān mudrāṇāṃ bandhanād api 
balīnāṃ gaṇacakrāṇāṃ pradānān mṛtyuvañcanam ।। 

cāturdiśāryasaṃghāya bhojyaśayyādidānataḥ 
kāṣāyamātrakaṇṭhebhyo api vā syān mṛtyuvañcanam ।। 

evamādikriyārambhād dharmaḥ syāt puṇyalakṣaṇaḥ 
puṇyāt syād āyuṣo vṛddhis tadvṛddher mṛtyuvañcanam ।। 

ityādi leśato dharmaḥ sūcitas tasya vistaraḥ 
svayam unnīya kartavyas taj jñātvā mṛtyuvañcakaiḥ ।। 

evaṃ dharmaparo bhūtvā siddhamantrauṣadhādibhiḥ 
mṛtyor viṣkambhaṇaṃ kuryāt sadyaḥpratyayakāribhiḥ ।। 

maṇimantrauṣadhīsiddharasāyanabalāt svayam 
mṛtyoḥ parājayo dṛṣṭaḥ siddhavidyādharair yathā ।। 

ratnalakṣaṇaśāstroktaparīkṣottīrṇavajradhṛk 
āyurmaṇidharo vāpi viśvarūpadharas tathā ।। 

anarghasyendranīlāder anyasyāpi mahāmaṇeḥ 
dhārakaḥ sarvamṛtyūnāṃ vañcakaḥ syān na saṃśayaḥ ।। 

na cātra maṇisāmarthye saṃśayo yuktibhāg yataḥ 
acintyaśaktiyuktatvaṃ maṇīnāṃ sarvasaṃmatam ।। 

paṭāder agrataḥ pūjāṃ kṛtvā vijanadeśagaḥ 
jāpamātreṇa mantrāṇāṃ jāyate mṛtyuvañcanam ।। 

ākāram uccaret pūrvaṃ tato ro lig iti sphuṭam 
trailokyavijayākhyo ayaṃ vidyādharapiṭoditaḥ ।। 

mantraṃ lokeśvarasyāsya japed akṣaralakṣakam 
śrāddhas tadagrato avaśyaṃ mṛtyuvañcanam aśnute ।। 

uktvā vairocanaṃ pūrvaṃ paścāt tāre dvir uccaret 
tuttāre dvis ture dviś ca svāhāntaṃ mṛtyuvañcanam ।। 

tārābhyudayatantroktaṃ mantram akṣaralakṣakam 
japtvāryatārāpurato dadhimadhvaktipūrvakam ।। 

pūrvottaraśikhādūrvāpravālāyutahomataḥ 
pūrvakarmaprabhāvottham api mṛtyuṃ nivārayet ।। 

oṃ ādau ruru tato ataḥ sphurupadam ataḥ param 
jvala tiṣṭha tathā siddhalocaneti padatrayam ।। 

sarvārthasādhani svāhā mantro aśokadale amale 
pradattadakṣiṇācāryair likhitaś candanadravaiḥ ।। 

lakṣaśaḥ koṭiśo vāpi śuddhaḥ śraddhāvidhānataḥ 
dhautaś ca nirjalaiḥ kṣīrair huto dūrvādināthavā ।। 

bhavaty avaśyaṃ niḥśeṣamṛtyuvañcanakārakaḥ 
niḥśeṣarogasaṃdohaśāntikṛc cāpi jāyate ।। 

mahāmaṇḍalakalpoktāryaśatākṣarasaṃjñinaḥ 
acintyaśakter mantrasya dṛṣṭaśakter anekaśaḥ ।। 

bhadracaryāsamādānavidhinā lakṣajāpataḥ 
mokṣo api jāyate nūnaṃ kiṃ punar mṛtyuvañcanam ।। 

namas traiyadhvikānāṃ prāk tathāgatānāṃ syāt tataḥ 
sarvatrāpratihatāvāptidharmateti padaṃ tataḥ ।। 

balinām iti padād oṃ tato asamasamapadam 
samantato anantatāvāptiśāsanīti padaṃ tataḥ ।। 

hara hara smara smaraṇa vigatarāgapadāt 
buddhadharmate tataḥ syāt sara sara samabalā ।। 

hasadvayaṃ trayadvayaṃ syād gaganamahāvara 
lakṣaṇe jvala jvalanasāgare syāt padaṃ tataḥ ।। 

dattvā svāhāpadaṃ cānte bhaven mantraḥ śatākṣaraḥ 
tathāgatānāṃ sarveṣāṃ hṛdayaṃ parikīrtitaḥ ।। 

āryamahāpratisarāmantraikalakṣajāpataḥ 
dūrvādaśāṃśahomena dhruvaṃ syān mṛtyuvañcanam ।। 

oṃ uktvā padaṃ vimale jaya vare anu cāmṛte 
hūṃ hūṃ phaṭ phaṭ tataḥ svāhāpadaṃ brūyād anantaram ।। 

vairocanād bharadvayaṃ saṃbharadvayam uccaret 
uktvendriyabalaśabdaṃ viśodhanipadaṃ vadet ।। 

hūṃ hūṃ ruru cale svāhā mantro ayaṃ mṛtyuvañcakaḥ 
āryamahāpratisarāhṛdayaṃ sarvasiddhidam ।। 

japtvā daśākṣaraṃ mantraṃ taṃ mṛtyuñjayasaṃjñinam 
daśalakṣaṃ tato homaṃ kurvīta kusumaiḥ sitaiḥ ।। 

ghṛtāktair lakṣasaṃkhyātair lokeśvaraguroḥ puraḥ 
mantraśaktes tato avaśyaṃ jāyate mṛtyuvañcanam ।। 

oṃ ādau tata āṅkāraḥ syād īṅkāras tataḥ param 
ūṃ oṃ mṛtyuñjaya oṃ syād ayaṃ mantro daśākṣaraḥ ।। 

kalpādijīvanārthaṃ ye proktā mantrās tathāgataiḥ 
tair āyuḥ sādhitaṃ dīrghaṃ tad dīrghaṃ mṛtyuvañcanam ।। 

mantrair yaiḥ śāntikarma taiḥ syād vṛddhir āyuṣaḥ 
yaiḥ syād āyuṣo vṛddhis taiḥ syān mṛtyuvañcanam ।। 

mantrāṇām api sāmarthye saṃśayo naiva yuktibhāk 
acintyaṃ mantrasāmarthyaṃ yataḥ sarveṣu saṃmatam ।। 

āha ca ।

na tad vastvantaraṃ loke yan na mantraiḥ prasidhyati 
cintāmaṇisamā mantrāḥ samyakśraddhābhiyogataḥ ।। 

kalpoktavidhibhir yad vā yathākāmaprayogataḥ 
bubhukṣābahule deśe amṛtām atyantam ācaret ।। 

ekākinīm athānyair vā bheṣajaiḥ sahitāṃ sadā 
tataḥ sthiravapur bhūtvā mṛtyudvāraṃ na paśyati ।। 

anyeṣām api divyānāṃ bheṣajānāṃ niṣevaṇāt 
āyurvṛddhir jarārogakṣayān mṛtyuṃ na paśyati ।। 

vyomahemādibhir baddhān mṛtakād vidhibhakṣitāt 
syān mṛtyuvañcanaṃ siddharasāyanasvabhāvakāt ।। 

pratipaccandrayogādiprāptenāntaradhātunā 
nasyapradānād vidhinā jāyate mṛtyuvañcanam ।। 

saṃpuṭādikatantroktapañcāmṛtaprayuktibhiḥ 
yathopadeśayuktābhir jāyate mṛtyuvañcanam ।। 

ūrdhvādhobhyāṃ saṃpuṭayogaṃ yaḥ satataṃ kurute hitayogam 
tasya suruṣṭo apy avihatatejāḥ kiñcitkārī no yamarājā ।। 

mṛtasaṃjīvanam api kva cid dṛṣṭam upāyataḥ 
ṛtumatkanyānarayoḥ śṛṅkhalāyāḥ samutthitaiḥ ।। 

śoṇitonmiśraśukrākhyadhātubhir nātiśītalaiḥ 
sadyo yad vā mṛtasyaiva patitair dhātubindubhiḥ ।। 

ghṛtāktanalikārandhranirgamena praveśitaiḥ 
vahed ātmīyanāsāgrapuṭonnītapuṭe kramāt ।। 

punarujjīvanaṃ dṛṣṭaṃ nirodhaś cātra śasyate 
pratyakṣadṛṣṭasāmarthyo yogo ayaṃ bahusaṃmataḥ ।। 

samastam athavā vyastaṃ triphalāmārkavānvitam 
ācaret satataṃ yogī pañcāmṛtarasāyanam ।। 

valīpalitanirmukto vajrakāyaḥ svarūpavān 
dhruvaṃ nivārayen mṛtyum api kalpādisaṃkhyayā ।। 

āgamoktam idaṃ siddhaṃ bahūnāṃ vārigandhayoḥ 
bhakṣaṇālepanāt sadyo bhaven mṛtyunivāraṇam ।। 

anayoś ca sadābhyāsān madhusarpiḥsamanvitāt 
kālākālasamudbhūtam api mṛtyuṃ nivārayet ।। 

bheṣajasyāpi sāmarthye saṃśayo naiva yujyate 
acintyauṣadhisāmarthyaṃ yataḥ sarvatra dṛśyate ।। 

iti vidhāya bahiṣkriyam ādarān maraṇavañcanam atra yathābalam 
prabalamṛtyumahābhayaśaṅkayā kuruta saṃprati tat punar āntaram ।। 

।। iti śrīpaṇḍitācāryavāgīśvarakīrtiviracite mṛtyuvañcanopadeśe tṛtīyaḥ paricchedaḥ ।।

ādhyātmikam idānīṃ tu mṛtyuvañcanakauśalam 
kathyate yena niyataṃ vañcyate mṛtyur udyataiḥ ।। 

rājacauramahāmāravyāghrādibhayavarjite 
saṃsargarahite ramye parvatādau kva cit sthitaḥ ।। 

mātrābhojī mitālāpī nāticaṅkramaṇe priyaḥ 
alpanidro bahudhyāno mantrajāpaparāyaṇaḥ ।। 

sveṣṭadaivatapūjādisaddharmapaṭhanakriyaḥ 
trisaṃdhyabalidānādimaitryacittāditatparaḥ ।। 

puṇyād eveṣṭasaṃsiddhir iti puṇyakriyāparaḥ 
śraddhayā sarvasaṃsiddhir iti śraddhāvivardhakaḥ ।। 

vīryāt prārabdhaniṣpattir iti vīryapuraḥsaraḥ 
dhyānād iṣṭaphalasthairyam iti dhyānaikacetanaḥ ।। 

saṅgatyāgād anuṣṭhānam iti saṅgavivarjitaḥ 
samyagjñānavataḥ sarvam iti jñānasamanvitaḥ ।। 

maraṇe duḥkhasaṃdarśī dharmadarśī ca jīvite 
vigatāśeṣasaṃdeho mṛtyuvañcanam ārabhet ।। 

tatrādau dhyānasahitaṃ vighnārer mantralakṣakam 
saṃjapyāyutahomena sarvavighnān nivārayet ।। 

namaḥ samantakāyeti prāg vākcittapadaṃ tataḥ 
vajrāṇāṃ ca namo vajrakrodhāyeti padaṃ vadet ।। 

mahādaṃṣṭrotkaṭapadād bhairavāyapadaṃ vadet 
asimusalaparaśupāśagṛhītahastāya ।। 

oṃ tato amṛtakuṇḍali khakhakhāhikhāhipadāt 
tiṣṭhatiṣṭhapadaṃ tasmād bandhadvayaṃ hanadvayam ।। 

tataḥ ।

dahadvayaṃ garjadvayaṃ visphoṭayadvayaṃ tataḥ 
sarvavajrapadād vighnavināyakānpadaṃ vadet ।। 

mahāgaṇapatipadāj jīvitāntakarāya ca 
hūṃ hūṃ phaṭ phaṭ svāhā mantro vighnanivāraṇaḥ ।। 

hṛdayenāthavāsyaiva sarvavighnān nivārayet 
pūrvavaj japahomābhyāṃ śaktir asyāpi tādṛśī ।। 

oṃ prāg uktvā huluhulutiṣṭhatiṣṭhapadaṃ tataḥ 
hanadvayaṃ dahadvayaṃ garjagarja tathā punaḥ ।। 

visphoṭayadvayād brūyāt sarvavighnavināyakān 
oṃ amṛta hūṃ hūṃ phaṭ phaṭ mama rakṣāṃ kuru svāhā ।। 

evaṃ vināśitāśeṣamāravighno nirākulaḥ 
yathābhivāñchitaṃ yogaṃ dhyāyād ekam aharniśam ।। 

buddhakṛtyakaraṃ śāntaṃ ghanavyūhanivāsinam 
ābaddhavajraparyaṅkaṃ padmacandrāsanopari ।। 

bodhyagrīmudrayā yuktaṃ sphuracchvetāṃśumaṇḍalam 
dhyātvā vairocanaṃ śuklam āsaṃsāraṃ na mṛtyubhāk ।। 

svavarṇamudrāveṣāḍhyān akṣobhyāditathāgatān 
dṛḍhāhaṅkārato dhyātvā vajrakāyo bhaved dhruvam ।। 

hṛccandrasaṃpuṭāntasthaṃ lokeśvaram anaśvaram 
śāntaṃ śaśāṅkasaṃkāśaṃ sarvālaṅkārabhūṣitam ।। 

raktavarṇāmitābhākhyasaṃbuddhāṅkajaṭādharam 
vajraparyaṅkasaṃśliṣṭaṃ pādāmbūjadvayopari ।। 

samādhimudrāsaṃsaktakaradvayam anākulam 
sarvasattvārthasaṃpādikāruṇyasyandivigraham ।। 

dhyātvā tasyāpi hṛdaye lokeśam aparaṃ tathā 
punas tasyāpi hṛdaye tathaivaikāgramānasaḥ ।। 

koṭisaṃkhyā bhaved yāvat tāvad dhyānam idaṃ smaret 
tataḥ sūkṣmātisūkṣmāntyalokeśagatacetasaṃ ।। 

dṛṣṭvā yogeśvaraṃ mṛtyur mriyate svayam eva hi 
anye api skandhajā doṣāḥ svayam āyanti nāśitām ।। 

sitāravindamadhyasthacandrabimbāsanopari 
ābaddhavajraparyaṅkāṃ varadotpaladhāriṇīm ।। 

śaraccandrakarākārāṃ pṛṣṭhacandrasamāśritām 
sarvālaṅkārasaṃpūrṇāṃ ṣoḍaśābdavapuṣmatīm ।। 

sarvasaṃbuddhatatputramātaraṃ kāmarūpadhām 
dhyātvāryatārāṃ hṛdaye tasyāś cakraṃ sitadyuti ।। 

aṣṭakoṣṭhakam aṣṭābhir akṣaraiḥ paripūritam 
oṃhāvyañjanamadhyasthasādhyanāmādyanābhikam ।। 

dhyāyād ekāgracittaḥ san ṣaḍ māsān dṛḍhaniścayaḥ 
japec cākhinnacittaḥ san mantram enaṃ daśākṣaram ।। 

oṅkāram ādito dattvā paścāt tāre prayojayet 
tuttāre syāt ture paścāt svāhāntaḥ sārvakarmikaḥ ।। 

brahmendraviṣṇucandrārkarudradikpālamanmathaiḥ 
apy akhaṇḍitaromāgraṃ mṛtyuṃ jayati mṛtyuvat ।। 

valīpalitadaurbhāgyavyādhidāridryasaṃkṣayaḥ 
siṃhādyaṣṭamahābhītiduḥkhasaṃdohanāśanam ।। 

ayācitānnapānādiharmyavastrādisaṃgamaḥ 
khaḍgāñjanapādalepabhadrakumbhādisiddhayaḥ ।। 

kavitā vaktṛtā medhā prajñā caikāntanirmalā 
anyā ca vāñchitā siddhiś cakrād asmāt prajāyate ।। 

locanāṃ māmakīṃ cundāṃ mahāśvetādidevatām 
svavarṇaveṣamudrābhir dhyātvā mṛtyuṃ na paśyati ।। 

sveṣṭadaivatayogād vai dṛḍhāhaṅkārasaṃgatāt 
cintāmaṇer ivāśeṣavāñchitārthaprasādhakāt ।। 

śaṅkhakundendusaṃkāśāt saṃhārakramasaṃsthitāt 
anāratakriyābhyāsān mṛtyur āyāti vañcanām ।। 

cakrasaṃvaratantroktaherukākārabhāvanām 
kṛtvā tatpariṇāmena kapālākārabhāvakam ।। 

kaṅkālarūpanidhyānaṃ yad vā khaṭvāṅgabhāvakam 
mārayitum ayogyatvān mṛtyur naivopasarpati ।। 

mūrdhni candramaso bimbāt kṣaratpīyūṣaśīkarān 
hlādayataḥ samastāṅgaṃ romakūpaiḥ samantataḥ ।। 

ṣaḍ māsān bhāvayed yogī sarvasaṅgavivarjitaḥ 
sarvarogān vinirjitya mṛtyuṃ jayati mṛtyuvat ।। 

śirasy adhomukhaṃ śuklaṃ sahasradalapaṅkajam 
dhyātvā candrakramān mṛtyuṃ kalpādīṃś ca vināśayet ।। 

asyaiva kṛṣṇarūpasya kṛṣṇarūpair gabhastibhiḥ 
saṃplāvayato nidhyānāt palitādivināśanam ।। 

jihvāṃ tālugatāṃ kṛtvā dantair dantān asaṃspṛśan 
acintyahetusāmarthyāc chraddheyam idam ārabhet ।। 

kaṇṭharandhrordhvasaṃlagnāṃ lālājihvām adhomukhīm 
jihvāgreṇālihed yāvat pīyūṣarasavedanam ।। 

tato mṛtyuṃ jayet sadyaḥ palitādīṃś ca nāśayet 
sarvāpadbhayarakṣā ca tatkāryakaraṇād bhavet ।। 

ānāpānasmṛter vātha prakārāntaram ārabhet 
saṃkhyāvṛddhyā tato avaśyam ajarāmaratāṃ vrajet ।। 

adhikāni śataiḥ ṣaḍbhiḥ sahasrāṇy ekaviṃśatiḥ 
ahorātreṇa sattvānāṃ prāṇasaṃkhyā prakīrtitā ।। 

praveśaḥ śasyate vāyor yataḥ so amṛtasaṃcayaḥ 
amṛtaṃ jīvanopāyas tena mṛtyur na bādhate ।। 

praveśaṃ gaṇayed vāyoḥ śanair lakṣādisaṃkhyayā 
ekāgracitto vidhivad aśabdo japa ucyate ।। 

aśabdajapalakṣeṇa vidhinā pūritena tu 
bhraṣṭāyuḥ san naro avaśyaṃ pañca varṣāṇi jīvati ।। 

dvilakṣādijapād evam unneyā vṛddhir āyuṣaḥ 
upāyakuśalo bhūtvā vāyujāpaparo bhavet ।। 

sahasram ekaṃ satataṃ gaṇayet prātar utthitaḥ 
sarvarogavinirmukto jīvec candrārkajīvitam ।। 

nābhyāsarahitas tiṣṭhen muhūrtam api na svapet 
jitvā nidrāṃ prayatnena bhakṣapānāsanādikam ।। 

vāhayed anulomena vāyusaṃcāratatparaḥ 
anulomaparo yogī mṛtyuṃ jayati niścitam ।। 

mriyante tu vilomena mūḍhā bhogaikatatparāḥ 
svabhāvas tv anulomaḥ syād vilomas tadviparyayāt ।। 

siṃhavikrīḍitāmudrābandhanād vāyurodhataḥ 
jitakumbhakayogād vā nirodho mṛtyuvañcakaḥ ।। 

āpādatalaparyantaṃ śanair āpūrya vāyunā 
nirodhanāt kumbhakaḥ syāt taj japakrama ucyate ।। 

trividhodghātayogena jito bhavati kumbhakaḥ 
hīnamadhyādibhedena udghātas trividho mataḥ ।। 

udghāta iti saṃjñaiṣā vakṣyamāṇārthadarśikā 
ṣaṭtriṃśanmātriko hīno madhyaḥ syād dviguṇas tathā ।। 

jyeṣṭhas tu triguṇo jñeyaḥ kumbhakas tena jīyate 
prathamaṃ kumbhakaṃ kṛtvā paścāt svaṃ jānumaṇḍalam ।। 

pāṇinā triḥ parāmṛśya ṣaḍ dadyāc choṭikās tataḥ 
eṣā mātrocyate kālaparicchedāya paṇḍitaiḥ ।। 

ṣaṭtriṃśan mātrā yāvat syus tāvad yā kumbhakakriyā

triguṇo jyeṣṭha udghāto aṣṭottaraśatamātrikaḥ

nāsāgranirgato vāyur ādāyāsecayen muhuḥ

ākāśadohanadhyānam idaṃ nityaṃ samācaret 
ākāśam iva taṃ mṛtyuḥ paśyann api na paśyati ।। 

saṃtyajya vāyum athavā tattvam eva sadācaret 
tattvena jīyate mṛtyus tathānye māraśatravaḥ ।। 

niḥśeṣadoṣapraśamo niḥśeṣaguṇasaṃbhavaḥ 
tattvena jīyate puṃsāṃ siddhayaś cābhivāñchitāḥ ।। 

tattvaṃ ca śūnyatā proktā sarvākāravarātmikā 
tām evābhyasyato nityaṃ mṛtyur āyāti vañcanām ।। 

dharmasaṃbhogakāyādidānaśīlādyabhedinī 
sarvālambanasaṃyogāt sarvālambanavarjitā 
sarvākāravaropetā śūnyatā kathitā jinaiḥ ।। 

na cintāṃ cintayec cintyām acintyāṃ naiva cintayet 
cintyācintyāṃ na cintayet tataḥ prāpsyati śūnyatām ।। 

na bhāvaṃ bhāvayed vidvān nāpy abhāvaṃ vibhāvayet 
bhāvābhāvavinirmuktaṃ bhāvayet tadvivarjitaḥ ।। 

sarvākāravaropetāṃ śūnyatāgrahavarjitām 
śūnyatāṃ bhāvayen nityaṃ mṛtyunā naiva dṛśyate ।। 

vikalpamātrasaṃbhūtaṃ mṛtyuṃ so api na paśyati 
anyāṃś ca bahulān mārān saṃsāraṃ ca na paśyati ।। 

tad āha ।

mṛtyur nāma vikalpo ayam abhāve sarvavastunaḥ 
hanyate svavikalpena pṛthagjanavijṛmbhitair iti ।। 

vikalpaḥ sarvaduḥkhāni vikalpo mṛtyur ucyate 
vikalpaṃ hānayet tasmān nirvikalpamanā bhavet ।। 

yad yad vibhāvyate abhīkṣṇam ādarādiviśeṣitam 
tat tat tu sphuṭatāṃ yāti kāmaśokādibuddhivat ।। 

sulabhāni hi śāstrāṇi prayogās te sudurlabhāḥ 
prayogajñeṣu lokeṣu prayogā naiva durlabhāḥ ।। 

prayogakuśalaḥ śrāddhaḥ kṣamāvīryapratiṣṭhitaḥ 
anuṣṭhānaparaḥ prājño mṛtyuvañcanabhājanam ।। 

śramāt pācanapānāder jaṭharānaladīpanam 
pradīptavahner āhāraḥ pariṇāmo api śaktitaḥ ।। 

pariṇāmāt tanoḥ puṣṭiḥ puṣṭer balavivardhanam 
balāt syād āyuṣo vṛddhis tadvṛddheś cirajīvitam ।। 

pratyakṣasiddhaṃ sarveṣāṃ yathā nāsty atra saṃśayaḥ 
tathopadeśāt kalpādijīvane api na saṃśayaḥ ।। 

nāśonmukho api dīpādis tailavartikṣayādibhiḥ 
punas tailādisaṃprāpteḥ kalpāntam api tiṣṭhati ।। 

yathā tathāyaṃ deho api mriyamāṇo jarādibhiḥ 
siddhopadeśasaṃprāpteḥ kalpāntaṃ kiṃ na tiṣṭhati ।। 

yathā jīrṇasya gehasya saṃskāreṇa cirasthitiḥ 
tathā jīrṇasya dehasya saṃskāreṇa na kiṃ bhavet ।। 

na duṣkaraṃ kiṃ cana vikrameṇa tasmād anikṣiptadhurā yatadhvam 
nityapravṛttā vibhinatti kāle śilātalaṃ mṛdv api vāridhārā ।। 

kāṣṭhād agnir jāyate mathyamānād bhūmis toyaṃ khanyamānā dadāti 
sotsāhānāṃ nāsty asādhyaṃ narāṇāṃ nyāyārabdhāḥ sarvayatnāḥ phalanti ।। 

subahūny api vijñāya vañcanāni mṛter iha 
ananuṣṭhānād viṣṭhābhir liptvā śayyāṃ mṛtā vayam ।। 

prayogakuśalā ye ca paṇḍitāḥ śāstrapāragāḥ 
ananuṣṭhānadoṣeṇa dantān niṣkāśya te mṛtāḥ ।। 

jñātvāpi yogaśāstrāṇi prayogāṃś cāpy anekaśaḥ 
adya śvo vā kariṣyāma iti cintāparā mṛtāḥ ।। 

idaṃ kartāsmi śvaḥ punar idam idaṃ cāsya parato vṛthā prāptāśvāso vimṛśati na lolāṃ bhavagatim 
agatvā kāryāntaṃ viṣaya-salilāvartapatitā prajā mṛtyor vaktraṃ makara-mukhavan nauḥ praviśati ।। 

iti nigaditam etad vañcanaṃ mṛtyuśatroḥ svaparasamayasiddhadhyānakarmaprasādhyam 
aghṛṇamaraṇabhīter yatnataḥ saṃvidheyaṃ yata iha kamanīyaṃ jīvitān nānyad asti ।। 

vajraḍākaṃ catuṣpīṭhaṃ buddhapañjarakādi ca 
kālottaraṃ kālāvalīṃ kālapañcāśikām api ।। 

jīvapañcāśikādīni śāstrāṇy ālokya yatnataḥ 
sattvānukampayā proktam idaṃ maraṇavañcanam ।। 

śāradacandranabhaḥ pariśuddhaṃ śuddhaviyacchaśināpy asamāptam 
āptam ataḥ kuśalaṃ yad amṛtyus tena jano astu sa mṛtyuvimuktaḥ ।। 

gaṇanāśramahānāya lekhakānāṃ tapasvinām 
granthapramāṇam ākhyātaṃ sāśītikaśatatrayam ।। 

।। iti śrīpaṇḍitācāryavāgīśvarakīrtiviracite mṛtyuvañcanopadeśe caturthaḥ paricchedaḥ samāptaḥ ।।
