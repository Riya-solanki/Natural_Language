Varāhamihira Yogayātrā

yaś cakṣur jagataḥ sahasrakarabad dhāmnāṃ ca dhāmā arkavan
mokṣadvāram apāvṛttaṃ ca ravivad dhvāntāntakṛt sūryavat।
ātmā sarvaśarīriṇāṃ savitṛvat timāṃśuvat kālakṛt
sādhvīṃ naḥ sa giraṃ karotu savitā yo aanyair atulyopamaḥ।। 

vakṣyāmi bhūpam adhikṛtya guṇopapannaṃ
vijñātajanmasamayaṃ pravibhktabhāgyam।
ajñātasūtim athavā aviditaiṣya bhāgyaṃ
sāmudrayātrikanimittaśataiḥ pṛthūktaiḥ।। 

karma anyajanmajanitaṃ sad asac ca daivaṃ
tat kevalaṃ bhavati janmani satkulādye।
bālyāt paraṃ vinayasauṣṭhavapātratā ca
puṃdaivajā kṛṣivad ity upapādyam etat।। 

sāmantakālayasudhodyamamantrabhṛtyair
daivena ceti kṛṣivan nṛpateḥ phalāptiḥ।
syāc chidram ekam api cet tata eva sarvaṃ
nāśaṃ prayāti hi dṛteḥ sravaṇād iva ambhaḥ।। 

graharkṣatithyudgamarāśihorā
dreṣkāṇabhāgādyanukūlamātram।
bhaved yiyāsor yadi siddhihetuḥ
svayaṃ bhaved daivavid eva rājā।। 

mantrābhiṣekamaṇibandhanaśāntikarma
homopavāsasurayāgajapādimātram।
syāt siddhihetur atha ced vijagīāato arīṇ
kasmāt tadā narapatir na bhavet purodhāḥ।। 

budhārthaśāstrāṇy api mantriṇo api
kuyuḥ praṇāmaṃ na nareśvarāṇām।
yady ābhijātyadvipavājipatti
kośādyapekṣā na bhavet pradhānā।। 

svadeśe santuṣṭaḥ karituragakośe sati na yo
bhaved yāyī saindhaiḥ kṛtaparibhavo yāti vilayam।
svacakreṇaiva asau kṣapitadhanarāṣṭro bhavati vā।
gato yo nādatte sunayacarito nītividhūrān।। 

ulūkasya dhvāṃkṣo niśi valibhujāṃ so ahni vaśagaḥ
sthale nakraṃ siṃho mṛgapam udke hanti jalajaḥ।
abudhyā yaḥ kālaṃ vrajati nṛpatir deśam athavā
hate darpe vākyaṃ smarati viduṣāṃ so arivaśagaḥ।। 

ripoḥ kṛtvā avajñāṃ jitam iti viditvā madabalān
na yatno hātavyaḥ śikhiviṣasamo alpo api hi ripuḥ।
pramatto adreḥ kuñjād ayutabalabhāgair api narair
hato baddhastambhe kakhati madalekhāṃ dvipapatiḥ।। 

vapraprākārayantrapraharaṇaparikhātoyameyendhanāḍhyaṃ[kdhānyendhanāḍhyaṃ
durgaṃ kṛtvā ātmagupataṃ dvipaturagabhiṣakśilpiviprābhyupetam।
lubdhatrastābhimāniprakupitakubhṛtāṃ jñātaśīlair vihīnaṃ
kṛtvā śūrāptasaṃsthaṃ paraviṣayam iyāc chuddhapārṣṇir narendraḥ।। 

budhvā śaktiṃ svaparabalayoḥ sāmam ekapradānaiḥ
kṛtvopāyair vyasanasamaropāyasaṃrakṣaṇaiś ca।
bhaktyā sādhūn dhanavirahitān arthalubdhāṃś ca dānair
neyād bhedais tribhir api na ye sādhayet tāṃś ca daṇḍaiḥ।। 

sāmno jīvaḥ sabhṛgutanayo daṇḍanāthau kujārkau
dānasyenduḥ śikhiyamabudhāḥ sāsurā bhedanāthāḥ।
vīryopetair upacayakarair lagnagair vīryagair vā
tat tat siddhiṃ vrajati tad ahaḥsv āṃśakair vāpi teṣāṃ।। 

ṣāḍguṇye sandhir ādau paripaṇaracito vigraho asyāpakārī
naitau ced āsanaṃ tat parapuragamanaṃ sarvasaineyna yānam।
sainyārddhenābhiyukte parapuram api yat tad dvidhāyānam uktaṃ
saṃśleṣo anyatra yaḥ syāt sa khalu nigaditaḥ saṃśrayo anyo guṇānam।। 

madhyāhne arkas tuhinakiraṇo nityam ākrandasaṃjñaḥ
pauraḥ pūrve bhavati dinakṛd yāyisaṃjño antyasaṃsthāḥ।
jīvaḥ sauris tuhinakiraṇāsyātmajaś ceti paurāḥ
ketur yāyī sabhṛgujakujaḥ siddhikānandanaś ca।। 

yānaṃ yāyibhir āsanaṃ śubhakarair vīryānvitair nāgarair
dvaidhībhāvam iyād yadā śubhakarāḥ paurāḥ sayāyigrahāḥ।
saumyaiḥ sandhir asadgrahaiś ca balibhir yuddhe anukūlair jayaḥ
sarvair apy aśubhapradair narapatir daivāmbitaṃ saṃśrayet।। 

kośo hi rājyatarumūlam ato asya śākhāḥ
pakṣī vipakṣa iva kiṃ nidhanaḥ karoti।
annād ivendriyagaṇo vasutas tathā annaṃ
tatprāptirakṣaṇavivṛddhiṣu yatnavān syāt।। 

nīcācirāḍhyanṛpavallabhataskarebhyo
rāṣṭraṃ nṛpeṇa parirakṣyam ato asya kośaḥ।
kāle karapraṇayanaṃ ca yathocitānāṃ
tan nārthayed bhavati yena janāpavādaḥ।। 

yātrā nṛpasya śaradīṣṭaphalā madhau ca
chidre ripor na niyamo asti ca kecid āhuḥ।
chidre apy arer bhavati daivayutasya siddhiḥ
sāmānyam āmiṣam idaṃ pratibhūmipānām।। 

śatror vadhāya sacivaṃ śubhadaivayuktam
ājñāpayen nṛpatir ātmani daivahīne।
jahyān na cārtham ati daivaparo api bhūtvā
dṛṣṭe dvipe dvipapadānusṛtiḥ kim artham।। 

kālo abhyupaiti sakṛd eva naraṃ kathaṃcit
prāpnoti tan na samayaṃ khalu kālakāṃksī।
kālena gocaragatān anapekṣya bhakṣyān
mandakramo apy ajagaraḥ samupaiti siddhim।। 

durgasthitaṃ phalam apakvam analpayatnāt
saṃsiddhim apy upakaroti guṇaṃ na puṃsām।
sādhāraṇaṃ svapatitaṃ ca bhaved yato ataḥ
kāle abhiyuktapatitaṃ rasavat sukhāya।।e 

raktāsitādyā hi yathāmbarasya varṇāḥ sitasyaiṣa bhavanti samyak।
vilagnatithyādiguṇās tathaiva viśuddhadoṣasya bhavanti yātuḥ।। 

madyāṅganāvāditanṛtyagītāny akṣā vṛthādyā mṛgayā dyunidrā।
parokṣanindeti ca kāmajāni daśaiva vindyād vyasanāni puṃsām।। 

vākpāruṣyaṃ daṇḍapāruṣyam īrṣyāṃ droho asūyā paiśunaṃ sāhasañ ca।
arthasyoktaṃ dūṣaṇaṃ ca aṣṭasaṃkhyaḥ krodhodbhūto varga eṣa pradiṣṭaḥ। 

mahad idam anayānām āspadaṃ madyam āhus tanudhanamatisattvaśreyasāṃ karṣaṇāya।
samupahatamatiḥ san tatpradhānendriyatvād gaṇayati na hi bhakṣyābhakṣyam anyāni caivam।। 

patnīyaty api mātaraṃ madavaśāt patnīṃ ca mātrīyati
śvabhrīyaty api mandiraṃ ślathaśilaṃ kūpaṃ ca gehīyati।
svalpaṃ vāry udadhīyatīśvaram apāṃ mohāt sthalīyaty api
mitrīyaty api pārthivaṃ kim aparaṃ kuryān na yan madyapaḥ।। 

bhūyo api dīvyati jito jayalipsayaiva prāpnoti tac ca saviśeṣataraṃ kadācit।
kṛtvā apriyāṇi madataḥ parito apy aṭanti bhūyaḥ pivet kim api doṣaviśeṣakāṃkṣī।। 

abhyāgamotsavabhiṣagvacanopadeśaiḥ kāmaṃ pibed amatilopakṛd aprakāśam।
dyūtaṃ vinendriyasukhāni hitāni yuktyā vijñasya jīvitaphalaṃ viṣayair vinā kim।। 

śaucācāravivarjitaḥ śava iva tyaktaḥ suhṛdbāndhavair
niśśaṅko nirapatrapo gataghṛṇaḥ kṛcchrād avāptāśanaḥ।
bandhur nāsya nibandhanaṃ na śapatho lokadvayaṃ nekṣyate
māyāvī kulapāṃśalaś ca kitavo doṣākaro niḥsukhaḥ।। 

daivajñamantrisuhṛdāptavacāṃsi rājā yo na ādriyen nijaviceṣṭitaduṣṭabuddhiḥ।
so agremareṇa rahito andha iva acireṇa hāsyatvam eti patito viṣaye ripūṇām।। 

medhāvī matimān adīnavacano dakṣaḥ kṣamāvān ṛjur
dharmātmāpy anasūyako laghukaraḥ ṣāḍguṇyavic chaktimān।
utsāhī pararandhravit kṛtadhṛtir vṛddhikṣayasthānavit
śūro na vyasanī smaraty upakṛtiṃ vṛddhopasevī ca yaḥ।। 

parīkṣyakārī na vikasthanaś ca dṛḍhapatijño atidṛḍhapahārī।
jitendriyaḥ syāj jitakopalobhanidrālasasthānaparigrahaś ca।। 

tyāgī vinītaḥ priyadarśanaś ca vyapetamohaḥ pratipattiyuktaḥ।
deśasya kālasya ca bhāgavijñaḥ svayaṃ ca yaḥ syād vyavahāradarśī।। 

śabdārthavin nyāyāpaṭuḥ pragalbhaḥ saṃgrāmavidyākuśalo abhijātaḥ।
smitābhibhāṣī mitasatyavaktā daivānvito yasya sa dhāma lakṣmyāḥ।। 

guṇaiḥ samastair api saṃprayuktā kanyeva yātrā viguṇāya dattā।
karoty akīrtiṃ sukhavittahāniṃ yātrāntarajñānajaḍasya dātuḥ। 

guṇānvitasyaiva guṇaṃ karoti yātrā śubharkṣagrahalagnayogāt।
vyarthā sadoṣasya guṇānvitā api vīṇeva śabdāśrayavarjitasya।। 

yātrā viśuddhā api samaṃ pravṛttā pātrānurūpāṇi phalāni dhatte।
jagaty udīrṇā api hi kauśikasya bhā bhānavī naiva tamaḥ pramārṣṭi।। 

vicintya kāryāṇi niśāvasāne dviṣatsuhṛnmaṇḍalasaṃśritānām।
balārthadeśeṣu niyojitānāṃ samāśritānāṃ ca kṛtākṛtāni।। 

anādśritaḥ khyātaguṇaḥ paro api śūro athavā sādhujanaḥ praśastaḥ।
sampūjanīyo janasaṃprahārtham mameti deśopagato vicintya।। 

saveṇubīṇāpaṇavasvanena gītena pūrvaṃ vyapanītanidraḥ।
śayyāṃ tyajet tūryaravāvasāne śṛṇvan giro maṅgalapāṭhakānām।। 

ajñātapūrvāṇi na dantakāṣṭhāny adyān na patraiś ca samanvitāni।
na cordhvaśuṣkāni na pāṭitāni na yugmaparvāṇi vinā tvacā ca।। 

udaṅmukhaḥ prāṅmukhasaṃsthito vā ṛjv abraṇaṃ tac ca vitastimātram।
adyān narendro viniyamya vācaṃ prakṣālya jahyāc ca śubhapradeśe।। 

abhimukhapatitaṃ praśāntadiksthaṃ śubham atiśobhanam ūrdhvasaṃsthitaṃ ca।
aśubhakaram ato anyathā pradiṣṭaṃ sthitapatitaṃ ca karoti miṣṭam annam।। 

praṇamya devāṃś ca gurūṃś pūrvaṃ datvā ca gāṃ vatsayutāṃ dvijāya।
dṛṣṭvā mukhaṃ sarpiṣi darpaṇe ca nakṣatram ādau śṛṇuyāt tithiṃ ca।। 

śrutvā tithiṃ bhagrahavāsaraṃ ca prāpnoti dharmārthayaśāṃsi saukhyam।
ārogyam āyur vijayaṃ sutāṃś ca duḥsvapnajātaṃ priyatāṃ ca loke।। 

dūrvebhadānāñjanatīrthatoyamṛdrocanāsarṣapapuṣpagandhān।
sitāmbaroṣṇīṣasuvaṇaratnāny āsevya kuryād bhiṣajāṃ vacāṃsi।। 

smitaprasannaprathamābhibhāṣitaiḥ prasādadṛṣṭyā karasaṃparigrahaiḥ।
yathābhirūpaṃ hṛdayāny api dviṣāṃ prasādayan dharmasabhāṃ samāśrayet।। 

vinītaveṣābharaṇaḥ sadakṣiṇaṃ kara samudyamya vicakṣaṇānvitaḥ।
sukhopaviṣṭaḥ sthita eva vā nṛpaḥ samārimitro vyavahāradarśane।। 

kṣamānvito asmīti vicintyam etad daṇḍyeṣu daṇḍakṣamaṇaṃ na dharmaḥ।
daṇḍaprabhoāvo hi sa durjanasya haste na yo jīvati sādhuvargaḥ।। 

sutarām abhivarddhate abhimāno nīcānāṃ kṣamayā anviteṣu yasmāt।
ata ugratareṇa te nivarttyā yena anye api khalās tathā na bhūyaḥ।। 

yasmin gṛhītaḥ sadṛśāparādho mahājanas trāsam upaiti tasmin।
daṇḍo nipātyo manujeśvarena kālāntare anyad vyapadiśya kāryam।। 

dvirada iva madena viprayukto viṣarahito bhujago vyasiś ca kośaḥ।
paribhavam upayāti na aparādhe yadi manujādhipatiḥ karoti daṇḍam।। 

ekasya tulyodarapāṇipādā daṇḍāt prabhītāḥ praṇamanti martyāḥ।
atyugradaṇḍād api codvijante daṇḍo aparādhapratimaḥ śivāya।। 

duṣṭasya daṇḍaḥ sujanasya pūjā nyāyena kośasya ca saṃpravṛddhiḥ।
apakṣapāto arthiṣu rāṣṭrarakṣā pañcaiva yajñāḥ kathitā nṛpāṇām।। 

śrāntaś ca tasmin viniyujya sādhūn saṃgrāmavidyādivibhaktakālaḥ।
sarvāṇi kāryāṇi yathākrameṇa kuryān nṛpaḥ pratyaham ātmavāṃś ca।। 

rājñā kāryaṃ pañcame pañcame ahni kṣaurarkṣe vā śmaśru bhasyodaye vā।
tyaktvā tārāḥ pañcasaptatripūrvā yātrākāle naiva kāryaṃ na yuddhe।। 

ācārasthaḥ sāgarāntāṃ dharitrīṃ bhuṅkte dīrdhaṃ kālam utkhātaśatruḥ।
yatrā ācāras tatra dharmasya vṛddhir dharmād bhogān dehabhede api bhuṅkte।।e 

abhihitaguṇasaṃyutena rājñā kathitaguṇātyayasaṃsthito abhiyojyaḥ।
upahatam upalabhya cāsya deśaṃ balam athavā nirupadruto abhiyuñjyāt।। 

pracuramaśakayūkaṃ makṣikādamśapūrvaṃ balam ajaladavṛṣṭyā pāṃśuvātāhataṃ vā।
piśitarudhiradhānyaprāṇivṛṣṭyā hatam vā karituragamanuṣyā yatra vādhyā na dīnāḥ।। 

śabdāyante muhur api śivā gardabhadhvānatulyaṃ
tyaktasnehāḥ parijanasuhṛdvāhanopaskareṣu।
kaṣṭaṃ ko naḥ śaraṇam iti vā vādino yasya sainye
vidviṣṭā vā pravarapuruṣāḥ so abhiyojyo nṛpeṇa।। 

nirālasyāvanatavadanāḥ ketanasvapnaśīlā
bhraṣṭācārā malinapuruṣacchāyayā ākrāntadehāḥ।
dīrghaśvāsāḥ sajalanayanāḥ śokalobhābhibhūtāḥ
sainye yasya dvijagurusuhṛddveṣiṇaś caiva yodhāḥ।। 

akāraṇaprodgataromakūpā jaye nirāśāḥ prakṛter apetāḥ।
amaṅgalāceṣṭitajātahāsāḥ sainye narā yasya sa cābhiyojyaḥ।। 

kapotakolūkamadhūni yasya samāśrayante dhvajacāmarāṇi।
chatrāyudhādīni ca so abhiyojyo yasyāthavā asyajalāśayānām। 

pratīpagatvaṃ saritām iṣoś ca śoṣo athavā aśoṣyajalāśayānām।
avārideśe salilapravṛtir ahaitukaṃ cāpsu tarec chilā vā।। 

bhaṅgapātacalanāny animittaṃ rodanāni ca surapratimānām।
agnirūpam anilena vinā vā niścalāni ca yadā pracalanti।। 

prasūtivaikṛtyam akālapuṣpāṇy āraṇyasattvasya purapraveśaḥ।
pradoṣakāle kṛkavākuśabdā himāgame ca anyabhṛtaḥ pralāpāḥ।। 

dīrghaṃ dīnaṃ saṃhatāḥ sārameyāḥ krośanty uccair nityam eva anṛtau ca।
hanyur yoṣā yoṣito nirghṛṇāś ca dṛṣyaṃ nityaṃ naktam idrāyudhaṃ vā।। 

tilā vitailā yadi vā arddhatailā sasyasya vṛddhir yadi vā atiriktā।
annasya vairasyam asṛk tarūṇāṃ śuṣkapraroho virujāṃ praṇāśaḥ।। 

vihāya sarpākhuviḍālamatsyān svajātimāṃsāny upabhuñjate vā।
vrajanti vā maithunam anyajātyāṃ dīptāś ca nityaṃ vihagā mṛgāś ca।। 

bhaṅgaḥ pātas toraṇendradhvajānāṃ śītoṣmānāṃ vyatyayo bhūvidāraḥ।
nimnoccānāṃ tuṅgatā nimnatā ca chāyā cā arkasya ābhimukhyena yātā।। 

tryahātiriktaḥ pavano aticaṇḍo gandharvasaṃjñasya bhavet purasya।
vyaktir bhavec ca ahani tārakāṇāṃ naktaṃ ca tārāgaṇasaṃpraṇāśaḥ।। 

prāsādaveśmavasudhāśaragulmanimneṣv āvāsakā balibhujām anapatyatā vā।
ekāṇḍajatvam athavā bhuvi maṇḍalāni kurvanti cakrakam ivopari vā bhramantaḥ।। 

ulkā abhighātena tamo atidīptyā vakrātivakrena suto dharitryāḥ।
ketur gatisparśana dhūpanena cāreṇa pīḍāṃ kurute arkaputraḥ।। 

tribhis tribhir bhair atha kṛttikādyair nipīḍitair bhūpatayo abhiyojyāḥ।
pāñcālanātho magadhādhipaś ca kaliṅgarāḍ ujjayinīpatiś ca।। 

ānarttarāṭ saindhavahārahorau madreśvaro anyaś ca kuraṅganāthaḥ।
ete hi kūrmāṅgasamāśritānāṃ viśeṣapīḍām upayānti bhūpāḥ।। 

aṅgeṣu sūryo yavaneṣu candro bhaumo hy avantyāṃ magadheṣu saumyaḥ।
sindhau gurur bhojakaṭe ca śukraḥ sauraḥ surāṣṭre viṣaye babhūva।। 

mleccheṣu ketuś ca tamaḥ kaliṅge yāto yato ataḥ paripīḍayanti।
svajanmadeśān paripīḍayanti tato abhiyojyā kṣitipena deśāḥ।। 

sampūjyante bhairavoccānunādai raktair māṃsais tālajaṅghādayo vā।
dṛśyante vā yātudhānāḥ prabhūtā bhraṣṭaśrīkaḥ so api deśo abhiyojyaḥ।। 

deśabhraṃśo yair nimittaiḥ pradiṣṭas tās tā vārttā vakti loke viśaṅkaḥ।
tyaktvā deśaṃ yānti yaṃ bhikṣukā vā gamyo deśo so apy asādhupravṛttaḥ।। 

rogābhibhūtaṃ viṣadūṣitaṃ vā yathā vināśābhimukhaṃ śarīram।
vaidyaḥ prayogaiḥ sudṛḍhaṃ karoti rāṣṭraṃ tathā śāntibhir agrajanmā।।e 

dehaḥ kośo yoddhā vāhyaṃ mantraḥ śatrur mārgo apy āyuḥ।
cittaṃ karma prāptir mantrī prāglagnādyā bhāvāś cintyāḥ।। 

trilābhavarjaṃ ravisauribhaumā nighnanti no karmaṇi sūryabhaumau।
puṣṇanti saumyā ripurāśivarjyam astaṃ bhṛgur mṛtyuvilagnam induḥ।। 

tithyudgamendukaraṇarkṣadinakṣaṇeṣu śuddheṣv abhīṣṭaphaladā nṛpater yathā syāt।
yātrā tathā param idaṃ kathayāmi guhyaṃ śiṣyāya nai atad acirād hy uṣitāya dadyāt।।

yat pṛṣṭhataḥ seṣugajāśvayodham niṣṭhitavyūhamarātisainyam।
yasyendracāpaṃ ca lalāṭasaṃstha tat pṛṣṭhabhāgaḥ kuśalo abhihanyāt।। 

nārttā na bhītā na tṛṇānanāś ca vimuktaśastrāḥ prapalayamānāḥ।
kṣīṇāyudhā vājigajāvatīrṇā hy ete na badhyā na ca pīḍanīyāḥ।। 

kulaikatantuḥ śaraṇāgato vā kṛtāñjaliryaś ca vadet tava asmi।
ayudhyamānān avagamya hanyāś na pālakān strīparirakṣitāṃś ca।। 

digdāhakṣitijarajośmavṛṣṭipātair nirghātakṣiticalanādivaikṛtaiś ca।
yuddhānte mṛgaśakunaiś ca dīptanādair no bhadraṃ bhavati jaye api pārthivasya।। 

śubhā mṛgapatatriṇo mṛdusamīraṇohlādakṛd grahāḥ sphuṭamarīcayo vigatareṇudigmaṇḍalam।
yad anyam api vaikṛtan na vijayāvasāne yadā tadā sukham akaṇṭakaṃ nṛpatir atti deśaṃ ripoḥ।। 

paraviṣayapurāptau sādhudevadvijasvaṃ kulajanavanitāś ca kṣmādhipo noparundhyāt।
janapadahitayuktān pūjayet pauramukhyān śubhatithikaraṇarkṣe hṛṣṭasainyo viśoc ca।। 

dṛti manujapatir yathopadeśaṃ bhagaṇavidāṃ prakaroti yo vacāṃsi।
sa sakalamahimaṇḍalādhipatyaṃ vrajati divīva purandaro acireṇa।। 

alabdhalipsā prathamaṃ nṛpasya labdhasya saṃrakṣaṇasampravṛddhī।
samyak pravṛddhasya ca lābhako ayaṃ tīrtheṣu samyak patipādanāni।। 

kendreṣu saumyair guruśukralagne pāpeṣu kendrāṣṭamavarjitesu।
prāmye sthire vā anidhane ca lagne gṛhaṃ viśec chrītakare anukūle। 

pauṣṇe dhaniṣṭhāsv atha vāruṇeṣu svāyambhuvarkṣeṣu triṣūttarāsu।
akṣīṇacandre śubhavāsare ca tithāv arikte ca gṛhapraveśaḥ।। 

dharmeṇa pṛthvīṃ kila pālayitvā jalāśayārāmasurālayāḍhyām।
kṛtvā sakūpāṃ citibhiś citāñ ca jayanti lokān amareśvarasya।। 

svaviṣayam upagamya mānavendro vallim upayācitakāni cādhikāni।
nigaditavidhinaiva sampradadyāt pramathagaṇāsurabhūtadevatābhyaḥ।। 

yātrāvatāra ādāv ācāraś vābhiyojyako deśaḥ।
yogādhyāyo bhiṣakabalyupahārau tathā snānam।। 

agninimittaṃ nakṣatrakaindubhaṃ lakṣaṇaṃ tathā vāryāḥ।
śālāvidhir gajeṅgitamadakaraṇaṃ vājiceṣṭā ca।। 

khaḍgavidhiḥ prasthānaṃ śākunam utsāhapuraniveśaś ca।
adhyāyasaṃpraho ayaṃ samāsataś cātra nirdiṣṭaḥ।।e
