Dharmakīrti Subandhaparīkṣā pāratantryaṃ hi sambandhaḥ siddhe kā paratantratā tasmāt sarvasya bhāvasya sambandho na asti tattvataḥ rūpa-śleṣo hi sambandho dvitve sa ca kathaṃ bhavet tasmāt prakṛti-bhinnānāṃ sambandho na asti tattvataḥ para-apekṣā hi sambandhaḥ so asan katham apekṣate saṃś ca sarva-nirāśaṃso bhāvaḥ katham apekṣate dvayor eka-abhisambandhāt sambandho yadi tad-dvayoḥ kaḥ sambandho anavasthā ca na sambandha-matis tathā tau ca bhāvau tad-anyaś ca sarve te sva-ātmani sthitāḥ ity amiśrāḥ svayaṃ bhāvās tān miśrayati kalpanā tām eva ca anurundhānaiḥ kriyā-kāraka-vācinaḥ bhāva-bheda-pratīty-arthaṃ saṃyojyante abhidhāyakāḥ kāryakāraṇabhāvo api tayor asahabhāvataḥ prasiddhyati kathaṃ dviṣṭho adviṣṭhe sambandhatā katham krameṇa bhāva ekatra vartamāṇo anya-nispṛhaḥ tadbhāve api tad-bhāvāt sambandho na eka-vṛttimān yady apekṣya tayor ekam anyatra asau pravartate upakārī hy apekṣyaḥ syāt kathaṃ ca upakaroty asan yady eka-artha-abhisambandhāt kāryakāraṇatā tayoḥ prāptā dvitva-ādi-sambandhāt savyetara-viśāṇayoḥ dviṣṭho hi kaścit sambandho na ato anyat tasya lakṣaṇam bhāva-abhāva-upadhir yogaḥ kāryakāraṇatā yadi yoga-upādhī na tāv eva kāryakāraṇatā atra kim bhedāc cen nanv ayaṃ śabdo niyoktāraṃ samāśritaḥ paśyann ekam adṛṣṭasya darśane tad-adarśane apaśyat kāryam anveti vinā vyākhyātṛbhir janaḥ darśanād adarśane muktvā kārya-buddher asambhavāt kārya-ādi-śrutir apy atra lāghava-arthaṃ niveśitā tad-bhāva-abhāvāt tat-kārya-gatir yā apy anuvarṇyate saṃketa-viṣaya-ākhyā sā sāsnā-āder go-gatir yathā bhāve bhāvini tad-bhāvo bhāva eva ca bhāvitā prasiddhe hetu-phalate pratyakṣa-anupalambhataḥ etāvanmātra-tattva-arthāḥ kārya-kāraṇa-gocarāḥ vikalpā darśayanty arthān mithyā-arthā ghaṭitān iva bhinne kā ghaṭanā abhinne kāryakāraṇatā api kā bhāve hy anyasya viśliṣṭau śilaṣṭau syātāṃ kathaṃ ca tau saṃyogi-samavāyya-ādi sarvam etena cintitam anyonya-anupakārāc ca na sambandhī ca tādṛśaḥ janane api hi kāryasya kenacit samavāyinā samavāyī tadā na asau na tato atiprasaṅgataḥ tayor anupakāre api samavāye paratra vā sambandho yadi viśvaṃ syāt samavāyi parasparam saṃyoga-janane api iṣṭau tataḥ saṃyoginau na tau karma-ādi-yogita-āpatteḥ sthiteś ca prativarṇitā saṃyoga-ādy-āśraye yogyam ayogyaṃ tac ca jāyate nityaṃ yogya-svabhāvasya tad-vaikalya-virodhataḥ iti tad-yogyatā-vācyaḥ svabhāvo asya nirucyatām vibhāga-yoga-gatibhiḥ kim anyair gamana-ādibhiḥ teṣu satsv api tasya iti sambandhasya aprasiddhitaḥ yuktaḥ svabhāva-bhedo ayam tat pratikṣaṇa-janmanām
