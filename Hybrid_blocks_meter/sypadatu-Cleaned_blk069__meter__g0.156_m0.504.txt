[] vibhrāntākṣyaḥ ketakīnāṃ palāśān
sītkurvāṇāḥ pādalagnān haranti
api caite viṭamukhyāḥ
śrīmantaḥ sakhibhir alaṃkṛtāsanārdhāḥ
kurvantaś caturam amarmavedhi narma
veśyābhiḥ samupagatāḥ samaṃ samantād
ukṣāṇo vraja iva bhānti sopasaryāḥ
api caiṣām etat sadaḥ
nabha ivaśatacandraṃ yoṣitāṃ vaktracandraiḥ
kṛtaśabaladigantaṃ saṃpatadbhiḥ kaṭākṣaiḥ
saparigraham iva yūnāṃ bāhubhiḥ saṃprahārair
nicitam iva śilābhiś candanārdrair urobhiḥ
api cāsmin
ete vibhānti gaṇikājanakalpavṛkṣās
tādātvikāś ca khalu mūlaharāś ca vīrāḥ
bālye api kāṣṭhakalahān kathayanti yeṣāṃ
vṛddhāḥ suyodhanavṛkodarayor ivoccaiḥ
tad etāvad aham api suhṛnnideśaveṣṭane śirasi bhagavate citteśvarāyāñjaliṃ kṛtvā suhṛnnideśād imam adhikāraṃ puraskṛtya prāyaścittārthaṃ tatrabhavatas tauṇḍikoker viṣṇunāgasya ghoṣaṇāpūrvakaṃ viṭān vijñāpayāmi bho bhoḥ
