[] tāḍyatāṃ mattayā tayā iti
etad api pratihatam anena gāndhārakeṇa hastimūrkheṇa kim idam ucyate bhavatā -
l nakhavilikhitaṃ karṇe nāryā niveśitabandhanaṃ
khacitaśabalaṃ dṛṣṭikṣepair apāṅgavilambibhiḥ
yadi narapaśor asyedaṃ bhoḥ śirasy abhipātyate
surabhirajasā prāyaścittaṃ kim asya bhaviṣyati iti
bāḍham evam etad iti pratipannā viṭamukhyāḥ imāv aparau mām āhvayataḥ
guptamaheśvaradattau
suhṛdāv ekāsanasthitāv etau
upagatakāvyapratibhau
vararucikāvyānusāreṇa
yāvad upasarpāmi haṇḍe guptaromaśa, kim āha bhavān -
pādaprakṣālanenāsyāḥ
śiraḥ prakṣālyatām iti
katham etad api pratiṣiddhaṃ traividyavṛddhair atisuhṛdbhir anugṛhītanāmnā maheśvaradattena
pādaprakṣālanaṃ tasyāḥ
pātum apy eṣa nārhati iti
