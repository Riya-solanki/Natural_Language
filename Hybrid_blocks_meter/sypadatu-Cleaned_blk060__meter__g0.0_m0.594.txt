[] taiḥ spṛṣṭvā ciram unmukhīṣu kiraṇair udyānaśākhāsv asau
yāty astaṃ valabhīkapotanayanair ākṣiptarāgo raviḥ
api cedānīm
prākārāgre gavākṣaiḥ patati khagarutaiḥ sūcyamāno biḍālaḥ
prāsādebhyo nivṛtto vrajati samucitāṃ vāsayaṣṭiṃ mayūraḥ
sāṃdhyaṃ puṣpopahāraṃ pariharati mṛgaḥ sthaṇḍile svaptukāmas
toyād uttīrya cāsau bhavanakamalinīvedikāṃ yāti haṃsaḥ
