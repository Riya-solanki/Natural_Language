atrottarottarasaṃkalanayā grathitasya śleṣamūlatvādābhāsatvaṃ spaṣṭameva । kiṃ bhūpatitvaṃ kutaḥ śrīsiṅgakṣitinandanasya tadidaṃ lakṣmīśvarollāsitamiti ।
jātyādyasaṅgaterbhedā bahavo hyanayormatāḥ ।।
tatra jāteḥ jātikriyāguṇadravyairyathā
śrīsiṅgabhūpālamano api bhūbhṛt- padmākaro api sphujapaṅkaśeṣaḥ ।
khyāto asi bhogījanavatsalo api bhūmau suparvo api ca sārvabhaumaḥ ।।
