[] visṛjati kadalīṣu svāḥ prabhādaṇḍarājīḥ
punar api ca sudhābhir varṇayan saudhamālāḥ
kṣarati kisalayebhyo mauktikānīva candraḥ
aho nu khalu kṣīrodenevodvelapravṛttavikīryamāṇavīcirāśinā jyotsnāsaṃjñakena payasā prasarpatānugṛhīta iva jīvalokaḥ saṃprati hi
ete vrajanti turagaiś ca kareṇubhiś ca
karṇīrathair api ca kambalavāhyakaiś ca
āliṅgitā yuvatibhir mṛditā yuvāno
gandharvasiddhamithunāni vihāyasīva
