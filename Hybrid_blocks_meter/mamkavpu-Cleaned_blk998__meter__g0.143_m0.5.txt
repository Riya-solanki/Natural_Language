[kvākāryaṃ śaśalakṣaṇaḥ kva ca kulaṃ bhūyo api dṛśyeta sā
doṣāṇāṃ praśamāya na śrutamaho kope api kāntaṃ mukham ।
kiṃ vakṣyantyapakalmaṣāḥ kṛtadhiyaḥ svapne api sā durlabhā
cetaḥ svāsthyamupaihi kaḥ khalu yuvā dhanyo adharaṃ dhāsyati ।।]
