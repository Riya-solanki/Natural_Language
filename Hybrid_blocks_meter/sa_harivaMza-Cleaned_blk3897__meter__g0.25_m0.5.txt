pañcabhiś cāpi vivyādha so avidūrād vidūratham 
vidūratho api taṃ ṣaḍbhir vivyādhājau śitaiḥ śaraiḥ ।। 
dantavaktro api vivyādha bhānumantaṃ vidūratham । 
triṃśatā pratyavidhyat taṃ balī bāṇair mahābalam 
kṛtavarmā bibhedājau rājaputraṃ tribhiḥ śaraiḥ ।।
