bhavanti cātra ślokāḥ
kanyābhiyujyamānā tu yaṃ manyetāśrayaṃ sukham anukūlaṃ ca vaśyaṃ ca tasya kuryāt parigraham
anapekṣya guṇān yatra rūpam aucityam eva ca kurvīta dhana-lobhena patiṃ sāpatnakeṣv api
tatra yukta-guṇaṃ vaśyaṃ śaktam balavad arthinam upāyair abhiyuñjānaṃ kanyā na pratilobhayet
varaṃ vaśyo daridro api nirguṇo apy ātma-dhāraṇaḥ guṇair yukto api na tv evaṃ bahu-sādhāraṇaḥ
prāyeṇa dhanināṃ dārā bahavo niravagrahāḥ bāhye saty upabhoge api nirvisrambhā bahiḥ-sukhāḥ
nīco yas tv abhiyuñjīta puruṣaḥ palito api vā videśa-gati-śīlaś ca na sa saṃyogam arhati
yadṛcchayābhiyukto yo dambha-dyūtādhiko api sapatnīkaś ca sāpatyo na sa saṃyogam arhati
guṇa-sāmye abhiyoktṝṇām eko varayitā varaḥ tatrābhiyoktari śraiṣṭhyam anurāgātmako hi saḥ
