iha śiṣṭānuśiṣṭānāṃ śiṣṭānām api sarvathā vācām eva prasādena lokayātrā pravartate ।।

idam andhatamaḥ kṛtsnaṃ jāyeta bhuvanatrayam yadi śabdāhavayaṃ jyotir āsaṃsārān na dīpyate ।।

ādirājayaśobimbam ādarśaṃ prāpya vāṅmayam teṣām asaṃnidhāne api na svayaṃ paśya naśyati ।।

gaur gauḥ kāmadughā samyak prayuktā smaryate budhaiḥ duṣprayuktā punār gotvaṃ prayoktuḥ saiva śaṃsati ।।

tad alpam api nopekṣyaṃ kāvye duṣṭaṃ kathaṃ cana syād vapuḥ sundaram api śvitreṇaikena durbhagam ।।

taiḥ śarīraṃ ca kāvyānām alaṃkārāś ca darśitāḥ śarīraṃ tāvad iṣṭārthavyavacchinnā padāvalī ।।

padyaṃ gadyaṃ ca miśraṃ ca tat tridhaiva vyavasthitam padyaṃ catuṣpadī tac ca vṛttaṃ jātir iti dvidhā ।।

chandovicityāṃ sakalas taprapañco nidarśitaḥ sā vidyā naur vivikṣūṇāṃ gambhīraṃ kāvyasāgaram ।।

nyūnam apy atra yaiḥ kaiś cid aṅgaiḥ kāvyaṃ na duṣyati yady upātteṣu saṃpattir ārādhayati tadvidhaḥ ।।

guṇataḥ prāg upanyasya nāyakaṃ tena vidviṣām nirākaraṇam ity eṣa mārgaḥ prakṛtisundaraḥ ।।

apādaḥ padasaṃtāno gadyam ākhyāyikā kathā iti tasya prabhedau dvau tayor ākhyāyikā kila ।।

api tv aniyamo dṛṣṭas tatrāpy anyair udīraṇāt anyo vaktā svayaṃ veti kīdṛg vā bhedakāraṇam ।।

vaktraṃ cāparavaktraṃ ca socchvāsatvaṃ ca bhedakam cihnam ākhyāyikāyāś cet prasaṅgena kathāsv api ।।

āryādivat praveśaḥ kiṃ na vaktrāparavaktrayoḥ bhedaś ca dṛṣṭo lambhādir ucchvāso vastu kiṃ tataḥ ।।

kavibhāvakṛtaṃ cihnam anyatrāpi na duṣyati mukham iṣṭāsrthasaṃsiddhau kiṃ hi na syāt kṛtātmanām ।।

miśrāṇi nāṭakādīni teṣām anyatra vistaraḥ gadyapadyamayī kā cic campūr ity abhidhīyate ।।

tad etad vāṅmayaṃ bhūyaḥ saṃskṛtaṃ prākṛtaṃ tathā apabhraṃśāś ca miśraṃ cety āhur āryāś caturvidhaṃ ।।

saṃskṛtaṃ nāma daivī vāg anvākhyātā maharṣibhiḥ tadbhavas tatsamo deśīty anekaḥ prākṛtakramaḥ ।।

śaurasenī ca gauḍī ca lāṭī cānyā ca tādṛśī yāti prākṛtam ity evaṃ vyavahāreṣu saṃnidhim ।।

saṃskṛtaṃ sargabandhādi prākṛtaṃ skandhakādi yat osarādir apabhraṃśo nāṭakādi tu miśrakam ।।

kathā hi sarvabhāṣābhiḥ saṃskṛtena ca badhyate bhūtabhāṣāmayīṃ prāhur adbhutārthāṃ bṛhatkathām ।।

lāsyacchalitaśampādi prekṣārtham itarat punaḥ śravyam eveti saiṣāpi dvayī gatir udāhṛtā ।।

asty aneko girāṃ mārgaḥ sūkṣmabhedaḥ parasparam tatra vaidarbhagauḍīyau varṇyete prasphuṭāntarau ।।

iti vaidarbhamārgasya prāṇā daśa guṇāḥ smṛtāḥ eṣāṃ viparyayaḥ prāyo dṛśyate gauḍavartmani ।।

madhuraṃ rasavad vāci vastuny api rasasthitiḥ yena mādyanti dhīmanto madhuneva madhuvratāḥ ।।

yayā kayā cic chrutyā yat samānam anubhūyate tadrūpā hi padāsattiḥ sānuprāsā rasāvahā ।।

eṣa rājā yadā lakṣmīṃ prāptavān brāhmaṇapriyaḥ tadā prabhṛti dharmasya loke asminn utsavo abhavat ।।

itīdaṃ nādṛtaṃ gauḍair anuprāsas tu tatpriyaḥ anuprāsād api prāyo vaidarbhair idam īpsitam ।।

cāru cāndramasaṃ bhīru bimbaṃ paśyaitad ambare manmano manmathākrāntaṃ nirdayaṃ hantum udyataṃ ।।

smaraḥ kharaḥ khalaḥ kāntaḥ kāyaḥ kopaś ca naḥ kṛśaḥ cyuto mānodhiko rāgo moho jātosavo gatāḥ ।।

ityādi bandhapāruṣyaṃ śaithilyaṃ ca niyacchati ato naivam anuprāsaṃ dākṣiṇātyāḥ prayuñjate ।।

kāmaṃ sarvo apy alaṃkāro rasam arthe niṣiñcati tathāpy agrāmyataivainaṃ bhāraṃ vahati bhūyasā ।।

kanye kāmayamānaṃ māṃ na tvaṃ kāmayase katham iti grāmyo ayam arthātmā vairasyāya prakalpate ।।

kāmaṃ kandarpacāṇḍālo mayi vāmākṣi nirdayaḥ tvayi nirmatsaro diṣṭyety agrāmo artho rasāvahaḥ ।।

padasaṃdhānavṛttyā vā vākyārthatvena vā punaḥ duṣpratītikaraṃ grāmyaṃ yathā yā bhavataḥ priyā ।।

kharaṃ prahatya viśrāntaḥ puruṣo vīryavān iti evamādi na śaṃsanti mārgayor ubhayor api ।।

ity anūrjita evārtho nālaṃkāro api tādṛśaḥ sukumāratayaivaitad ārohati satāṃ manaḥ ।।

dīptam ity aparair bhūmnā kṛcchrodyam api badhyate nyakṣeṇa kṣapitaḥ pakṣaḥ kṣatriyāṇāṃ kṣaṇād iti ।।

nedṛśaṃ bahu manyatne mārgayor ubhayor api na hi pratītiḥ subhagā śabdanyāyavilaṅghinī ।।

utkarṣavān guṇaḥ kaś cid yasminn ukte pratīyate tadudārāhvayaṃ tena sanāthā kāvyapaddhatiḥ ।।

arthiṇāṃ kṛpaṇā dṛṣṭis tvanmukhe patitā sakṛt tadavasthā punar deva nānyasya mukham īkṣate ।।

iti tyāgasya vākye asminn utkarṣaḥ sādhu lakṣyate anenaiva pathānyac ca samānanyāyam ūhyatām ।।

ojaḥ samāsabhūyastvam etad gadyasya jīvitam padye apy adakṣiṇātyānām idam ekaṃ parāyaṇam ।।

iti padye api paurastyā badhnanty ojasvinīgiraḥ anye tv anākulaṃ hṛdyam icchanty ojo girāṃ yathā ।।

kāntaṃ sarvajagatkāntaṃ laukikārthān atikramāt tac ca vārtābhidhāneṣu varṇanāsv api dṛśyate ।।

gṛhāṇi nāma tāny eva taporāśir bhavādṛśaḥ saṃbhāvayati yāny eva pāvanaiḥ pādapāṃsubhiḥ ।।

lokātīta ivātyartham adhyāropya vivakṣitaḥ yo arthas tenātituṣyanti vidagdhā netare janāḥ ।।

alpaṃ nirmitam ākāśam anālocyaiva vedhasā idam evaṃvidhiṃ bhāvi bhavatyāḥ stanajṛmbhaṇam ।।

idam atyuktir ity uktam etad gauḍopalālitam prasthānaṃ prākpraṇītaṃ tu sāram anyasya vartmanaḥ ।।

anyadharmas tato anyatra lokasīmānurodhinā samyag ādhīyate yatra sa samādhiḥ smṛto yathā ।।

kumudāni nimīlanti kamalāny unmiṣanti ca iti netrakriyādhyāsāl labdhā tadvācinī śrutiḥ ।।

iti hṛdamahṛdyaṃ tu niṣṭhīvati vadhūr iti yugapan naikadharmāṇām adhyāsaś ca smṛto yathā ।।

utsaṅgaśayanaṃ sakhyāḥ stananaṃ gauravaṃ klamaḥ itīme garbhiṇīdharmā bahavo apy atra darśitāḥ ।।

tad etat kāvyasarvasvaṃ samādhir nāma yo guṇaḥ kavisārthaḥ samagro api tam enam anugacchati ।।

iti mārgadvayaṃ bhinnaṃ tatsvarūpanirūpaṇāt tadbhedās tu na śakyante vaktuṃ pratikavisthitāḥ ।।

naisargikī ca pratibhā śrutaṃ ca bahunirmalam amandaś cābhiyogo asyāḥ kāraṇaṃ kāvyasaṃpadaḥ ।।

na vidyate yady api pūrvavāsanāguṇānubandhi pratibhānam adbhutam śrutena yatnena ca vāgupāsitā dhruvaṃ karoty eva kam apy anugraham ।।

tadastatandrair aniśaṃ sarasvatī śramād upāsyā khalu kīrtim īpsubhiḥ kṛśe kavitve api janāḥ kṛtaśramā vidagdhagoṣṭhīṣu vihartum īśate ।।

kāvyaśobhākārān dharmān alaṃkārān pracakṣate te cādyāpi vikalpyante kas tān kārtsnyena vakṣyati ।।

kiṃ tu bījaṃ vikalpānāṃ pūrvācāryaiḥ pradarśitam tad eva parisaṃskartum ayam asmatpariśramaḥ ।।

kāś cin mārgavibhāgārtham uktāḥ prāg apy alaṃkriyāḥ sādhāraṇam alaṃkārajātam anyat pradarśyate ।।

samāsātiśayotprekṣā hetuḥ sūkṣmo lavaḥ kramaḥ preyo rasavad ūrjasvi paryāyoktaṃ samāhitam ।।

sahoktiḥ parivṛttyāśīḥ saṃkīrṇam atha bhāvikam iti vācām alaṃkārā darśitāḥ pūrvasūribhiḥ ।।

nānāvasthaṃ padārthānāṃ rūpaṃ sākṣād vivṛṇvatī svabhāvoktiś ca jātiś cetyādyā sālaṃkṛtir yathā ।।

badhnann aṅgeṣu romāñcaṃ kurvan manasi nirvṛtim netre cāmīlayann eṣa priyāsparśaḥ pravartate ।।

rājīvam iva te vaktraṃ netre nīlotpale iva iyaṃ pratīyamānaikadharmā vastūpamaiva sā ।।

tavānanam ivāmbhojam ambhojam iva te mukham ity anyonyopamā seyam anyonyotkarṣaśaṃsinī ।।

tvanmukhaṃ kamalenaiva tulyaṃ nānyena kena cit ity anyasāmyavyāvṛtter iyaṃ sā niyamopamā ।।

padmaṃ tāvat tavānveti mukham anyac ca tādṛśam asti ced astu tatkārīty asāv aniyamopamā ।।

samuccayopamāpy asti na kāntyaiva mukhaṃ tava hlādanākhyena cānveti karmaṇendum itīdṛśī ।।

tvayy eva tvanmukhaṃ dṛṣṭaṃ dṛśyate divi candramāḥ iyaty eva bhidā nānyety asāv atiśayopamā ।।

mayy evāsyā mukahśrīr ity alam indor vikatthanaiḥ padme api sā yad asty evety asāv utprekṣitopamā ।।

yadi kiṃ cid bhavet padmaṃ subhru vibhrāntalocanam tat te mukhaśriyaṃ dhattām ity asāv adbhutopamā ।।

śaśīty utprekṣya tanvaṅgi tvanmukhaṃ tvanmukhāśayā indum apy anudhāvāmīty eṣā mohopamā smṛtā ।।

kiṃ padmam antarbhrāntāli kiṃ te lolekṣaṇaṃ mukham mama dolāyate cittam itīyaṃ saṃśayopamā ।।

śiśirāṃśupratspardhi śrīmat surabhigandhi ca ambhojam iva te vaktram iti śleṣopamā smṛtā ।।

padmaṃ bahurajaś candraḥ kṣayī tābhyāṃ tavānanam samānam api sotsekam iti nindopamā smṛtā ।।

brahmaṇo apy udbhavaḥ padmaś candraḥ śaṃbhuśirodhṛtaḥ tau tulyau tvanmukheneti sā praśaṃsopamocyate ।।

candreṇa tvanmukhaṃ tulyam ity ācikhyāsu me manaḥ sa guṇo vāstu doṣo vety ācikhyāsopamāṃ viduḥ ।।

na jātu śaktir indos te mukhena pratigarjitum kalaṅkino jaḍasyeti pratiṣedhopamaiva sā ।।

mṛgekṣaṇāṅkaṃ te vaktraṃ mṛgeṇaivāṅkitaḥ śaśī tathāpi sama evāsau notkarṣīti caṭūpamā ।।

na padmaṃ mukham evedaṃ na bhṛṅgau cakṣuṣī ime iti vispaṣṭasādṛśyāt tattvākhyānopamaiva sā ।।

candrabimbād iva viṣaṃ candanād iva pāvakaḥ paruṣā vāg ito vaktrād ity asaṃbhāvitopamā ।।

pūṣṇyātapa ivāhṇīva pūṣā vyomnīva vāsaraḥ vikramas tvayy adhāl lakṣmīm iti mālopamā matā ।।

vākyarthenaiva vākyārthaḥ ko api yady upamīyate ekānekevaśabdatvāt sā vākyārthopamā dvidhā ।।

nalinyā iva tanvaṅgyās tasyāḥ padmam ivānanam mayā madhuvrateneva pāyaṃ pāyam aramyata ।।

naiko api tvādṛśo adyāpi jāyamāneṣu rājasu nanu dvitīyo nāsty eva pārijātasya pādapaḥ ।।

adhikena samīkṛtya hīnam ekakriyāvidhau yad bruvanti smṛtā seyaṃ tulyayogopamā yathā ।।

divo jagarti rakṣāyai pulomārir bhuvo bhavān asurās tena hanyante sāvalepās tvayā nṛpāḥ ।।

kāntyā candramasaṃ dhāmnā sūryaṃ dhairyeṇa cārṇavam rājann anukaroṣīti saiṣā hetūpamā matā ।।

na liṅgavacane bhinne na hīnādhikatāpi vā upamādūṣaṇāyālaṃ yatrodvego na dhīmatām ।।

strīva gacchati ṣaṇḍho ayaṃ vakty eṣā strī pumān iva prāṇā iva priyo ayaṃ me vidyā dhanam ivārjitā ।।

bhavān iva mahīpāla devarājo virājate alam aṃśumataḥ kakṣām āroḍhuṃ tejasā nṛpah ।।

ityevamādau saubhāgyaṃ na jahāty eva jātu cit asty eva kva cid udvegaḥ prayoge vāgvidāṃ yathā ।।

haṃsīva dhavalaś candraḥ sarāṃsīvāmalaṃ nabhaḥ bhartṛbhakto bhaṭaḥ śveva khadyoto bhāti bhānuvat ।।

īdṛśaṃ varjyate sadbhiḥ kāraṇaṃ tatra cintyatāṃ guṇadoṣavicārāya svayam eva manīṣibhiḥ ।।

tasya muṣṇāti saubhāgyaṃ tasya kāntiṃ vilumpati tena sārdhaṃ vigṛhṇāti tulāṃ tenādhirohati ।।

tatpadavyāṃ padaṃ dhatte tasya kakṣāṃ vigāhate tam anvety anubadhnāti tacchīlaṃ tan niṣedhati ।।

tasya cānukarotīti śabdāḥ sādṛśyasūcakāḥ upamāyām ime proktāḥ kavīnāṃ buddhisaukhyadāḥ ।। ।।

aṅgulyaḥ pallavāny āsan kusumāni nakhārciṣaḥ bāhū late vasantaśrīs tvaṃ naḥ pratyakṣacāriṇī ।।

madapāṭalagaṇḍena raktanetrotpalena te mukhena mugdhaḥ so apy eṣa jano rāgamayaḥ krtaḥ ।।

ekāṅgarūpakaṃ caitad evaṃ dviprabhṛtīny api aṅgāni rūpayanty atra yogāyogau bhidākarau ।।

rūpaṇād aṅgino aṅgānāṃ rūpaṇārūpaṇāśrayāt rūpakaṃ viṣamaṃ nāma lalitaṃ jāyate yathā ।।

na mīlayati padmāni na nabho apy avagāhate tvanmukhendur mamāsūnāṃ haraṇāyaiva kalpate ।।

akriyā candrakāryāṇām anyakāryasya ca kriyā atra saṃdarśyate yasmād viruddhaṃ nāma rūpakam ।।

gāmbhīryeṇa samudro asi gauraveṇāsi parvataḥ kāmadatvāc ca lokānām asi tvaṃ kalpapādapaḥ ।।

gāmbhīryapramukhair atra hetubhiḥ sāgaro giriḥ kalpadrumaś ca kriyate tad idaṃ heturūpakam ।।

mukhacandrasya candratvam ittham anyopatāpinaḥ na te sundari saṃvādīty etad ākṣeparūpakam ।।

mukhendur api te caṇḍi māṃ nirdahati nirdayam bhāgyadoṣān mamaiveti tat samādhānarūpakam ।।

naitan mukham idaṃ padmaṃ ta netre bhramarāv imau etāni kesarāṇy eva naitā dantārciṣas tava ।।

na paryanto vikalpānāṃ rūpakopamayor ataḥ diṅmātraṃ darśitaṃ dhīrair anuktam anumīyatām ।। ।।

pavano dakṣiṇaḥ pārṇaṃ jīrṇaṃ harati vīrudhām sa evāvanatāṅgīnāṃ mānabhaṅgāya kalpate ।।

viṣṇunā vikramasthena dānavānāṃ vibhūtayaḥ kvāpi nītāḥ kuto apy āsann ānītā devetarddhayaḥ ।।

ityādidīpakāny uktāny evaṃ madhyāntayor api vākyayor darśayiṣyāmaḥ kāni cit tāni tad yathā ।।

nṛtyanti niculotsaṅge gāyanti ca kalāpinaḥ badhnanti ca payodeṣu dṛśo harṣāśrugarbhiṇīḥ ।।

mando gandhavahaḥ kṣāro vahnir induś ca jāyate carcācandanapātaś ca śastrapātaḥ pravāsinām ।।

jalaṃ jaladharodgīrṇaṃ kulaṃ gṛhaśikhaṇḍinām calaṃ ca taḍitāṃ dāma balaṃ kusumadhanvanaḥ ।।

tvayā nīlotpalaṃ karṇe smareṇāstraṃ śarāsane mayāpi maraṇe cetas trayam etat samaṃ kṛtam ।।

śuklaḥ śvetārciṣo vṛddhyai pakṣaḥ pañcaśarasya saḥ sa ca rāgasya rāgo api yūnāṃ ratyutsavaśriyaḥ ।।

haraty ābhogam āśānāṃ gṛhṇāti jyotiṣāṃ gaṇam ādatte cādya me prāṇān asau jaladharāvalī ।।

atra dharmair abhinnānām abhrāṇāṃ dantināṃ tathā bhramaṇenaiva saṃbandha iti śliṣṭārthadīpakam ।।

anenaiva prakāreṇa śeṣāṇām api dīpake vikalpānām avagatir vidhātavyā vicakṣaṇaiḥ ।। ।।

vikasanti kadambāni sphuṭanti kuṭajadrumāḥ unmīlanti ca kandalyo dalanti kakubhāni ca ।।

utkaṇṭhayati meghānāṃ mālā vṛndaṃ kalāpinām yūnāṃ cotkaṇṭhayaty eṣa mānasaṃ makaradhvajaḥ ।।

jitvā viśvaṃ bhavān atra viharaty avarodhanaiḥ viharatv apsarobhis te ripuvargo divaṃ gataḥ ।। ।।

kutaḥ kuvalayaṃ karṇe karoṣi kalabhāṣiṇi kim apāṅgam aparyāptam asmin karmaṇi manyase ।।

sa vartamānākṣepo ayaṃ kurvaty evāsitotpalam karṇe kā cit priyeṇaivaṃ cāṭukāreṇa rudhyate ।।

satyaṃ bravīmi na tvaṃ māṃ draṣṭuṃ vallabha lapsyase anyacumbanasaṃkrāntalākṣāraktena cakṣuṣa ।।

so ayaṃ bhaviṣyad ākṣepaḥ prāg evātimanasvinī kadācid aparādho asya bhāvīty evam arunddha yat ।।

tava tanvaṅgi mithyaiva rūḍham aṅgeṣu mārdavam yadi satyaṃ mṛdūny eva kim akāṇḍe rujanti mām ।।

sundarī sā na vety eṣa vivekaḥ kena jāyate prabhāmātraṃ hi taralaṃ dṛśyate na tadāśrayaḥ ।।

dharmyākṣepo ayam ākṣipto dharmī dharmaṃ prabhāhvayam anujñāyaiva yadrūpam atyāścaryaṃ vivakṣatā ।।

cakṣuṣī tava rajyete sphuraty adharapallavaḥ bhruvau ca bhugne na tathāpy aduṣṭasyāsti me bhayam ।।

sa eṣa kāraṇākṣepaḥ pradhānaṃ kāraṇaṃ bhiyaḥ svāparādho niṣiddho atra yat priyeṇa paṭīyasā ।।

dūre priyatamaḥ so ayam āgato jaladāgamaḥ dṛṣṭaś ca phullā niculā na mṛtā cāsmi kiṃ nv idam ।।

na ciraṃ mama tāpāya tava yātrā bhaviṣyati yadi yāsyasi yātavyam alam āśaṅkayātra te ।।

dhanaṃ ca bahu labhyaṃ te sukhaṃ kṣemaṃ ca vartmani na ca me prāṇasaṃdehas tathāpi priya mā sma gāḥ ।।

jīvitāśā balavatī dhanāśā durbalā mama gaccha tiṣṭha vā kānta svāvasthā tu niveditā ।।

asāv anādarākṣepo yad anādaravad vacaḥ priyaprayāṇaṃ rundhatyā prayuktam iha raktayā ।।

gaccha gacchasi cet kānta panthānaḥ santu te śivāḥ mamāpi janma tatraiva bhūyād yatra gato bhavān ।।

yadi satyaiva yātrā te kāpy anyā mṛgyatāṃ tvayā aham adyaiva ruddhāsmi randhrāpekṣeṇa mṛtyunā ।।

gantā ced gaccha tūrṇaṃ te karṇau yānti purā ravāḥ ārtabandhumukhodgīrṇāḥ prayāṇaparipanthinaḥ ।।

gaccheti vaktum icchāmi matpriya tvatpriyaiṣiṇī nirgacchati mukhād vāṇī mā gā iti karomi kim ।।

kṣaṇaṃ darśanavighnāya pakṣmaspandāya kupyataḥ premṇaḥ prayāṇaṃ tvaṃ brūhi mayā tasyeṣṭam iṣyate ।।

sahiṣye virahaṃ nātha dehy adṛśyāñjanaṃ mama yad aktanetrāṃ kandarpaḥ prahartā māṃ na paśyati ।।

pravṛttaiva prayāmīti vāṇī vallabha te mukhāt ayatāpi tvayedānīṃ mandapremṇā mamāsti kim ।।

mugdhā kāntasya yātroktiśravaṇād eva mūrchitā buddhvā vakti priyaṃ dṛṣṭvā kiṃ cireṇāgato bhavān ।।

nāghrātaṃ na kṛtaṃ karṇe strībhir madhuni nārpitam tvaddviṣāṃ dīrghikāsyeva viśīrṇaṃ nīlam utpalam ।।

amṛtātmani padmānāṃ dveṣṭari snigdhatārake mukhendau tava satyasminn apareṇa kim indunā ।।

artho na saṃbhṛtaḥ kaś cin na vidyā kā cid arcitā na tapaḥ saṃcintaṃ kiṃ cid gataṃ ca sakalaṃ vayaḥ ।।

kim ayaṃ śaradambhodaḥ kiṃ vā haṃsakadambakam rutaṃ nūpurasaṃvādi śrūyate tan na toyadaḥ ।।

citram ākrāntaviśvo api vikramas te na tṛpyati kadā vā dṛśyate tṛptir udīrṇasya havir bhujaḥ ।।

na stūyase narendra tvaṃ dadāsīti kadā cana svam eva matvā gṛhṇanti yatas tvad dhanam arthinaḥ ।।

ityevamādir ākṣepo hetvākṣepa iti smṛtaḥ anenaiva diśānye api vikalpāḥ śakyam ūhitum ।। ।।

jñeyah so arthāntaranyāso vastu prastutya kiṃ cana tatsādhanasamarthasya nyāso yo anyasya vastunaḥ ।।

bhagavantau jagannetre sūryācandramasāv api paśya gacchata evāstaṃ niyatiḥ kena laṅghyate ।।

jagad ānandayaty eṣa malino api niśākaraḥ anugṛhṇāti hi parān sadoṣo api dvijeśvaraḥ ।।

madhupānakalāt kaṇṭhān nirgato apy alināṃ dhvaniḥ kaṭur bhavati karṇasya kāmināṃ pāpam īdṛśam ।।

kṣiṇotu kāmaṃ śītāṃśuḥ kiṃ vasanto dunoti mām malinācaritaṃ karma surabher nanv asāṃpratam ।।

kumudāny api dāhāya kim ayaṃ kamalākaraḥ na hīndugṛhyeṣūgreṣu sūryagṛhyo madhur bhavet ।।ity arthāntaranyāsacakram।। ।।

śabdopātte prātīte vā sādṛśye vastunor dvayoḥ tatra yad bhedakathanaṃ vyatirekaḥ sa kathyate ।।

dhairyalāvaṇyagāmbhīryapramukhais tvam udanvataḥ guṇais tulyo asi bhedas tu vapuṣaivedṛśena te ।।

abhinnavelau gambhīrāv amburāśir bhavān api asāv añjanasaṃkāśas tvaṃ tu cāmīkaradyutiḥ ।।

ubhayavyatireko ayam ubhayor bhedakau guṇau kārṣṇyaṃ piśaṅgatā cobhau yat pṛthag darśitāv iha ।।

tvaṃ samudraś ca durvārau mahāsattvau satejasau ayaṃ tu yuvayor bhedaḥ sa jaḍātmā paṭur bhavān ।।

sa eṣa śleṣarūpatvāt saśleṣa iti gṛhyatām sākṣepaś ca sahetuś ca darśyate tad api dvayam ।।

sthitimān api dhīro api ratnānām ākaro api san tava kakṣāṃ na yāty eva malino makarālayaḥ ।।

vahann api mahīṃ kṛtsnāṃ saśailadvīpasāgarām bhartṛbhāvād bhujaṅgānāṃ śeṣas tvatto nikṛṣyate ।।

tvanmukhaṃ kamalaṃ ceti dvayor apy anayor bhidā kamalaṃ jalasaṃrohi tvanmukhaṃ tvadupāśrayam ।।

tvanmukhaṃ puṇḍarīkaṃ ca phulle surabhigandhinī bhramadbhramaram ambhojaṃ lolanetraṃ mukhaṃ tu te ।।

candro ayam ambarottaṃso haṃso ayaṃ toyabhūṣaṇam nabho nakṣatramālīdam utphullakumudaṃ payaḥ ।।

sajātīvyatireko ayaṃ tamojāter idaṃ tamaḥ dṛṣṭirodhitayā tulyaṃ bhinnam anyair adarśi tat ।।iti vyatirekacakram।। ।।

prasiddhahetuvyāvṛttyā yat kiṃciṭ kāraṇāntaram yatra svābhāvikatvaṃ vā vibhāvyaṃ sā vibhāvanā ।।

anañjitāsitā dṛṣṭir bhrūr anāvarjitā natā arañjito aruṇaś cāyam adharas tava sundari ।।

nisargādipadair atra hetuḥ sākṣān nivartitaḥ uktaṃ ca surabhitvādi phalaṃ tat sā vibhāvanā ।।iti vibhāvanācakram।। ।।

piban madhu yathākāmaṃ bhramaraḥ phullapaṅkaje apy asaṃnaddhasaurabhyaṃ paśya cumbati kuḍmalam ।।

rūḍhamūlaḥ phalabharaiḥ puṣṇann aniśam arthinaḥ sāndracchāyo mahāvṛkṣaḥ so ayam āsādito mayā ।।

ubhayatra pumān kaś cid vṛkṣatvenovarṇitaḥ sarve sādhāraṇā dharmāḥ pūrvatrānyatra tu dvayam ।।

mallikāmālabhāriṇyaḥ sarvāṅgīṇārdracandanāḥ [[com sarvāṅgaṃ vyāpnotīti sarvāṅgīṇam]] kṣaumavatyo na lakṣyante jyotsnāyām abhisārikāḥ ।।

stanayor jaghanasyāpi madhye madhyaṃ priye tava asti nāstīti saṃdeho na me adyāpi nivartate ।।

aho viśālaṃ bhūpāla bhuvanatritayodaram māti mātum aśakyo api yaśorāśir yad atra te ।।

alaṃkārāntarāṇām apy ekam āhuḥ parāyaṇam vāgīśamahitām uktim imām atiśayāhvayam ।।ity atiśayokticakram।। ।।

anyathaiva sthitā vṛttiś cetanasyetarasya vā anyathotprekṣyate yatra tām utprekṣāṃ vidur yathā ।।

karṇasya bhūṣaṇam idaṃ mamāyātivirodhinaḥ iti karṇotpalaṃ prāyas tava dṛṣṭyā vilaṅghyate ।।

apāṅgabhāgapātinyā dṛṣṭer aṃśubhir utpalam spṛśyate vā na vety evaṃ kavinotprekṣya varṇyate ।।

yadi lepanam eveṣṭaṃ limpatir nāma ko aparaḥ sa eva dharmo dharmī cety anunmatto na bhāṣate ।।

kartā yady upamānaṃ syān nyagbhūto asau kriyāpade svakriyāsādhanavyagro nālam anyad apekṣitum ।।

yo limpaty amunā tulyaṃ tama ity api śaṃsataḥ aṅgānīti na saṃbaddhaṃ so api mṛgyaḥ samo guṇaḥ ।।

yathendur iva te vaktram iti kāntiḥ pratīyate na tathā limpater lepād anyad atra pratīyate ।।

tad upaśleṣaṇārtho ayaṃ limpatir dhvāntakartṛkaḥ aṅgakarmā ca puṃsaivam utprekṣyata itīṣyatām ।।

manye śaṅke dhruvaṃ prāyo nūnam ityevamādibhiḥ utprekṣā vyajyate śabdair ivaśabdo api tādṛśaḥ ।। ity utprekṣācakram।। ।।

hetuś ca sūkṣmaleśau ca vācām uttamabhūṣaṇam kārakajñāpakau hetū tau cānekavidhau yathā ।।

nivartye ca vikārye ca hetutvaṃ tadapekṣayā prāpye tu karmaṇi prāyaḥ kriyāpekṣaiva hetutā ।।

utpravālāṇy araṇyāni vāpyaḥ saṃphullapaṅkajāḥ candraḥ pūrṇaś ca kāmena pānthadṛṣṭer viṣaṃ kṛtam ।।

gato astam arko bhātīndur yānti vāsāya pakṣiṇaḥ itīdam api sādhv eva kālāvasthānivedane ।।

avadhyair indupādānām asādhyaiś candanāmbhasām dehoṣmabhiḥ subodhaṃ te sakhi kāmāturaṃ manaḥ ।।

iti lakṣyāḥ prayogeṣu ramyā jñāpakahetavaḥ abhāvahetavaḥ ke cid vyāhriyante manoharāḥ ।।

gataḥ kāmakathonmādo galito yauvanajvaraḥ kṣato mohaś cyutā tṛṣṇā kṛtaṃ puṇyāśrame manaḥ ।।

vanāny amūni na gṛhāṇy etā nadyo na yoṣitaḥ mṛgā ime na dāyādās tan me nandati mānasam ।।

tvadapāṅgāhvayaṃ jaitram anaṅgāstraṃ yad aṅgane muktaṃ tad anyatas tena so apy ahaṃ manasi kṣataḥ ।।

iti hetuvikalpānāṃ darśitā gatir īdṛśī ।। iṅgitākāralakṣyo arthaḥ saukṣmyāt sūkṣma iti smṛtaḥ ।।

kadā nau saṃgamo bhāvīty ākīrṇe vaktum akṣamam avetya kāntam abalā līlāpadmaṃ nyamīlayat ।।

rājakanyānuraktaṃ māṃ romodbhedena rakṣakaḥ avagaccheyur ā jñātam aho śītānilaṃ vanam ।।

ānandāśru pravṛttaṃ me kathaṃ dṛṣṭvaiva kanyakām akṣi me puṣparajasā vātoddhūtena dūṣitam ।।

ity evam ādisthāne ayam alaṃkāro atiśobhate leśam eke vidur nindāṃ stutiṃ vā leśataḥ kṛtām ।।

yuvaiṣa guṇavān rājā yogyas te patir ūrjitaḥ raṇotsave manaḥ saktaṃ yasya kāmotsavād api ।।

capalo nirdayaś cāsau janaḥ kiṃ tena me sakhi āgaḥpramārjanāyaiva cāṭavo yena śikṣitāḥ ।।

doṣābhāso guṇaḥ ko api darśitaś cāṭukāritā mānaṃ sakhijanoddiṣṭaṃ kartuṃ rāgād aśaktayā ।। ।।

uddiṣṭānāṃ padārthānām anūddeśo yathākramam yathāsaṃkhyam iti proktaṃ saṃkhyānaṃ krama ity api ।।

preyaḥ priyatarākhyānaṃ rasavad rasapeśalam ūrjasvi rūḍhāhaṃkāraṃ yuktotkarṣaṃ ca tat trayam ।।

adya yā mama govinda jātā tvayi gṛhāgate kālenaiṣā bhavet prītis tavaivāgamanāt punaḥ ।।

ity āha yuktaṃ viduro nānyatas tādṛśī dhṛtiḥ bhaktimātrasamārādhyaḥ suprītaś ca tato hariḥ ।।

somaḥ sūryo marudbhūmir vyoma hotānalo jalam iti rūpāṇy atikramya tvāṃ draṣṭuṃ deva ke vayam ।।

iti sākṣātkṛte deve rājño yad rātavarmaṇaḥ prītiprakāśanaṃ tac ca preya ity avagamyatām ।।

mṛte ati pretya saṃgantuṃ yayā me maraṇaṃ matam saivāvantī mayā labdhā katham atraiva janmani ।।

prāk prītir darśitā seyaṃ ratiḥ śṛṅgāratāṃ gatā rūpabāhulyayogena tad idaṃ rasavadvacaḥ ।।

nigṛhya keśeṣv ākṛṣṭā kṛṣṇā yenāgrato mama so ayaṃ duḥśāsanaḥ pāpo labdhaḥ kiṃ jīvati kṣaṇam ।।

ity āruhya parāṃ koṭiṃ krodho raudrātmatāṃ gataḥ bhīmasya paśyataḥ śatrum ity etad rasavadvacaḥ ।।

ajitvā sārṇavām urvīm aniṣṭvā vividhair makhaiḥ adatvā cārtham arthibhyo bhaveyaḥ pārthivaḥ katham ।।

ity utsāhaḥ prakṛṣṭātmā tiṣṭhan vīrarasātmanā rasavattvaṃ girām āsāṃ samarthayitum īśvaraḥ ।।

pāyaṃ pāyaṃ tavārīṇāṃ śoṇitaṃ pāṇisaṃpuṭaiḥ kauṇapāḥ saha nṛtyanti kabandhair antrabhūṣaṇāḥ ।।

idam amlānamānāyā lagnaṃ stanataṭe tava chādyatām uttarīyeṇa navaṃ nakhapadaṃ sakhi ।।

aṃśukāni pravālāni puṣpaṃ hārādibhūṣaṇam śākhāś ca mandirāṇy eṣāṃ citraṃ nandanaśākhinām ।।

vākyasyāgrāmyatāyonir mādhurye darśito rasaḥ iha tv aṣṭarasāyattā rasavattā smṛtā girām ।। ।।

apakartāham asmīti hṛdi te mā sma bhūd bhayam vimukheṣu na me khaḍgaḥ prahartuṃ jātu vāñchati ।।

iti muktaḥ paro yuddhe niruddho darpaśalinā puṃsā kenāpi taj jñeyam ūrjasvītyevamādikam ।। ।।

artham iṣṭam anākhyāya sākṣāt tasyaiva siddhaye yat prakārāntarākhyānaṃ paryāyoktaṃ tad iṣyate ।।

daśaty asau parabhṛtah sahakārasya mañjarīm tam ahaṃ vārayiṣyāmi yuvābhyāṃ svairam āsyatām ।।

saṃgamayya sakhīṃ yūnā saṃkete tadratotsavaṃ nirvartayitum icchantyā kayāpy apasṛtaṃ tataḥ ।। ।।

mānam asyā nirākartuṃ pādayor me patiṣyataḥ upakārāya diṣṭyaitad udīrṇaṃ ghanagarjitam ।। ।।

āśayasya vibhūter vā yan mahatvam anuttamam udāttaṃ nāma taṃ prāhur alaṃkāraṃ manīṣibhiḥ ।।

guroḥ śasanam atyetuṃ na śaśāka sa rāghavaḥ yo rāvaṇaśiraśchedakāryabhāre apy aviklavaḥ ।।

apahnutir apahnutya kiṃcid anyārthadarśanam na pañceṣu smaras tasya sahasraṃ patriṇām iti ।।

candanaṃ candrikā mando gandhavāhaś ca dakṣiṇaḥ seyam agnimayī sṛṣṭir mayi śītā parān prati ।।

upamāpahnutiḥ pūrvam upamāsv eva darśitā ity apahnutibhedānāṃ lakṣyo lakṣyeṣu vistaraḥ ।। ।।

śliṣṭam iṣṭam anekārtham ekarūpānvitaṃ vacaḥ tad abhinnapadaṃ bhinnapadaprāyam iti dvidhā ।।

asāv udayam ārūḍhaḥ kāntimān raktamaṇḍalaḥ rājā harati lokasya hṛdayaṃ mṛdubhiḥ karaiḥ ।।

doṣākareṇa saṃbadhnannakṣatrapathavartinā rājñā pradoṣo mām ittham apriyaṃ kiṃ na bādhate ।।

asty abhinnakriyaḥ kaś cid aviruddhakriyo aparaḥ viruddhakarmā cāsty anyaḥ śleṣo niyamavān api ।।

vakrāḥ svabhāvamadhurāḥ śaṃsantyo rāgam ulbaṇam dṛśo dūtyaś ca karṣanti kāntābhiḥ preṣitāḥ priyān ।।

nistriṃśatvam asāv eva dhanuṣy evāsya vakratā śareṣv eva narendrasya mārgaṇatvaṃ ca vartate ।।

padmānām eva daṇḍeṣu kaṇṭakas tvayi rakṣati athavā dṛśyate rāgimithunāliṅganeṣv api ।।

mahībhṛdbhūrikaṭakas tejasvī niyatodayaḥ dakṣaḥ prajāpatiś cāsīt svāmī śaktidharaś ca saḥ ।।

acyuto apy avṛṣacchedī rājāpy aviditakṣayaḥ devo apy avibudho jajñe śaṃkaro apy abhujaṅgavān ।। ।।

na kaṭhoraṃ na vā tīkṣṇam āyudhaṃ puṣpadhanvanaḥ tathāpi jitam evāsīd amunā bhuvanatrayam ।।

na devakanyakā nāpi gandharvakulasaṃbhavā tathāpy eṣā tapobhaṅgaṃ vidhātuṃ vedhaso apy alam ।।

na baddhā bhrūkuṭir nāpi sphurito daśanacchadaḥ na ca raktābhavad dṛṣṭir jitaṃ ca dviṣatāṃ kulam ।।

na rathā na ca mātaṅgā na hayā na ca pattayaḥ strīṇām apāṅgadṛṣṭyaiva jīyate jagatāṃ trayam ।।

ekacakro ratho yantā vikalo viṣamā hayāḥ ākrāmaty eva tejasvī tathāpy arko nabhastalam ।।

saiṣā hetuviśeṣoktis tejasvīti viśeṣaṇāt ayam eva kramo anyeṣāṃ bhedānām api kalpane।।iti viśeṣokticakram।। ।।

vivakṣitaguṇotkṛṣtair yat samīkṛtya kasya cit kīrtanaṃ stutinindārthaṃ sā matā tulyayogitā ।।

yamaḥ kubero varuṇaḥ sahasrākṣo bhavān api bibhranty ananyaviṣayāṃ lokapāla iti śrutim ।।

saṃgatāni mṛgākṣīṇāṃ taḍidvilasitāni ca kṣaṇadvayaṃ na tiṣṭhanti ghanārabdhāny api svayam ।। ।।

prāvṛṣeṇyair jaladharair ambaraṃ durdināyate rāgeṇa punar ākrāntaṃ jāyate jagatāṃ manaḥ ।।

tanumadhyaṃ pṛthuśroṇi raktauṣṭham asitekṣaṇam natanābhi vapuḥ strīṇāṃ kaṃ na haraty unnatastanam ।।

mṛṇālabāhu rambhoru padmotpalamukhekṣaṇam api te rūpam asmākaṃ tanvi tāpāya kalpate ।।

ity anekaprakāro ayam alaṃkāraḥ pratīyate ।। aprastutapraśaṃsā syād aprakrānteṣu yā stutiḥ ।।

yadi nindann iva stauti vyājastutir asau smrtā doṣābhāsā guṇā eva labhante hy atra saṃnidhim ।।

tāpasenāpi rāmeṇa jiteyaṃ bhūtadhāriṇī tvayā rājñāpi saiveyaṃ jitā mā bhūn madas tava ।।

puṃsaḥ purāṇād ācchidya śrīs tvayā paribhujyate rājann ikṣākuvaṃśasya kim idaṃ tava yujyate ।।

bhujaṅgabhogasaṃsaktā kalatraṃ tava medinī ahaṅkāraḥ parāṃ koṭim ārohati kutas tava ।।

arthāntarapravṛttena kiṃ cit tatsadṛśaṃ phalam sad asad vā nidarśyeta yadi tat syān nidarśanam ।।

udayann eṣa savitā padmeṣv arpayati śriyam vibhāvayitum ṛddhīnāṃ phalaṃ suhṛdanugraham ।।

yāti candrāṃśubhiḥ spṛṣṭā dhvāntarājī parābhavam sadyo rājaviruddhānāṃ sūcayantī durantatām ।। ।।

sahoktiḥ sahabhāvena kathanaṃ guṇakarmaṇām arthānāṃ yo vinimayaḥ parivṛttis tu sā smṛtā ।।

saha dīrghā mama śvāsair imāḥ samprati rātrayaḥ pāṇḍurāś ca mamaivāṅgeḥ saha tāś candrabhūṣaṇāh ।।

vardhate saha pānthānāṃ mūrcchayā cūtamañjarī patanti ca samaṃ teṣām asubhir malayānilāḥ ।।

ity udāhṛtayo dattāḥ sahokter atra kāś cana ।। kriyate parivṛtteś ca kiṃ cid rūpanidarśanam ।।

śastraprahāraṃ dadatā bhujena tava bhūbhujām cirārjitaṃ hṛtaṃ teṣāṃ yaśaḥ kumudapāṇḍuram ।। ।।

āśīrnāmābhilaṣite vastuny āśaṃsanaṃ yathā pātu vaḥ paramaṃ jyotir avāṅmanasagocaram ।। ।।

ākṣipanty aravindāni mugdhe tava mukhaśriyam kośadaṇḍasamagrāṇāṃ kim eṣām asti duṣkaram ।।

śleṣaḥ sarvāsu puṣṇāti prāyo vakroktiṣu śriyam bhinnaṃ dvidhā svabhāvoktir vakroktiś ceti vāṅmayam ।। ।।

tad bhāvikam iti prāhuḥ prabandhaviṣayaṃ guṇam bhāvaḥ kaver abhiprāyaḥ kāvyeṣv āsiddhi saṃsthitaḥ ।।

vyaktir uktikramabalād gambhīrasyāpi vastunaḥ bhāvāyattām idaṃ sarvam iti tad bhāvikaṃ viduḥ ।। ।।

panthāḥ sa eṣa vivṛtaḥ parimāṇavṛttyā saṃhṛtya vistaram anantam alaṅkriyāṇām vācām atītya viṣayaṃ parivartamānān abhyāsa eva vivarītum alaṃ viśeṣān ।।
