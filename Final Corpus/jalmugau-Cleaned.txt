

pīyūṣāhiniveśa eṣa rabhasād asmākam adyāayā bandhūka-dyuti-bāndhavāhara-rasa-syandena mandīkṛtaḥ
itthaṃ daitya-cayaḥ samudra-mathane yena kṣaṇād vañcitas tasmai sādaram oṃ-namo astu kapaṭa-strī-rūpiṇe viṣṇave ।।

hāridra-drava-sodarāya rasanā-maṃḍo paristhāyine satyād anyatamāya vitta-haraṇa-pratyakṣa-caurāya ca
citraṃ durbhaga-rūpa-varṇana-mahā-pāṇḍitya-vidyāya te vyājānāṃ prathamāya garbha-gaṇikā-rāgāya tubhyaṃ namaḥ ।।

dhyānaṃ yat parameśvaraṃ prati sadā yad vīta-rāgaṃ mano yat karmāikaṭhora-ghora-gahanaṃ bhūtyai yad atyādaraḥ
āścaryaṃ ca śarīra-dānam api yat kāryaṃ vikāraṃ vinā tad veśyaia tapasvinī kali-yuge loko api tad-bhāvitaḥ ।।

śrīmantaṃ śirasā bibharti kurute viśva-bhramaṃ helayā dhatte pāṭaka-pakṣa-pātam aniśaṃ naiāiśais tṛpyati
raktaṃ pātum apīate ca sakalaṃ tārkṣyasya mūrtir yathā veśyā kiṃ tu bhujaṅga-bhakṣaṇa-vidhau nāyāi viśrāmyati ।।

te gandharva-pure vasantu vibhavaṃ svapnājitaṃ bhuñjatāṃ kurvantu kṣaṇam antarikṣakusumaiḥ srag-dāma-śobhām api
veśyānāṃ śaśa-śṛṅga-bhaṅgi-sadṛśaṃ vāllabhyam āsādya ye sarvasva-kṣapaṇāya kevalam aho mūrkhāḥ sukhenāate ।।

śvaityaṃ kalpaya kajjale kapi-kuleṣv āropayāāpalaṃ kodaṇḍe janayājavaṃ viracaya grāvṇāṃ gaṇe mārdavam
nimbe sādhaya mādhurīṃ surabhitām ādau rasoe kuru premāṇaṃ gaṇikā-jane api caturaḥ paścāt sakhe drakṣyasi ।।

kālaś cet karuṇā-paraḥ kali-yugaṃ yady adya dharma-priyaṃ nistriṃśo yadi peśalo viṣadharaḥ saṃtoṣadāyī yadi
agniś ced atiśītalaḥ khala-janaḥ sarvoakārī sa ced āyuṣyaṃ yadi vā bhaviṣyati viṣaṃ veśyāpi tad-rāgiṇī ।।

mūḍho nirvyasano vaṇig vikapaṭo rājāhikārī śuciḥ svādhīno bhṛtako jaḍo guṇa-rataś cauro vyapeta-spṛhaḥ
bhītaḥ śānta-matir vitaḥ sa-vinayaḥ karṇejapo dhārmikaḥ kiṃ kutrāi kadāi ko api bhavitā veśyā-jano rāgavān ।।
{ for bhītaḥ var nītaḥ}

kāmo nāti napuṃsakasya kulaṭā-vargasya nāti trapā toyaṃ nāti marīcikāsu satataṃ nāti sthiratvaṃ śriyaḥ
dharmo nāti ca nāstikasya vibhavo nāti pramattāmanaḥ snehānāṃ kaṇikāi nāti gaṇikā-lokasya ca prāyaśaḥ ।।

aśvīye yama-vāhanasya nakulasyāśīviṣāṇāṃ kule mārjārasya ca mūṣakeṣu ghaṭate yā prītir ātyantikī
kṣīṇe arthe vidhureṣu bandhuṣu dṛḍhe lokāavāde śanair jñeyā kāmi-janeṣu saia gaṇikā-vargasya naisargikī ।।

yad yūtena yudhiṣṭhirasya vihitaṃ yad viṣṇunā vā baler yac chukreṇa dhanādhipasya kalinā rājño nalasyāi yat
saṃbhūyāi ca yat surāura-balair unmathya pāthonidher veśyā paśyata līlayaia kurute tat tad gṛhe kāminām ।।

vāk-śurasya raṇo avadhiḥ kutanayopattiḥ kulasyāadhir bāndhavyasya vivāda-bhūmir avadhiḥ saukhyaṃ śrutasy avadhiḥ
vācāṭasya vidagdha-parṣad avadhir laulyaṃ vratasyāadhir bandho viddhi dhanāasānam avadhir veśyāurāgasya ca ।।

svāmī krūram ivāśritaṃ pariṇato yogīa saṃsāriṇaṃ śuddho vipra ivātyajaṃ jana-padaḥ śrīmān iva śvitriṇam
dāsī-putram ivodhataḥ kula-dharo nīco dhanīāthinaṃ paryante tṛṇa-tulyam eva gaṇikā niḥsvaṃ janaṃ paśyanti ।।

vipro dakṣiṇayā mahaiḥ parijano meghoayaiḥ kārṣiko durmantrī vyasanena sāhasa-śataiḥ śūro mṛgair lubdhakaḥ
bandī tyāgibhir akṣarair janayitā lābhāirekair vaṇig rogātair bhiṣag arbhakaiś ca gaṇikā garbheśvarais tuṣyati ।।

ajñānaḥ kalahaiḥ subhāṣita-padaiḥ sabhyo adhama-ṛṇair dhanī jīvaiḥ śākuniko durodarakaraḥ śārair yamaḥ prāṇibhiḥ
grāmīṇair viṭa-peṭakaḥ parijanaiḥ svāmī śiśuḥ kandukaiḥ śrīputrais taruṇais tathā nava-navair vārāṅganā krīḍati ।।

bhṛtyaḥ kuprabhu-sevayā narapatiḥ svacchandatā-durnayair hevākena kavir madena kulajaḥ krodhāninā tāpasaḥ
yogī bhoga-viśeṣa-lampaṭatayā rogī kupathyāśanaī rāgī catvara-kāminī-paricayaiḥ kṣiprād adhaḥ kṣipyate ।।

śālīnāṃ śalabha-vrajo dhuṇa-gaṇo vaṃśa-pratānonater mattehaḥ kamalāarasya kusumoyānasya durmārutaḥ
svas-bhānur vidhu-maṇḍalasya vaḍavā-vakro nidher ambhasā māsāṃ kāmuka-saṃpadāṃ ca gaṇikā-kṣudraḥ kīloadravaḥ ।।

śauryasyāivikatthanaṃ ripu-natir mānasya duḥśīlatā saundaryasya samunnates taralatā jñānasya garva-grahaḥ
aiśvaryasya vikāratā mukharatā vidyā-prakarṣasya ca bhrātar-vibhrama-narmaṇaś ca gaṇikā sarvāmanā dūṣaṇam ।।

cchāyām abhra-dalaiḥ khalaiḥ sujanatāṃ mānonatiṃ yācñayā kārpaṇyena yaśāṃsi dhūrta-kalayā maitrīṃ sukhaṃ sevayā
dharmaṃ prāṇivadhena vāñchati dhanaṃ dyūta-prasādeba yo veśyābhiś ca vilāsam ātmani narāāro asti gaur eva saḥ ।।

nādho mūrkha-samo na sevaka-samo duḥkhī na yakṣmoamo vyādhir nādhir asat-suta-pratinidhis trāso na bhṛtyoḥ samaḥ
kaṣṭaṃ nāhva-samānam indriya-samo nāriḥ para-strī-samaṃ nāāyuṣyam avaśya-nindyam atha no veśyā-samaṃ dṛśyate ।।

kiṃ kāko api vihaṃgamo marur api sthānaṃ karīro api kiṃ śākhī rāhur api grahaḥ sa ca bhaved uṣṭro api kiṃ vā[ha]tam
kiṃ kāco api maṇir mṛgo api karaṭiḥ preṣyo api vā mānuṣaḥ kiṃ veśyāi vilāsinī sa ca tad-āsakto api kiṃ kāmukaḥ ।।

kiṃ tat karma yad asta-śarma rasanā kiṃ sā na yāruṃtudā kiṃ tat prema yad asthiraṃ sa ca pumān kiṃ nāma yo nirguṇaḥ
sā kiṃ śrīr upabhogam arhati na yā sevyaḥ sa kiṃ yo antaraṃ no jānāti tad asti kiṃ vilasitaṃ veśyāsu yat kṛtrimam ।।

dūre tāni dhanāni yeṣu malanaṃ mānasya saṃjāyate bandhuḥ sa astu sukhī mukhe madhumayo yo antaś ca māyāmayaḥ
bhogāḥ pralayaṃ prayāntu katham apy aucitya-bhaṅgena ye yāyātur nidhanaṃ varaṃ nidhuvanaṃ veśyāsu mūlyena yat ।।

asthānāhiniveśitā rati-pater aucitya-bhaṅgo rater vaiyarthyaṃ nava-yauvanasya kim api premṇaḥ kalaṅkāṅkuraḥ
saubhāgyasya vimānanā viguṇatā saundarya-sāra-śriyaḥ śṛṅgārasya viḍambanā kim aparaṃ veśyā-rata-ḍambaraḥ ।।

kruddho yasya manobhavas taruṇimā yenāmano vañcitas tais tair yaś ca kaṭākṣitaḥ paribhavaiḥ sākṣād alakṣmī-mukhaiḥ
yuktāukta-vicāraṇā-parihṛtaḥ kāmaṃ sa kāmī krimiḥ paṇya-strīṣu nitamba-nāmani mahā-nimbe ratiṃ vindati ।।

śāpaḥ ko api mahānayaṃ pariṇatir duṣkarmaṇām īdṛśī daurjanyaṃ paramaṃ vidher idam iyaṃ sākṣād alakṣmī sthirā
sauhārdasya ca vibhramasya ca rater bhartuś ca yūnāṃ tathā yat saṃtyajya kulāṅganāṃ bahu-mukho dāsī-janaḥ sevyate ।।

saṃparko apy atikarkaśaḥ paricayāhāso apy asaṃpat-karo dṛkpāto apy avasādasūr api sakṛt saṃbhāṣaṇaṃ dūṣaṇam
chāyāi cchalanāmikā parimalogāro api hālāhalaṃ veśyānāṃ viṣayeṣu kiṃ puno aho saktir mahā-sāhasam ।।

droho yasya pitā kalāḥ kila catuḥṣaṣṭis tathā mātaraḥ prāṇāḥ sarvam alīkam artha-haraṇaṃ nāma pradhāna-vratam
vikreyaṃ nijam aṅgam aṅgam api cāaṅgaḥ sahāyaḥ svayaṃ tasyāartha-śatāmakasya gaṇikā-vyādheḥ kim asty auṣadham ।।

gṛdhrī nirbharam āmiṣeṣu saraghā ghorā madhūnāṃ bhare tīkṣṇa-nakha-kṣateṣu bhujagī daṃśa-prakāreṣu ca
uttāneṣu vivartaneṣu śaphari vitta-cchale mūṣikā veśyā kāmuka-vañcanāya bhuvane rūpair anekaiḥ sthitā ।।

sāmrājyaṃ madhu-māṃsa-matsya-gilanaṃ mantrī jarat-kuṭṭanī daṇḍyaḥ sārtha-kirāṭasūnuradhanāḥ śṛṅgāriṇaḥ kiṅkarāḥ
ācāro bahu-gāli-dānam adhamo mitraṃ dhanaṃ jīvitaṃ veśyānāṃ puracāra tha bhagnaḥ priyaḥ ।।

rāgīi pratipatti-bhūr vita iti prāpto ativandyaṃ padaṃ ślāghāho vyasanīi hīna-kula ity agryaḥ kuṭumbaḥ svayam
mānyaḥ prāhata ity aho kitava ity āptaś ca kiṃ vāhikaṃ sarvāastha iti pramāṇa-puruṣo veśyābhir abhyarcyate ।।

māṃspākokaṭa-madya-gandhi viharac-ceṭī-naṭī-saṃkulaṃ yad veśyāatanaṃ bhujaṃga parayā bhaktyā puraḥ paśyasi
atrāte gṛha-devatā vidadhatī dig-ghaṭṭanaṃ kuṭṭanī yasyāḥ pratyaham āḍhya-kāmuka-paśu-vrātoahārair baliḥ ।।
{ on māṃspākā see Pāṇini aparasaparāḥ kriyāsātatye}

pratyaṅgāpaṇa-cāru-cāṭu-karaṇa-premoita-prārthanā śrī-vātāyana-darśanāi-gaṇikā-lokasya dharmaḥ paraḥ
kuṭṭanyāḥ puna utkaṭokaṭam idaṃ tatrāty agastya-vrataṃ yat prāṇāutir ekakaia sakalaī ratnāaraiḥ kāmibhiḥ ।।

kandarpa-pratibhū-niveśita-valī-rekhāalī-śobhite līloañcita-bāhu-pāśa-yugalāātaiś ca bhoḥ kāmukāḥ
veśyānāṃ vipule nitamba-phalake śāraiḥ kaṭākṣair ito yad vaḥ krīḍitam atra dāsyati puro dāridram evotaram ।।

śūlenāya mṛtāam adya sa-rajāḥ kartavyam adya vrataṃ tyāgī nāti tato adhiko apara iti ślāghā vipakṣasya ca
kaṃ cin nosahate sa rāja-tanayo jānāti mātā mamey evaṃ mugdha-bhujaṃga rūkṣa-gaṇikā-vāco acirāc chroṣyasi ।।

mālinyaṃ prakaṭi-karoti nibiḍaṃ nairguṇyam ātanvate jīrṇa-sneha-paramparā vidadhate pātre apy aho dūṣaṇam
veśyā dīpa-śikhea bhāti rajanau rūpa-bhramādhīkṛto yatrāaṃ kurute pataṅga-patanaṃ hā hā bhujaṃga-vrajaḥ ।।

keśāarṣaṇa-karmaṇi vyavasitā vīryaṃ harantī paraṃ mlāniṃ kām api tanvatī smita-mukhī lagnā ca karṇātike
utkampaṃ mati-vibhramaṃ vidadhatī yūnāṃ śanair ninditā tāruṇyasya janāavāda veśyā ca saṃdṛśyate ।।

yo ayaṃ nirdaya-danta-khaṇḍana-karo buddhi-prayogair haṭhān nīrandhro bhuja-yantra-pīḍana-vidhiḥ solāsam utpāditaḥ
teneṣor iva kāmukasya sa-rasasyāāya sāraṃ paraṃ veśyābhiḥ kriyate bahir yadi paraṃ niṣkāsanaṃ śalkavat ।।
{ śalkaśakala}

vāgbhir bhartsanam aṃśukāi-haraṇaṃ saṃtāḍanaṃ muṣṭibhiḥ kiṃ cāyad viparīta-darśnam adhas-kāro atha pādāatiḥ
ity-ādi prathamaṃ yadea surata-prauḍhoacāra-krame veśyā paśyata tat tad eva kurute nirvāsane kāminām ।।

yad bhūyaḥ paricumbanaṃ yad asakṛt-sarvāṅgam āliṅganaṃ yat paryutsukam īkṣitaṃ sa-rabhasaṃ yac cāṭukokaṭṭanam 
yat sadbhāva-kathānakaṃ rahasi yat kiṃ cid abhyarthanaṃ veśyānāṃ viṣa-vat tad eva nipuṇaṃ cintyaṃ sadā kāmukaiḥ ।।

bibboke baka-ceṣṭitaṃ bahu-vidha-bhrū-vibhrameṣu bhramaṃ kauṭilyaṃ nakhara-kṣateṣu suratoāyeṣu māyāṃ svayam
sītkārātara-satyam artha-haraṇaṃ sarvatra ye jānate veśyānāṃ paramaṃ rahasyam avanau te ke api medhāvinaḥ ।।

kiṃ cit prāhuṇakāamena kim api prastāra-vistāra-vat- yātrā-caitra-tithi-kramaiḥ kim api ca krodha-prasādoayaiḥ
kiṃ cit kāñcana-ratna-vastra-viṣayair nānā-vidhair dohadair dvārāṇy artha-vinirgamasya gaṇikāḥ kurvanti śṛṅgāriṇām ।।
{ prāhuṇaka a guest}

arth-uṣmā pitṛ-lālanaṃ viṭa-ghaṭā-melaḥ priyam-manyatā tāruṇyaṃ nagare sthitis taralatā dhiḥ kāma-śāstraṃ prati
saṅgītaṃ rajanī vidhur madhu-madaḥ spardhā sa-patnais tathā veśyānām anurakta-vitta-haraṇe kurvanti sāhāyakam ।।

vaidagdhīm avadhīraya vyavahitaṃ kāryaṃ kalā-kauśalaṃ saujanyena gataṃ kim anyad aphalo rāśir guṇānām api
yasmān na praṇayena na praṇatibhiḥ premṇā na na prīṇanair na prāṇair api te bhujaṃga gaṇikā vittaṃ vinā tuṣyati ।।

dāsī nātha tavāam eva vibhavaḥ sarvas tvadīyaḥ sthito mā māṃ nirdaya muñca śūnyam akhilaṃ manye jagat tvāṃ vinā
ity uktvā sahasā bhujaṅga-purato yad veśyayā rudyate bhuktochiṣṭa-daridra-kāmi-vibhavās te kevalaṃ tad viduḥ ।।

svapne kām api saṃsmaran priya mayā dṛṣṭo asi tat te balād evaṃ māṃ prati sāṃprataṃ kim ucitā niṣkāraṇaṃ vañcanā
taj jāgarmi varaṃ niśāsu yad asau nidrāi me drohiṇīy ālāpair gaṇikā-gaṇena na ca ko viśvāsyate kāmukaḥ ।।

bhrū-bhaṅgair atibhaṅguraiḥ kuṭilita-prāntas tathā kuntalaiḥ soekṣair iva cakṣuṣoḥ saralitāāṅgaiś ca bhaṅgy-antaraiḥ
ātmīyāṃ cala-citta-vṛtti-racanāṃ veśyā varākī sadā yūnāṃ darśayatīa te yadi paraṃ mūḍhā na taj jānate ।।

vakrotyā prathamaṃ nirādaratayā paścāt tataḥ kena cid dātavyaṃ bahu tatra yāmi bhavatām bhūyo api nau saṃgamaḥ
ity ukto api na budhyate khalu yadā rāgī daridro jaḍaś ceṭībhir bahu-bhāṣitair api tadā hastāgalair vāryate ।।

bhūyaḥ kāla-vaśāt tam arjita-dhanaṃ dṛṣṭvāha mad-vallabhaḥ saḥ kruddho gamitas tvayei kalaho mātrā samaṃ jāyate
kṛtvā taṃ prati ca vratāi virahaṃ svaṃ nāṭayitvā tatas tasmin mitra-mukhena saṃdhir aparo veśyābhir utpādyate ।।

prauḍha prāṇa-da kānta nātha subhagoāra priya tvāṃ vinā kiṃ vittena gṛheṇa kiṃ kim asubhir hyo apīi yo bhāṣitaḥ
tasyaiāya nirāśa durbhaga paśo nirlajja gacchāhunā dhik tvāṃ nirdhana-caṅgam ity apavadan hrīto na veśyā-janaḥ ।।

ke cid dyūta-karā bhavanti kati-cid bhikṣā-carāḥ kiṅkarāḥ ke cit karma-karāḥ pare dhanavatāṃ ślāghā-karāḥ ke cana
skandhāakta-paṭac-carā dya nucarā śokād a-kiṃ-cit-karās te amī kāmi-varāḥ purā samabhavan veśyāsu ye gocarāḥ ।।

veśyābhir vivaśī-kṛtaḥ kupuruṣaḥ saṃjāyate durgato daurgatyena durodare nipatitaḥ svaṃ hārayaty eva sa
ruddho dyūta-karaiḥ karoti vidhuraś cauryaṃ tatas taskaro vadhyaḥ syān nṛpater aho nu viṣayāakter durantā gatiḥ ।।

ājīvaḥ kapaṭāurāga-kalayā doṣo na duḥśīlatā vaidhavyaṃ na ca bādhate sad-asatoḥ saṃbḥāvanā vyatyayāt
yat-kim-cit-karaṇe para-sva-haraṇe vrīḍā na pīḍā-karī no vā rāja-bhayaṃ ca hī bata sukhaṃ jīvanti vāra-striyas ।।

strīi prīti-karaṃ puraḥ pariṇatau hālāhalaṃ kevalaṃ sarvasya vyasanaṃ kilaiad adhikaṃ tatrāi veśyei ca
prādhānyena tatas tad eva kathitaṃ yat tat-prasaṅgena ca vyākhyātaṃ guṇa-doṣa-jātam ucitaṃ cintyaṃ tad apy ādarāt ।।

cetaḥ-śvāpada-vāgurā daśa-guṇa-sphāra-smaroḍḍāmarāḥ kartavyāvadhayaḥ samudra-laharī-pūrair ivopāditāḥ
nāi svāmy-asamarpaṇena na sakhe sakhyā virodhena vā gūḍhātarmukha-sāhasa-vyatikarā-rakṣyāḥ paraṃ yoṣitaḥ ।।

bhasma-snāna-mahā-vrataṃ katipaya-śrī-bhraṣṭa-saṃbhāvitaṃ sarvāahnava-hasta-lāghava-kalā-kūṭāṣa-śikṣāmakam
pratyāśā-punarukta-hāraṇa-jagad-drohaṃ vivādāpadaṃ dāridryasya nimantraṇaṃ kim aparaṃ dhig dyūta-līlāyitam ।।

hiṃsā-nirghṛṇa-karma-bhūḥ pratipadāeka-pramāda-prasūs- avyāpāra-dhurandharā pratikṛtā grāmyair jaghanyaiḥ śvabhiḥ
śūnyāaṇya-sadā-pravāsa-virasa-vyāyāma-mithyā-guṇāḥ kāya-kleśa-phalāadhir matimatāṃ sevyā mṛgavyāi kim ।।

śaucāāra-vicāra-vāhyam akhilāāryāitāāryakaṃ tat-tad-gopya-rahasya-mantra-bhiduraṃ nirnaiśam akṣṇos tamaḥ
yat-tad-vādda-kali-pramāda-vividhomādaiḥ piśācāyitaṃ madyaṃ mūrtam amedhyam etad itarād anyasya kasya priyam ।।

ulkā-pāta-sahodaraṃ sahacaraṃ naidāgha-jhañjhā-marut- jhampānāṃ hara-kaṇṭha-lālita-gara-droṇī-kuṭumbī-kṛtam
jihvāre kara-pattra-mitram aniśaṃ tat karkaśaṃ durvaco yasyāte vada kad-vadaḥ katham aho sa api svayaṃ jīvati ।।

caṇḍaṃ daṇḍam akāṇḍa eva kalayan rājā prajānāṃ yamaḥ kurvan durvyayam artha-dūṣaṇa-ruciḥ svasyaia sa droha-kṛt
saumyaḥ saumya-karo ativiśva-nayanāando abhinandyaḥ satāṃ sa-śrīkaḥ kamalāaraś ca suṣamāṃ kāṃ kām aho nāhati ।।

saṃsārasya maheśvaro dina-patir dhvāntasya vaidyo rujāṃ prāyaścittam aghasya śāstram aparijñānasya toyaṃ tṛṣaḥ
siddhāñā garalasya tīrtha-saraṇaṃ vṛddhatva-vaiklavyayoḥ saptāṅga-vyasanāaleḥ praśamanoāyaś ca śikṣā satām ।।

oṃkāraḥ sāhasānāṃ nija-guṇa-nipuṇa-pratyavekṣā sukhānāṃ kādācitkaḥ prayogaḥ sakala-paricayoādhi-vaidagdhya-bandhuḥ
kaṣṭānām antya-kāṣṭhā dhana-rasika-vaṇig-lokayātrā-prasaṅgo jīvaj-janmātaraṃ cey alam atibahunā svasti deśātarāya ।।

yady apy asti sa-vistarābhuta-kathā-bhūyiṣṭha-nānā-vidha- vyākhyāyāṃ niravadya-hṛdya-caritoāraṃ ca deśātaram
tatrāi sva-gṛheṣu guṇavad-goṣṭhī-gariṣṭhī-kṛta- sphāro anārata-bhāratī-rasa-pariṣyandaḥ sakhe sundaraḥ ।।

te vandyā makaranda-bindu-madhurair abhyutthitād akṣarair yeṣā vānti sarasvatī-parimalogārā mukhābho-ruhāt
vandyās te api tad-antarāla-patitā ye bhṛṅga-bhaṅgī-juṣaḥ sāndrāanda-karambitaṃ jagad idaṃ vindanti nindanti ca ।।

dṛṣṭvā deśam aśeṣam ā jalanidher ālokya kautūhalād āsthānīr avanībhṛtāṃ ca punar apy āgatya deśaṃ nijam
kāruṇyāt taruṇaṃ janaṃ prati satām abhyarthanābhis tathā so ayaṃ saṃprati Jalhaṇena kavinā Mudghopadeśaḥ kṛtaḥ ।।

vidyāhyāsa-paramparā-paricayair bālyaṃ kṛtāthī-kṛtaṃ tāruṇyaṃ taruṇī-nirantara-parīrambhaiś ca saṃbhāvitam
asmākaṃ sura-sindhu-rodhasi punaḥ śaṃbhāva-daṃbhācanair vṛddhatvasya viśuddhatāṃ janayituṃ kartavya-śeṣaḥ sthitaḥ ।।
