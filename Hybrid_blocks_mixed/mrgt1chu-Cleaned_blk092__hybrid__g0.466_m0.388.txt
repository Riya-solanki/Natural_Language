yadi pramāṇam asatyarūpaṃ paramārthataḥ paramātmana eva satyatvāt tathāvidhena pramāṇenaitat pramīyamāṇaṃ manor nimitena pradīpena san tamasāvasthitapadārthapravivecanaprāyam ।। 
atha satyam eva pramāṇam evaṃ tarhi sa paramātmā prameyatvena sthitaḥ ।। 
pramāṇaṃ hi prameyaṃ paricchindat pramāṇatām āsādayati itarathā pramāṇataivāsya na syāt ।। 
kiṃca ataḥ ।। 
kim anyad yatraitad dvitayaṃ pramāṇaprameyalakṣaṇaṃ tatra pramātṛpramityātmakam anyad api dvayam anyonyasattvavyapekṣatvāt sthitam eva ।। 
na hi pramātāraṃ kartāram antareṇa pramāṇaprameyayoḥ kvacit kiṃcitkaratvaṃ karaṇakarmaṇoḥ kriyāsiddhau kartrāśrayatvāt ।। 
pramitir api kriyārūpo vyāpāras tebhyaḥ pṛthag evānvayavyatirekābhyām upalabhyata iti catuṣṭayam avaśyaṃbhāvi ।। 
yad āha ākṣapādaḥ ।
catasṛṣu caivaṃvidhāsu sarvo api vyavahāraḥ parisamāpyata iti ।। 
kiṃ cāta ity ata āha ।।
