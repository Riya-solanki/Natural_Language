lakṣīr vrajam anu tulasī-vanam anu bhavad-udaya-sphurad-udayā ।
dṛśyata iti tava pada-dhūlyāplavam icchaty api pati-hṛdayā ।।yasyā vīkṣaṇam api valita-kṣaṇa-pārṣada-vṛnda-nidhānam ।
tadvad vayam api hṛdi vāñchām api nahyāmaḥ savitānam ।।
atrāpi-śabdaḥ samuccaye pūrvavat kākvā niṣedhe vā ।
