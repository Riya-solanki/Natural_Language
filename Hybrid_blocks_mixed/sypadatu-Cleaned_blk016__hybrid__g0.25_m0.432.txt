[] saṃjalpān iva kurvate vyatikarāt prāsādamālāḥ sitāḥ
api ca
giribhyo araṇyebhyaḥ salilanidhikacchād api maror
narendrair āyātair diśi disi niviṣṭais ca śatasaḥ
vicitrām ekasthām anavagatapūrvām avikalām
iha sraṣṭuḥ sṛṣṭer bahuviṣayatāṃ pasyati janaḥ
śakayavanatukhārapārasīkair
magadhakirātakaliṅgavaṅgakāśaiḥ
nagaram atimudāyutaṃ samantān
mahiṣakacolakapāṇḍyakeralaiś ca
