darśiteṅgitākāraṃ tu pravirala-darśanām apūrvāṃ ca dūtyopasarpayet
saināṃ śīlato anupraviśyākhyāna-kapaṭaiḥ subhagaṃ-karaṇa-yogair loka-vṛttāntaiḥ kavi-kathābhiḥ pāradārika-kathābhiś ca tasyāś ca rūpa-vijñāna-dākṣiṇya-śīlānupraśaṃsābhiś ca tāṃ rañjayet
katham evaṃ-vidhāyās tavāyam itthaṃ-bhūtaḥ patir iti cānuśayaṃ grāhayet
na tava subhage dāsyam api kartuṃ yukta iti brūyāt
manda-vegatām īrṣyālutāṃ śaṭhatām akṛta-jñatāṃ cāsaṃbhoga-śīlatāṃ kadaryatāṃ capalatām anyāni ca yāni tasmin guptāny asyā abhyāśe sati sad-bhāve atiśayena bhāṣeta
yena ca doṣeṇodvignāṃ lakṣayet tenaivānupraviśet
yadāsau mṛgī tadā naiva śaśatā-doṣaḥ
etenaiva baḍava-hastinī-viṣayaś coktaḥ
nāyikāyā eva tu viśvāsyatām upalabhya dūtītvenopasarpayet prathama-sāhasāyāṃ sūkṣma-bhāvāyāṃ ceti goṇikāputraḥ
sā nāyakasya caritam anulomatāṃ kāmitāni ca kathayet
prasṛta-sadbhāvāyāṃ ca yuktyā kārya-śarīram itthaṃ vadet
śṛṇu vicitram idaṃ subhage tvāṃ kila dṛṣtvāmutrāsāv itthaṃ gotra-putro nāyakaś cittonmādam anubhavati prakṛtyā sukumāraḥ kadā cid anyatrāparikliṣṭa-pūrvas tapasvī tato adhunā śakyam anena maraṇam apy anubhavitum iti varṇayet
tatra siddhā dvitīye ahani vāci vaktre dṛṣṭyāṃ ca prasādam upalakṣya punar api kathāṃ pravartayet
śṛṇvatyāṃ cāhalyāvimāraka-śākuntalādīny anyāny api laukikāni ca kathayet tad-yuktāni
vṛṣatāṃ catuḥṣaṣṭi-vijñatāṃ saubhāgyaṃ ca nāyakasya ślāghanīyatāṃ cāsya pracchannaṃ saṃprayogaṃ bhūtam abhūta-pūrvaṃ vā varṇayet
ākāraṃ cāsya lakṣayet
savihasitaṃ dṛṣṭvā saṃbhāṣate
āsane copanimantrayate
kvāsitaṃ kva śayitaṃ kva bhuktaṃ kva ceṣṭitaṃ kiṃ vā kṛtam iti pṛcchati
vivikte darśayaty ātmānam
ākhyānakāni niyuṅkte cintayantī niḥśvāsīti vijṛmbhate ca
prīti-dāyaṃ ca dadāti
iṣṭeṣūtsaveṣu ca smarati
punar-darśanānubandhaṃ visṛjati
sādhu-vādinī satī kim idam aśobhanam abhidhatsa iti kathām anubadhnāti
nāyakasya śāṭhya-cāpalya-saṃbaddhān doṣān dadāti
pūrva-pravṛttaṃ ca tat-saṃdarśanaṃ kathābhiyogaṃ ca svayam akathayantī tayocyamānam ākāṅkṣati
nāyaka-manoratheṣu ca kathyamāneṣu saparibhavaṃ nāma hasati na ca nirvadatīti
dūty enāṃ darśitākārāṃ nāyakābhijñānair upabṛṃhayet
asaṃstutāṃ tu guṇa-kathanair anurāga-kathābhiś cāvarjayet
nāsaṃstutādṛṣṭākārayor dūtyam astīty auddālakiḥ
asaṃstutayor api saṃsṛṣṭākārayor astīti bābhravīyāḥ
saṃstutayor apy asaṃsṛṣṭākārayor astīti goṇikāputraḥ
asaṃstutayor adṛṣṭākārayor api dūtī-pratyayād iti vātsyāyanaḥ
tāsāṃ mano-harāṇy upāyanāni tāmbūlam anulepanaṃ srajam aṅgulīyakaṃ vāso vā tena prahitaṃ darśayet
teṣu nāyakasya yathārthaṃ nakha-daśana-padāni tāni tāni ca cihnāni syuḥ
vāsasi ca kuṅkumāṅkam añjaliṃ nidadyāt
patra-cchedyāni nānābhiprāyākṛtīni darśayet lekha-pattra-garbhāṇi karṇa-pattrāṇy āpīḍāṃś ca
teṣu sva-manorathākhyāpanam pratiprābhṛta-dāne caināṃ niyojayet
evaṃ kṛta-paraspara-parigrahayoś ca dūtī-pratyayaḥ samāgamaḥ
sa tu devatābhigamane yātrāyām udyāna-krīḍāyāṃ jalāvataraṇe vivāhe yajña-vyasanotsaveṣv agny-utpāte caura-vibhrame janapadasya cakrārohaṇe prekṣa-vyāpāreṣu teṣu teṣu ca kāryeṣv iti bābhravīyāḥ
sakhī-bhikṣukī-kṣapaṇikā-tāpasī-bhavaneṣu sukhopāya iti goṇikāputraḥ
tasyā eva tu gehe vidita-niṣkrama-praveśe cintitātyaya-pratīkāre praveśanam upapannaṃ niṣkramaṇam avijñāta-kālaṃ ca tan nityaṃ sukhopāyaṃ ceti vātsyāyanaḥ
nisṛṣṭārthā[] parimitārthā[] patra-hārī[] svayaṃdūtī[] mūḍha-dūtī[] bhāryā-dūtī[] mūka-dūtī[] vāta-dūtī[] ceti dūtī-viśeṣāḥ
nāyakasya nāyikāyāś ca yathāmanīṣitam artham upalabhya sva-buddhyā kārya-saṃpādinī nisṛṣṭārthā[]
sā prāyeṇa saṃstuta-saṃbhāṣaṇayoḥ
nāyikayā prayuktā asaṃstuta-saṃbhāṣaṇayor api
kautukāc cānurūpau yuktāv imau parasparasyety asaṃstutayor api
kāryaikadeśam abhiyogaikadeśaṃ copalabhya śeṣaṃ saṃpādayatīti parimitārthā[]
sā dṛṣṭa-parasparākārayoḥ pravirala-darśanayoḥ
saṃdeśa-mātraṃ prāpayatīti patra-hārī[]
sā pragāḍha-sadbhāvayoḥ saṃsṛṣṭayoś ca deśa-kāla-saṃbodhanārtham
dautyena prahitānyayā svayam eva nāyakam abhigacched ajānatī nāma tena sahopabhogaṃ svāpne vā kathayet gotra-skhalitaṃ bhāryāṃ cāsya nindet td-vyapadeśena svayam īrṣyāṃ darśayet nakha-daśana-cihnitaṃ vā kiṃ cid dadyāt bhavate aham ādau dātuṃ saṃkalpiteti cābhidadhīta mama bhāryāyā kā ramaṇīyeti vivikte paryanuyuñjīta sā svayaṃ-dūtī[]
tasya vivikte darśanaṃ pratigrahaś ca
pratigraha-cchalenānyām abhisaṃdhāyāsyāḥ saṃdeśāśravaṇa-dvāreṇa nāyakaṃ sādhayet tāṃ copahanyāt sāpi svayaṃ-dūtī[]
etayā nāyako apy anya-dūtaś ca vyākhyātaḥ
nāyaka-bhāryāṃ mugdhāṃ viśvāsyāyantraṇayānupraviśya nāyakasya ceṣṭitāni pṛcchet yogāñ śikṣayet sākāraṃ maṇḍayet kopam enāṃ grāhayet evaṃ ca pratipadyasveti śrāvayet svayaṃ cāsyāṃ nakha-daśanapadāni nirvartayet tena dvāreṇa nāyākam ākārayet sā mūḍha-dūtī[]
tasyās tayaiva pratyuttarāṇi yojayet
sva-bhāryāṃ vā mūḍhaṃ prayojya tayā saha viśvāsena yojayitvā tayaivākārayet ātmanaś ca vaicakṣaṇyaṃ prakāśayet sā bhāryā-dutī[] tasyās tayaivākāra-grahaṇam
bālāṃ vā paricārikām adoṣa-jñām aduṣṭenopāyena prahiṇuyāt tatra sraji karṇapattre vā gūḍha-lekha-nidhānaṃ nakha-daśana-padam vā sā mūka-dūtī[] tasyās tayaiva pratyuttara-prārthanam
pūrva-prastutārtha-liṅga-saṃbaddham anya-janāgrahaṇīyaṃ laukikārthaṃ vyarthaṃ vā vacanam udāsīnā yā śrāvayet sā vāta-dūtī[] tasyā api tayaiva pratyuttara-prārthanam
iti tāsāṃ viśeṣāḥ bhavanti cātra ślokāḥ
vidhavekṣaṇikā dāsī bhikṣukī śilpa-kārikā praviśaty āsu viśvāsaṃ dūtī-kāryaṃ ca vindati
saṃkṣepeṇa dūtī-karmāṇy āha
vidveṣaṃ grāhayet patyau ramaṇīyāni varṇayet citrān surata-saṃbhogān anyāsām api darśayet
nāyakasyānurāgaṃ ca punaś ca rati-kauśalam prārthanāṃ cādhika-strībhir avaṣṭambhaṃ ca varṇayet
asaṃkalpitam apy artham utsṛṣṭaṃ doṣa-kāraṇāt punar āvartayaty eva dūtī vacana-kauśalāt
