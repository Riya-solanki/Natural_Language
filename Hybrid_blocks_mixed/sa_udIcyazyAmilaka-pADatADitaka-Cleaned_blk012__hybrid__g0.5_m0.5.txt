bravīṣi - sarvaṃ tāvat tiṣṭhatu dayitaviṣṇur api bhavato viṭaḥ saṃmata iti kaḥ saṃdehaḥ kiṃ bravīṣi - eṣa yo ayaṃ rājño baleṣvadhikṛtaḥ pāraśavaḥ kavir iti bāḍham evam evaitat kiṃ bravīṣi - mā tāvad bhoḥ yaḥ saṃkucaty upahitapraṇayo api rājño yo maṅgalaiḥ svapiti ca pratibudhyate ca devārcanād api ca gulgulugandhavāsā yo asau kiṇatrayakaṭhoralalāṭajānuḥ api ca devakulād rājakulaṃ rājakulād yāti devakulam eva iti yasya yānti divasāḥ kuladvaye saṃprasaktasya katham asāv api viṭa iti ā evam etat astīdam asya viṭabhāvapratyanīkabhūtaṃ purāṇaghṛtagandha iva kiṃ tu pūrvāvantiṣu yasya veśakalahe hastāgraśākhā hṛtā
