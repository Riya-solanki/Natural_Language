rāga-vṛddhau saṃgharṣātmakaṃ nakha-vilekhanam
tasya prathama-samāgame pravāsa-pratyāgamane pravāsa-gamane kruddha-prasannāyāṃ mattāyāṃ ca prayogaḥ na nityam acaṇḍa-vegayoḥ
tathā daśana-cchedyasya sātmya-vaśād vā
tad [] ācchuritakam [] ardha-candro [] maṇḍalaṃ [] rekhā [] vyāghra-nakhaṃ [] mayūra-padakaṃ [] śaśa-plutakam [] utpala-pattrakam iti rūpato aṣṭa-vikalpam
kakṣau stanau galaḥ pṛṣṭhaṃ jaghanam ūrū ca sthānāni
pravṛtta-rati-cakrāṇāṃ na sthānam asthānaṃ vā vidyata iti suvarṇanābhaḥ
tatra savya-hastāni pratyagra-śikharāṇi dvi-tri-śikharāṇi caṇḍa-vegayor nakhāni syuḥ
anugata-rāji samam ujjvalam amalinam avipāṭitaṃ vivardhiṣṇu mṛdu snigdha-darśanam iti nakha-guṇāḥ
dīrghāṇi hasta-śobhīny āloke ca yoṣitāṃ citta-grāhīṇi gauḍāṇāṃ nakhāni syuḥ
hrasvāni karma-sahiṣṇūni vikalpa-yojanāsu ca svecchāpātīni dākṣiṇatyānām
madhyamāny ubhaya-bhāñji mahārāṣṭrakāṇām iti
taiḥ suniyamitair hanu-deśe stanayor adhare vā laghu-karaṇam anudgata-lekhaṃ sparśa-mātra-jananād romāñca-karam ante saṃnipāta-vardhamāna-śabdam [] ācchuritakam
prayojyāyāṃ ca tasyāṅga-saṃvāhane śirasaḥ kaṇḍūyane piṭaka-bhedane vyākulī-karaṇe bhīṣaṇena prayogaḥ
grīvāyāṃ stana-pṛṣṭhe ca vakro nakha-pada-niveśo [] ardha-candraḥ
tāv eva dvau parsa-parābhimukhau maṇḍalam
nābhi-mūla-kakundara-vaṃkṣaṇeṣu tasya prayogaḥ
sarva-sthāneṣu nātidīrghā [] lekhā 
saiva vakrā [] vyāghra-nakhakam ā stana-mukham
pañcabhir abhimukhair lekhā cūcukābhimukhī [] mayūra-padakam
tat-saṃprayoga-ślāghāyāḥ stana-cūcuke saṃnikṛṣṭāni pañca-nakha-padāni [] śaśa-plutakam
stana-pṛṣṭhe mekhalā-pathe cotpala-pattrākṛtīty [] utpala-pattrakam
ūrvoḥ stana-pṛṣṭhe ca pravāsaṃ gacchataḥ smāraṇīyakaṃ saṃhatāś catasras tisro vā lekhāḥ
iti nakha-karmāṇi
ākṛti-vikāra-yuktāni cānyāny api kurvīta
vikalpānām anantatvād ānantyāc ca kauśala-vidher abhyāsasya ca sarva-gāmitvād rāgātmakatvāc chedyasya prakārān ko abhisamīkṣitum arhatīty ācāryāḥ bhavati hi rāge api citrāpekṣā vaicitryāc ca parasparaṃ rāgo janayitavyaḥ vaicakṣaṇya-yuktāś ca gaṇikās tat-kāminaś ca parasparaṃ prārthanīyā bhavanti dhanur-vedādiṣv api hi śastra-karma-śāstreṣu vaicitryam evāpekṣyate kiṃ punar iheti vātsyāyanaḥ
na tu para-parigṛhītāsv evaṃ kuryāt pracchanneṣu pradeśeṣu tāsām anusmaraṇārthaṃ rāga-vardhanāc ca viśeṣān darśayet
nakha-kṣatāni paśyantyā gūḍha-sthāneṣu yoṣitaḥ cirotsṛṣṭāpy abhinavā prītir bhavati peśalā 
cirosṛṣṭeṣu rāgeṣu prītir gacchet parābhavam rāgāyatana-saṃsmāri yadi na syān nakha-kṣatam
paśyato yuvatiṃ dūrān nakhocchiṣṭa-payodharām bahu-mānaḥ parasyāpi rāga-yogaś ca jāyate
puruṣaś ca pradeśeṣu nakha-cihnair vicihnitaḥ cittaṃ sthiram api prāyaś calayaty eva yoṣitaḥ
nānyat paṭutaraṃ kiṃ cid asti rāga-vivardhanam nakha-danta-samutthānāṃ karmaṇāṃ gatayo yathā
