śaśo vṛṣo aśva iti liṅgato nāyaka-viśeṣāḥ nāyikā punar mṛgī baḍavā hastinī ceti
tatra sadṛśa-saṃprayoge sama-ratāni trīṇi
viparyayeṇa viṣamāṇi ṣaṭ viṣameṣv api puruṣādhikyaṃ ced anantara-saṃprayoge dve ucca-rate vyavahitam ekam ucca-tara-ratam viparyaye punar dve nīca-rate vyavahitam ekaṃ nīca-tara-rataṃ ca teṣu samāni śreṣṭhāni tara-śabdāṅkite dve kaniṣṭhe śeṣāṇi madhyamāni
sāmye apy uccāṅkaṃ nīcāṅkāj jyāyaḥ iti pramāṇato nava-ratāni
yasya saṃprayoga-kāle prītir udāsīnā vīryam alpaṃ kṣatāni ca na sahate sa manda-vegaḥ
tad-viparyayau madhyama-caṇḍa-vegau bhavataḥ tathā nāyikāpi
tatrāpi pramāṇavad eva nava-ratāni
tadvat kālato api śīghra-madhya-cira-kālā nāyakāḥ
tatra striyāṃ vivādaḥ
na strī puruṣavad eva bhāvam adhigacchati
sātatyāt tv asyāḥ puruṣeṇa kaṇḍūtir apanudyate
sā punar ābhimānikena sukhena saṃsṛṣṭā rasāntaraṃ janayati tasmin sukha-buddhir asyāḥ
puruṣa-pratīteś cānabhijñatvāt kathaṃ te sukham iti praṣṭum aśakyatvāt
katham etad upalabhyata iti cet puruṣo hi ratim adhigamya svecchayā viramati na striyam apekṣate na tv evaṃ strīty auddālikaḥ
tatraitat syāt cira-vege nāyake striyo anurajyante śīghra-vegasya bhāvam anāsādyāvasāne abhyasūyinyo bhavanti tat sarvaṃ bhāva-prāpter aprāpteś ca lakṣaṇam
tac ca na kaṇḍūti-pratīkāro api hi dīrgha-kālaṃ priya iti etad upapadyata eva tasmāt saṃdigdhatvād alakṣaṇam iti
saṃyoge yoṣitaḥ puṃsā kaṇḍūtir apanudyate tac cābhimāna-saṃsṛṣṭaṃ sukham ity abhidhīyate
sātatyād yuvatir ārambhāt prabhṛti bhāvam adhigacchati puruṣaḥ punar anta eva etad upapannataram na hy asatyāṃ bhāva-prāptau garbha-saṃbhava iti bābhravīyāḥ
tatrāpi tāv evāśaṅkā-parihārau bhūyaḥ
tatraitat syāt ṇ sātatyena rasa-prāptāv ārambha-kāle madhyastha-cittatā nātisahiṣṇutā ca tataḥ krameṇādhiko rāga-yogaḥ śarīre nirapekṣatvam ante ca virāmābhīpsety etad upapannam iti
tac ca na sāmānye api bhrānti-saṃskāre kulāla-cakrasya bhramarakasya vā bhrāntāv eva vartamānasya prārambhe manda-vegatā tataś ca krameṇa pūraṇaṃ vegasyety upapadyate dhātu-kṣayāc ca virāmābhīpseti tasmād anākṣepaḥ
suratānte sukhaṃ puṃsāṃ strīṇāṃ tu satataṃ sukham dhātu-kṣaya-nimittā ca virāmecchopajāyate
tasmāt puruṣavad eva yoṣito api rasa-vyaktir draṣṭavyā
kathaṃ hi samānāyām evākṛtāv ekārtham abhiprapannayoḥ kārya-vailakṣaṇyaṃ syāt
upāya-vailakṣaṇyād abhimāna-vailakṣaṇyāc ca
katham upāya-vailakṣaṇyaṃ tu sargāt kartā hi puruṣo adhikaraṇaṃ yuvatiḥ anyathā hi kartā kriyāṃ pratipadyate anyathā cādhāraḥ tasmāc copāya-vailakṣaṇyāt sargād abhimāna-vailakṣaṇyam api bhavati abhiyoktāham iti puruṣo anurajyate abhiyuktāham aneneti yuvatir iti vātsyāyanaḥ
tatraitat syād upāya-vailakṣaṇyavad eva hi kārya-vailakṣaṇyam api kasmān na syād iti tac ca na hetumad upāya-vailakṣaṇyam tatra kartr-ādhārayor bhinna-lakṣaṇatvād ahetumat-kārya-vailakṣaṇyam anyāyyaṃ syāt ākṛter abhedād iti
tatraitat syāt saṃhatya kārakair eko artho abhinirvartyate pṛthak pṛthak svārtha-sādhakau punar imau tad ayuktam iti
tac ca na yugapad anekārtha-siddhir api dṛśyate yathā meṣayor abhighāte kapitthayor bhede mallayor yuddha iti na tatra kāraka-bheda iti ced ihāpi na vastu-bheda iti upāya-vailakṣaṇyaṃ tu sargād iti tad abhihitaṃ purastāt tenobhayor api sadṛśī sukha-pratipattir iti
jāter abhedād daṃpatyoḥ sadṛśam sukham iṣyate tasmāt tathopacaryā strī yathāgre prāpnuyād ratim sadṛśatvasya siddhatvāt kāla-yogīny api bhāvato api kālataḥ pramāṇavad eva nava ratāni
raso ratiḥ prītir bhāvo rāgo vegaḥ samāptir iti rati-paryāyāḥ saṃprayogo rataṃ rahaḥ śayanaṃ mohanaṃ surata-paryāyāḥ
pramāṇa-kāla-bhāva-jānāṃ saṃprayogānām ekaikasya nava-vidhatvāt teṣāṃ vyatikare surata-saṃkhyā na śakyate kartum atibahutvāt
teṣu tarkād upacārān prayojayed iti vātsyāyanaḥ
prathama-rate caṇḍa-vegatā śīghra-kālatā ca puruṣasya tad-viparītam uttareṣu yoṣitaḥ punar etad eva viparītam ā dhātu-kṣayāt
prāk ca strī-dhātu-kṣayāt puruṣa-dhātu-kṣaya iti prāyovādaḥ
mṛdutvād upamṛdyatvān nisargāc caiva yoṣitaḥ prāpnuvanty āśu tāḥ prītim ity ācāryā vyavasthitāḥ
etāvad eva yuktānāṃ vyākhyātaṃ sāmprayogikam mandānām avabodhārthaṃ vistaro ataḥ pravakṣyate
