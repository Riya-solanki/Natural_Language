nāntaḥpurāṇāṃ rakṣaṇa-yogāt puruṣa-saṃdarśanaṃ vidyate patyuś caikatvād aneka-sādhāraṇatvāc cātṛptiḥ tasmāt tāni prayogata eva paras-paraṃ rañjayeyuḥ
dhātreyikāṃ sakhīṃ dāsīṃ vā puruṣavad alaṃkṛtyākṛti-saṃyuktaiḥ kanda-mūla-phalāvayavair apadravyair vātmābhiprāyaṃ nivartayeyuḥ
puruṣa-pratimā avyakta-liṅgāś cādhiśayīran
rājānaś ca kṛpā-śīlā vināpi bhāva-yogād āyojitāpadravyā yāvad artham ekayā rātryā bahvībhir api gacchanti yasyāṃ tu prītir vāsaka ṛtur vā tatrābhiprāyataḥ pravartanta iti prācyopacārāḥ
strī-yogeṇaiva puruṣāṇām apy alabdha-vṛttīnāṃ viyoniṣu vijātiṣu strī-pratimāsu kevalopamardanāc cābhiprāya-nivṛttir vyākhyātā
yoṣāveṣāṃś ca nāgarakān prāyeṇāntapurikāḥ paricārikābhiḥ saha praveśayanti
teṣām upāvartane dhātreyikāś cābhyantara-saṃsṛṣṭā āyatiṃ darśayantyaḥ prayateran
sukha-praveśitām apasāra-bhūmiṃ viśalatāṃ veśmanaḥ pramādaṃ rakṣiṇām anityatāṃ parijanasya varṇayeyuḥ
na cāsad-bhūtenārthena praveśayituṃ janam āvartayeyur doṣāt
nāgarakas tu suprāpam apy antaḥpuram apāya-bhūyiṣṭhatvān na praviśed iti vātsyāyanaḥ
sāpasāraṃ tu pramada-vanāvagāḍhaṃ vibhakta-dīrgha-kākṣyam alpa-pramatta-rakṣakaṃ proṣita-rājakaṃ kāraṇāni samīkṣya bahuṣa āhūyamāno artha-buddhyā kakṣyā-praveśaṃ ca dṛṣṭvā tābhir eva vihitopāyaḥ praviśet
śakti-viṣaye ca pratidinaṃ niṣkrāmet
bahiś ca rakṣibhir anyad eva kāraṇam apadiśya saṃsṛjeta
antaścāriṇyāṃ ca paricārikāyāṃ viditārthāyāṃ saktam ātmānaṃ rūpayet tad-alābhāc ca śokam antaḥpraveśinībhiś ca dūtī-kalpaṃ sakalam ācaret
rāja-pranidhīṃś ca budhyeta
dūtyās tv asaṃcāre yatra gṛhītākārāyāḥ prayojyāyā darśana-yogas tatrāvasthānam
tasminn api tu rakṣiṣu paricārikā-vyapadeśaḥ
cakṣur anubadhnantyām iṅgitākāra-nivedanam yatra saṃpāto asyās tatra citra-karmaṇas tad-yuktasya vyarthānāṃ gīta-vastukānāṃ krīḍanakānāṃ kṛta-cihnānām āpīnakānām aṅgulīyakasya ca nidhānam
pratyuttaraṃ tayā dattaṃ prapaśyet tataḥ praveśane yateta
yatra cāsya niyataṃ gamanam iti vidyāt tatra prachannasya prāg evāvasthānam
rakṣi-puruṣa-rūpo vā tad-anujñāta-velāyāṃ praviśet
āstaraṇa-prāvaraṇa-veṣṭitasya vā praveśa-nirhārau
puṭāpuṭa-yogair vā naṣṭacchāyā-rūpaḥ
tatrāyaṃ prayogaḥ ṇ nakula-hṛdayaṃ corakatumbī-phalāni sarpākṣīṇi cāntardhūmena pacet tato añjanena sama-bhāgena peṣayet anenābhyakta-nayano naṣṭa-cchāyā-rūpaś carati [anyaiṣ ca jala-brahma-kṣema-śiraḥ-praṇītair bāhya-pānakair vā
rātri-kaumudīṣu ca dīpikā-saṃbādhe suraṅgayā vā
tatraitad bhavati 
dravyāṇām api nirhāre pānakānāṃ praveśane āpānakotsavārthe api ceṭikānāṃ ca saṃbhrame
vyatyāse veśmanāṃ caiva rakṣiṇāṃ ca viparyaye udyāna-yātrā-gamane yātrātaś ca praveśane
dīrgha-kālodayāṃ yātrāṃ proṣite cāpi rājani praveśanaṃ bhavet prāyo yūnāṃ niṣkramaṇaṃ tathā
parasparasya kāryāṇi jñātvā cāntaḥpurālayāḥ eka-kāryās tataḥ kuryuḥ śeṣāṇām api bhedanam
dūṣayitvā tato anyonyam eka-kāryārpaṇe sthiraḥ abhedyatāṃ gataḥ sadyo yatheṣṭaṃ phalam aśnute
tatra rāja-kula-cāriṇya eva lakṣaṇyān puruṣān antaḥpuraṃ praveśayanti nātisurakṣatvād aparāntikānām
kṣatriya-saṃjñakair antaḥpura-rakṣibhir evārthaṃ sādhayanty ābhīrakāṇām
preṣyābhiḥ saha tad-veṣān nāgaraka-putrān praveśayanti vātsagulmakānām
svair eva putrair antaḥpurāṇi kāma-cārair jananī-varjam upayujyante vaidarbhakānām
tathā praveśibhir eva jñāti-saṃbandhibhir nānyair upayujyante strairājakānām
brāhmaṇair mitrair bhṛtyair dāsa-ceṭaiś ca gauḍānām
parispandāḥ karmakarāś cāntaḥpureṣv aniṣidddhā anye api tad-rūpāś ca saindhavānām
arthena rakṣiṇam upagṛhya sāhasikāḥ saṃhatāḥ praviśanti haimavatānām
puṣpa-dāna-niyogān nagara-brāhmaṇā rāja-viditam antaḥpurāṇi gacchanti paṭāntaritaś caiṣām ālāpaḥ tena prasaṅgena vyatikaro bhavati baṅga-aṅga-kaliṅgakānām
saṃhatya navadaśety ekaikaṃ yuvānaṃ pracchādayanti prācyānām iti evaṃ para-striyaḥ prakurvīta
