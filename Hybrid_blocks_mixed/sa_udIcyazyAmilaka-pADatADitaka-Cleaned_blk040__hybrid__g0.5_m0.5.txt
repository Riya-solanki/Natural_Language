mām uktavatī na ca rahasyānākhyānena bhavatīm avakṣeptum arhāmi śrūyatām idam apūrvam iti tato anayā yathāvṛttaṃ sarvaṃ mahyam ākhyātam tena hi tvam apy anena śrotrāmṛtena saṃvibhaktum arhasīti eṣā satalaghātaṃ prahasya kathayati sundari, kiṃ bravīṣi - śrūyatām idānīṃ yan mama priyasakhyā kathitam sā hi mām uktavatī priyasakhi, sa hi mayā āliṅgito api bahudhā paricumbito api śroṇyarpito api karajair upacodito api khinnāsmi dārv iva yadā na sa mām upaiti śayyāṅgam ekam upagūhya tato asmi suptā iti tato mayoktā kṛcchraṃ batānubhūtavaty asi kim etan nāvagacchāmīti tato niḥśvasya mām uktavatī yadā sarvopāyaiś caṭubhir upayāto api sa mayā na yatnaṃ kurvāṇo mayi manasijecchām alabhata tatas tasmin sarvapratihatavidhānāsmi sahasā svadaurbhāgyaṃ matvā stanataṭavikampaṃ praruditā tataḥ sa māṃ rudatīm utsaṅgam āropya muhur muhur vyarthaiś cumbanapariṣvaṅgair āśvāsayan nāma dṛḍham ātmānam āyāsitavān uktaṃ ca mayā kiṃ te pāṇibhyāṃ spṛṣṭayeti tato
