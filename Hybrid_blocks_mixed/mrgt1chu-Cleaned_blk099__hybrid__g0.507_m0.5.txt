acetanam api pradhānaṃ buddhimat kartṛpreraṇaṃ vinā svātantryeṇa kathaṃ kāryakaraṇe pravartate ।। 
na hy anusaṃdhānaśūnyasya buddhimato api ghuṇakīṭasyevākṣaralekhane pravṛttir yuktā kiṃ punaḥ pariṇāmino jaḍasya ।। 
yaś cātra cetanaḥ puruṣaḥ sa tadadhīno na karteti prakṛtisthānāṃ neyam uktir ucitā ।। 
idānīm anekāntavādinirākaraṇāya saṃkhyātas tanmataṃ pūrvapakṣayitum ucyate ।।
