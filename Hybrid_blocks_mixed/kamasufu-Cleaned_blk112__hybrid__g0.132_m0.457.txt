vyākhyāta-kāraṇāḥ para-parigrahopagamāḥ 
teṣu sādhyatvam anatyayaṃ gamyatvam āyatiṃ vṛttiṃ cādita eva parīkṣeta
yadā tu sthānāt sthānāntaraṃ kāmaṃ pratipadyamānaṃ paśyet tadātma-śarīropaghāta-trāṇārthaṃ para-parigrahān abhyupagacchet
daśa tu kāmasya sthānāni
[]cakṣuḥ-prītir []manaḥ-saṅgaḥ []saṃkalpotpattir []nidrācchedas []tanutā []viṣayebhyo vyāvṛttir []lajjā-pranāśa []unmādo []mūrcchā []maraṇam iti teṣāṃ liṅgāni
tatrākṛtito lakṣaṇataś ca yuvatyāḥ śīlaṃ satyaṃ śaucaṃ sādhyatāṃ caṇḍavegatāṃ ca lakṣayed ity ācāryāḥ
vyabhicārād ākṛti-lakṣaṇa-yogānām iṅgitākārābhyām eva pravṛttir boddhavyā yoṣita iti vātsyāyanaḥ
yaṃ kiṃ cid ujjvalaṃ puruṣaṃ dṛṣtvā strī kāmayate tathā puruṣo api yoṣitām apekṣayā tu na pravartate iti goṇikāputraḥ
tatra striyaṃ prati viśeṣaḥ
na strī dharmam adharmaṃ cāpekṣate kāmayata eva kāryāpekṣayā tu nābhiyuṅkte
svabhāvāc ca puruṣeṇābhiyujyamānā cikīrṣanty api vyāvartate
punaḥ punar abhiyuktā siddhyati
puruṣas tu dharma-sthitim ārya-samayaṃ cāpekṣya kāmayamāno api vyāvartate
tathā-buddhiś cābhiyujyamāno api na siddhyati
niṣkāraṇam abhiyuṅkte abhiyujyāpi punar nābhiyuṅkte siddhāyāṃ ca mādhyasthyaṃ gacchati
sulabhām avamanyate durlabhām ākaṅkṣata iti prāyovādaḥ
