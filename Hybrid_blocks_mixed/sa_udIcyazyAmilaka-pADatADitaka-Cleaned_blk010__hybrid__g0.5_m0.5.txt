tasyāṃ pariṣadi tac chrutvā viṣṇunāgo apy anugṛhīta iti prasthitaḥ tvaṃ cāpi viṭasaṃnipātakarmaṇi niyukta iti bāḍham kiṃ bravīṣi - ke punar iha bhavato viṭāḥ saṃmatā iti nanu bhavān eva tāvad agre viṭaḥ kiṃ bravīṣi - katham aham api nāma viṭaśabdenānugṛhīta iti kaḥ saṃśayaḥ? śrūyatām divasam akhilaṃ kṛtvā vādaṃ saha vyavahāribhir divasavigame bhuktvā bhojyaṃ suhṛdbhavane kvacit nisi ca ramase veśastrībhiḥ kṣipasy api cāyudhaṃ jalam api ca te nāsty āvāse tathāpi ca katthase tat kathaṃ tvam aviṭaḥ kiṃ bravīṣi - yady evam anugṛhītaḥ saṃnipātayiṣyasi viṭān viṭalakṣaṇaṃ tāvac chrotum icchāma iti prathamaḥ kalpaḥ śrūyatām svaiḥ prāṇair api vidviṣaḥ praṇayiṇām āpatsu yo rakṣitā yasyārtau bhavati sva eva śaraṇaṃ khaḍgadvitīyo bhujaḥ saṃgharṣān madanāturo mṛgayate yaṃ vāramukhyājanaḥ sa jñeyo viṭa ity apāvṛtadhano yo nityam evārthiṣu api ca caraṇakamalayugmair arcitaṃ sundarīṇāṃ
