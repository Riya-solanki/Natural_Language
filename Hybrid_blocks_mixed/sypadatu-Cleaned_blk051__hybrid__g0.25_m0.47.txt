[] haridattasya pūrvapraṇayinīm akalyarūpām adya vārttāṃ praṣṭuṃ tenaiva prahito asmi tvaṃ tu māṃ katham apy avagacchasīti āścaryam idaṃ hi bhavataḥ suhṛdvyāpāreṣu sthairyaṃ tasyāś ca vāramukhyāyāḥ pūrvapraṇayiṣv āpadgateṣv api pratipattiś ca ataś cainām
varṇānurūpojjvalacāruveṣāṃ
lakṣmīm ivālekhyapaṭe niviṣṭām
sāpahnavāṃ kāmiṣu kāmavanto
arūpāṃ virūpām api kāmayante
api ca - atiduṣkarakāriṇīṃ cainām avagacchāmi kutaḥ asaṃśayaṃ hi sā
kārānirodhād avikāragauraṃ
devārcanājātakiṇaṃ lalāṭe
āsyaṃ bṛhacchmaśruvitānanaddhaṃ
kālāsthinirbhugnam ivāvaleḍhi
kim āha bhavān - ata evāsyām asmākam ādara iti bhavatv evam suhṛdanuraktaṃ bhavantaṃ khyāpayāmo vayam eṣa
