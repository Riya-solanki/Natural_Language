yad api sadasacchabdābhidheyaṃ śreṣṭhāśreṣṭharūpaṃ vastu tad api tadvido naikatrābhidadhati tasyāpi bhinnaviṣayatvāt ।। 
na hi yad evāśreṣṭhaṃ tad eva śreṣṭham iti vaktuṃ śakyam ।। 
ayam eva hi bhāvānāṃ bhedaḥ yad viruddhadharmādhyāsaḥ ।। 
atha mataṃ yat tac chreṣṭhāśreṣṭhatvaṃ tad ekatra sambhavaty eva apekṣāvaśāt ।। 
yathāyaṃ devadatto yajñadattasakāśād abhirūpaḥ caitrāpekṣayā tu nīrūpa iti ।। 
evaṃ tarhi yajñadattasyāśreṣṭhatvaṃ caitrasya ca śreṣṭhatvam ity ubhayaṃ tata ity apekṣātaḥ ।। 
evaṃ bhinnaṃ labdhaṃ yad api dvayaṃ tadāpekṣikatvād asatyam ।। 
apekṣā hi nāma na vāstavī ।। 
tasyāḥ kila vastunaḥ sati sadbhāve kiṃ prayojanaṃ siddhasattākatvenānapekṣatvāt ।। 
tathā cāhuḥ saṃś ca sarvo nirāśaṃso bhāvaḥ katham apekṣate ।। 
na cālabdhasattāke vastuny apekṣāyāḥ kim api karaṇīyam asti tasyāsattvād eva tadapekṣānupapatteḥ ।। 
tad idam uktam apekṣā na satāṃ siddher asiddher api nāsatām iti ।। 
tad evaṃ na kathaṃcid api sadasator abhedopapattiḥ ।।
