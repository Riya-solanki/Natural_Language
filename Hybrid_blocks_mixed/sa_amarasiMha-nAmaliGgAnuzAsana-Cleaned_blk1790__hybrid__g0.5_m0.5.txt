ālokau darśanadyotau bherīpaṭakamānakau utsaṅgacihnayoraṅkaḥ kalaṅko aṅkāpavādayoḥ takṣako nāgavarddhakyorarkaḥ sphaṭikasūryayoḥ marute vedhasi braghne puṃsi kaḥ kaṃ śiro ambunoḥ syāt pulākas tucchadhānye saṃkṣepe bhaktasikthake ulūke kariṇaḥ pucchamūlopānte ca pecakaḥ kamaṇḍalau ca karakaḥ sugate ca vināyakaḥ kiṣkur haste vitastau ca śūkakīṭe ca vṛścikaḥ pratikūle pratīkas triṣvekadeśe tu puṃsyayam syād bhūtikaṃ tu bhūnimbe kattṛṇe bhūstṛṇe api ca jyotsnikāyāṃ ca ghoṣe ca kośātakyatha kaṭphale site ca khadire somavalkaḥ syādatha sihvake tilakalke ca piṇyāko bāhlīkaṃ rāmaṭhe api ca kapikroṣṭuśvānaḥ svarṇe api gairikam pīḍārthe api vyalīkaṃ syādalīkaṃ tvapriye anṛte śīlānvayāvanūke dve śalke śakalavalkale sāṣṭe śate suvarṇānāṃ hemnyurobhūṣaṇe pale dīnāre api ca niṣko astrī kalko astrī śamalainasoḥ dambhe apyatha pināko astrī śūlaśaṅkaradhanvanoḥ dhenukā tu kareṇvāṃ ca meghajāle ca kālikā kārikā yātanāvṛttyoḥ karṇikā karṇabhūṣaṇe karihaste aṅgulau padmabījakośyāṃ triṣūttare vṛndārakau rūpimukhyāveke mukhyānyakevalāḥ syād dāmbhikaḥ kaukkuṭiko yaś cādūreritekṣaṇaḥ lalāṭikaḥ prabhor bhāladarśī kāryākṣamaś ca yaḥ bhūbhṛnnitambavalayacakreṣu kaṭako astriyām sūcyagre kṣudraśatrau ca romaharṣe ca kaṇṭakaḥ pākau paktiśiśū madhyaratne netari nāyakaḥ paryaṅkaḥ syāt parikare syād vyāgre api ca lubdhakaḥ peṭakas triṣu vṛnde api gurau deśye ca deśikaḥ kheṭakau grāmaphalakau dhīvare apica jālikaḥ puṣpareṇau ca kiñjalkaḥ śulko astrī strīdhane api ca syāt kallole apyutkalikā vārdhakaṃ bhāvavṛndayoḥ kariṇyāṃ cāpi gaṇikā dārakau bālabhedakau andhe apyaneḍamūkaḥ syāt ṭaṅkau darpāśmadāraṇau mṛdbhāṇḍe apyuṣṭrikā manthe khajakaṃ rasadarvake iti kāntāḥ mayūkhas tviṭkarajvālāsvalibāṇau śilīmukhau śaṅkho nidhau lalaṭāsthnikambau na strīndriye api kham dhṛṇijvāle api śikhe śailavṛkṣau nagāvagau iti khāntāḥ āśugau vāyuviśikhau śarārkavihagāḥ khagāḥ pataṅgau pakṣisūryau ca pūgaḥ kramukavṛndayoḥ paśavo api mṛgā vegaḥ pravāhajavayorapi parāgaḥ kausume reṇau snānīyādau rajasyapi gaje api nāgamātaṅgāvapāṅgas tilake api ca sargaḥ svabhāvanirmokṣaniścayādhyāyasṛṣṭiṣu yogaḥ saṃnahanopāyadhyānasaṃgatiyuktiṣu bhogaḥ sukhe stryādibhṛtāvaheś ca phaṇakāyayoḥ cātake hariṇe puṃsi sāraṅgaḥ śavale triṣu kapau ca plavagaḥ śāpe tvabhiṣaṅgaḥ parābhave yānādyaṅge yugaḥ puṃsi yugaṃ yugme kṛtādiṣu svargeṣupaśuvāgvajra diṅnetradhṛṇibhūjale lakṣyadṛṣṭyā striyāṃ puṃsi gaur liṅgaṃ cihna śephasoḥ śṛṅgaṃ prādhānyasānvoś ca varāṅgaṃ mūrdhaguhyayoḥ bhagaṃ śrīkāmamāhātmyavīryayatnārkakīrtiṣu iti gāntāḥ parighaḥ parighāte astre apyogho vṛnde ambhasāṃ raye mūlye pūjāvidhāvargho ahoduḥkhavyasaneṣvagham triṣviṣṭe alpe laghuḥ kācāḥ śikyamṛdbhedadṛgrujaḥ
